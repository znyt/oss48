        但是，司机来了后，说只看到她的行李箱，未见到她人，他因为担心她，所以才亲自来跑这一趟。
<br/>
<br/>        却不料遇上了这事……
<br/>
<br/>        不管那个死者跟她是什么关系，死因是否与她有关，他都将她从这事里给捞了出来。
<br/>
<br/>        傅夜沉不希望童筱颖深陷其中，更不希望看到她被警察叫去警察局里一遍又一遍地像审犯人一样接受询问，做着录口供的事情。
<br/>
<br/>        她是他的心尖宠，岂能容许他人那样待她。
<br/>
<br/>        只要他一句话，没有人敢动她一丝一毫。
<br/>
<br/>        “是何明旭找人把他打死的！他们两……”童筱颖声音哽咽，不依不饶地接着说道。
<br/>
<br/>        但她的话，再次被傅夜沉冷冷地打断：“筱颖，够了！我说过，这件事！与你无关！警察自会查明一切！你无需多嘴！”
<br/>
<br/>        “大哥，他是因为我才……唔——”
<br/>
<br/>        再一次，童筱颖的话还未说完，就被傅夜沉给放了下来，用手捂住了她的嘴。
<br/>
<br/>        他把她按到了墙上，大手捂着她嘴巴时，手指上全是她的眼泪。
<br/>
<br/>        童筱颖愤愤不满地瞪着傅夜沉，眼神里的忧伤与愤怒，和那天他强上她的眼神一样。
<br/>
<br/>        如果可以，他一定会吻上她，吻到她不会再说不该说的话为止。
<br/>
<br/>        但此时此刻，他不能再拥吻她了。
<br/>
<br/>        他的头脑，再清醒不过，甚至时时刻刻地在提醒着他，她是他的弟妹，她讨厌，甚至憎恶他对她的亲热。
<br/>
<br/>        傅夜沉不想惹童筱颖不开心，但是这件涉及命案的事情，她必须听他的。
<br/>
<br/>        “筱颖，我再说最后一遍。这个案子，与你没有任何关系。你若再多嘴，别怪我对你不客气！”傅夜沉疾言厉色道。
<br/>
<br/>        童筱颖只觉鼻子一酸，豆大的眼泪，从眼眶里滚了出来。
<br/>
<br/>        她的眼泪，让他感到无比的心痛。
<br/>
<br/>        傅夜沉慢慢地放开了手，爱怜地看着童筱颖，却不知道要怎么去安慰她。
<br/>
<br/>        童筱颖背靠着墙壁，抬起双手，捂住眼睛，再次失声痛哭。
<br/>
<br/>        现在，不是在这儿逗留的时候。
<br/>
<br/>        这件事情很快会传遍整个学校，楼下也会聚集许多因为好奇而过来凑热闹的学生和闻讯赶来的媒体记者。
<br/>
<br/>        傅夜沉果断地再次将童筱颖横抱了起来，疾步离开了第四教学楼。
<br/>
<br/>        回到傅家大庄园里后，傅夜沉找了家里一个做事最心细入微的女佣照顾童筱颖。
<br/>
<br/>        童筱颖整个人不是在哭就是在傻愣着，最后还是女佣伺候她洗澡换衣。
<br/>
<br/>        女佣从童筱颖的屋内出来后，将童筱颖的情况跟傅夜沉说了后，傅夜沉放心不下，准备自己进屋去照顾童筱颖时，这才想起，筱颖现在最需要的人，不是他，而是阿景。
<br/>
<br/>        傅夜沉凝了凝眉，站在童筱颖的房门口犹豫了许久后，才决定转身去给弟弟傅昼景打电话。
<br/>
<br/>        “阿景，筱颖刚刚在临海大学里遇上了有学生被杀的案子，现在她的情绪似乎不太稳定。她需要你的陪伴。”电话接通后，傅夜沉也不拐弯抹角，直接开门见山地说。
<br/>
<br/>        傅昼景却只是一脸无奈地回答道：“筱颖胆小，看到这种事情，心里肯定会害怕。大哥，你帮我多多开导一下她。我现在在开车，待会到了公司，我还得修改方案。我没法陪她了，所以，筱颖就拜托大哥和大嫂帮我照顾了。”
<br/>
<br/>        “阿景，我觉得，你人亲自回来一趟比较好。”傅夜沉剑眉微蹙。
<br/>
<br/>        傅昼景果断地拒绝道：“大哥，我真的很忙。就这样了，我在开车，真不方便太久接听电话。”
<br/>
<br/>        “好！我明白，那你先忙。”傅夜沉应了声。
<br/>
<br/>        傅昼景直接将电话给挂断了。
<br/>
<br/>        只是令傅夜沉想不明白的是，阿景为什么那么忙？明明他已经将集团里很多项目分派给堂叔冷岩峰去做了，阿景这段日子相对来说应该会比较轻松才对。
<br/>
<br/>        傅昼景不回来，童筱颖又一个人坐在床上发呆，傅夜沉站在门外不敢进去。
<br/>
<br/>        床幔下，童筱颖身上穿着白色睡裙，披散着长发，双手抱膝，下巴抵在膝盖上，目光呆滞地看着前方。
<br/>
<br/>        马智杰死前的样子，还有他死前所对她说过的话，一幕幕在脑海里回放。
<br/>
<br/>        这个男生，只不过是她生命里的一个过客，甚至她和他几乎没有什么过多的交集，但为什么他是为了她而死？
<br/>
<br/>        他为什么要为了她去跟何明旭打架？
<br/>
<br/>        他又不是她的谁谁谁，甚至在她离开临海城后，他们彼此之间都从未再有过任何联系，他又为什么要那么做？
<br/>
<br/>        屋内，童筱颖心力憔悴。
<br/>
<br/>        屋外，傅夜沉默默无闻地守候。
