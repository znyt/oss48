　　彻底解决了巫森之后，苏莫在古树下盘膝而坐，把玩起了手中的空间玉镯。
<br/>　　空间玉镯乃是巫森的空间宝器，自然早已被巫森炼化，其上拥有巫森的意念。
<br/>　　但巫森已死，此空间玉镯，自然便成了无主之物。
<br/>　　苏莫的意念倾泄而出，将手中的空间玉镯完全包裹了起来，而后静静的用意念炼化。
<br/>　　秦不死三人见此，便也在苏莫不远处盘膝坐了下来，打坐调息，等待苏莫。
<br/>　　炼化空间玉镯的过程并不长，仅仅只是一刻钟的时间，苏莫便将空间玉镯炼化，他的意念完全与玉镯融为了一体。
<br/>　　至此，这枚空间玉镯，便成了他的宝器，完全受他操控。
<br/>　　苏莫对于这只空间玉镯，也有了一定的了解，此玉镯和虚界神图的品质差不多，但其上的防御阵法，远不如虚界神图。
<br/>　　意念一探，玉镯空间中的一切，全部出现在了他的脑海中。
<br/>　　随即，苏莫意念一动，便将一人从玉镯空间中放了出来。
<br/>　　……
<br/>　　玉镯空间，方圆三百余里，是一片平坦的平原，没有高山、也没有河流，并没有神图空间那般景色。
<br/>　　在平原的中央，耸立着一座雄伟的宫殿，这座宫殿通体是由青钢岩打造，占地千顷，大气滂湃。
<br/>　　而此刻，宫殿的四周，聚集着大量的武者，有黄泉魔宗的武者、有阴阳神宫的武者、还有北堂世家的武者。
<br/>　　这些人密密麻麻，人数不下十余万人！
<br/>　　其中黄泉魔宗的人较少，阴阳神宫和北堂世家的人居多。
<br/>　　宫殿中。
<br/>　　主厅之内，有百余名强者在座。
<br/>　　这些人中，有很多苏莫熟悉之人，巫山、皇甫擎、阎泰、二魔老、阳鼎天、殷棕等等。
<br/>　　所有的人都是面带笑容，红光满面，因为他们即将离开苍穹世界，去往那外面那璀璨的世界。
<br/>　　所有人都充满了向往，所有人都是心情愉悦。
<br/>　　就算是家族被灭的殷棕，心情都很好。
<br/>　　“诸位，据古籍记载，外界广阔无比，有亿万星辰、百万种族，天才如云强者如雨，真是期待啊！”阳鼎天眸中精光四溢，微笑着说道。
<br/>　　“阳兄，我们的实力，到了外界也就是个普通武者而已，努力生存下去才是正途！”北堂世家家主北堂平华道。
<br/>　　北堂平华为了这一天，也是付出了不小的代价，他北堂世家族人众多，但很多都舍弃了，只带走了一部分高手。
<br/>　　“待离开了苍穹世界，我等再和巫兄商议，具体怎么安排！”阳鼎天道，他脸上的笑容也收敛了下来。
<br/>　　他们还是面临一个问题，那就是巫族放不放他们离去！
<br/>　　虽然有巫森这层关系，他们应该都能活命，但怕是不会让他们轻易离去。
<br/>　　“以我们和巫兄的关系，想必巫族不会为难我等！”殷棕沉声说道，叹了口气，他继续道：“唯一比较遗憾是，没能亲眼看到苏莫这个孽畜死亡！”
<br/>　　殷棕对于苏莫的恨，即便是江河之水也无法洗刷，不能亲眼看着苏莫被杀死，让他感觉颇为遗憾。
<br/>　　“哼！试炼者降临，苏莫就算是有十条命，也是死路一条！”阎泰冷笑一声说道。
<br/>　　巫山端坐在阎泰的身边，面无表情，眸中却是闪过一丝隐晦的冷意。
<br/>　　阴阳神宫和北堂世家的人，待出去之后，若是乖乖的成为奴仆，或许还有活命的机会，不然的话，只有死路一条。
<br/>　　当然，此事他自然不会说出来。
<br/>　　就在此刻，巫山神情一怔，因为他突然感觉到一股空间之力将他包裹住了。
<br/>　　下一刻，巫山还未反应过来，身形便瞬间消失无踪。
<br/>　　“这是……！”其他人见此，均是神情一怔，不过也并未多想，应该巫森将巫山放了出去。
<br/>　　外界，古树下。
<br/>　　光芒一闪，巫山的身形凭空出现。
<br/>　　乍一出来，巫山还有些愣神，待看清眼前的苏莫，他顿时面色大变。
<br/>　　“苏莫！”巫山惊呼出声，随即，他脚下一动，就欲逃离此地。
<br/>　　唰！
<br/>　　然而，他的身形还未飞起，苏莫如铁钳般的手掌，便已经闪电般的捏在了他的脖子上。
<br/>　　嗬！
<br/>　　巫山顿时被捏的双眼翻白，口中嗬嗬作响，一身的玄力都差点被捏散了。
<br/>　　他心中惊恐欲绝，全身都在颤.抖。
<br/>　　怎么会这样？
<br/>　　这是怎么回事？
<br/>　　父亲呢？
<br/>　　巫山的心沉到了谷底。
<br/>　　“三位前辈，如何能灭杀此人的神魂？”苏莫向秦不死三人问道。
<br/>　　神魂便是武者的灵魂，武者精神意念的源泉，无形无色，深藏于识海之中。
<br/>　　他虽然知道怎么灭杀，但为了不伤害李峰的肉身，他还是问了一句。
<br/>　　“他的修为和实力比你弱的多，精神力也很弱，你直接用精神力侵入他的识海，绞杀他的神魂便可！”秦不死道。
<br/>　　苏莫点了点头，随即心神一动，精神力直接化为一柄尖刀，冲进了巫山的识海之中。
<br/>　　巫山顿时身躯一震，面上显出痛苦之色。
<br/>　　巫山的识海内，是一片朦胧的世界，一团巨大的绿色云雾，悬浮在虚空中，这正是巫山的神魂。
<br/>　　神魂是武者真正的生命，神魂若灭，必死无疑。
<br/>　　而武者到了武帝境之后，神魂能和武魂融合，形成战魂，才能彻底脱离肉身。
<br/>　　苏莫的精神力进入巫山的识海，不仅看到了巫山的神魂，还在识海的角落里发现了另一团神魂。
<br/>　　这团神魂很弱小，但却仿佛和整个识海相连，其上传出的气息，苏莫非常的熟悉，正是李峰。
<br/>　　咻！
<br/>　　毫不犹豫，苏莫精神力如刀，瞬间便向巫山的神魂斩去。
<br/>　　嗤！
<br/>　　没有任何悬念，巫山的神魂瞬间便被苏莫的精神力撕裂，一分为二。
<br/>　　而后，苏莫的精神力不断的绞杀，几乎是眨眼之间，便将巫山的神魂绞成了碎片，随即缓缓消散。
<br/>　　神魂被绞杀，巫山的身躯顿时软了下去，彻底没了声息。
<br/>　　苏莫将精神力收了回来，立刻将巫山缓缓放下，靠在了大树边。
<br/>　　随即，苏莫便盘坐在巫山的身边，静静的等待了起来。
<br/>　　巫山的神魂已灭，李峰的神魂很快就能掌控这具身体，而后苏醒过来。
<br/>　　果然，过了约莫半刻钟之后，巫山的双眼缓缓睁开，是李峰苏醒了！
