　　也是这一役，修罗的危险程度直接提升到了SSS级，排名甚至超过了大部分成名已久的SSS喰种。
<br/>　　可惜，因为时间太过于仓促，CCG能做的准备并不多，除了原本负责这次行动的有马贵将之外，也没有来得及调动其他高手。
<br/>　　否则的话，还真的很有希望将修罗永远留在那座大厦里。
<br/>　　“一件很容易的事情。”翘起了二郎腿，刘哔放开了男孩，十分轻松的开口：“告诉我和修一族的地址。”
<br/>　　他花费了这么多心思来这里，目标可不仅仅是一个区区特种搜查官，和修一族，才是他最终的目的。
<br/>　　“如果我不知道呢。”丸手斋挪动了一下脖子，脸上的表情有些无奈。
<br/>　　作为CCG的高层，他自然清楚和修一族是怎样的存在。
<br/>　　修罗要和修一族的地址，肯定也不会怀着什么好心思，如果两边冲突起来，不知道会对东京造成多大的影响。
<br/>　　虽然修罗的危险程度被标记为SSS，但所有知情的人士都不会把他当做一般的SSS看待。
<br/>　　天知道这家伙爆发起来，将会有多么恐怖。
<br/>　　“不知道的话，那你可能就没什么利用价值了，不过，和修政和你这么熟悉，你不可能一点消息都提供不了吧。”
<br/>　　顿了一下，刘哔脸上泛起一抹残忍的笑容：“你要是不肯告诉我，那我可不能担保将会对你和你的儿子做什么，毕竟你知道的，我是喰种嘛。”
<br/>　　“在13区杭山顶部的别墅区里，一个叫白日庭的地方，那是我唯一知道的和和修家族有关系的地方。”心中纠结了片刻，丸手斋叹了口气，语气有些挣扎的回答道。
<br/>　　“和修政住在什么地方，我不知道，他每次来都是直接出现在总部。”
<br/>　　想了一下，丸手斋补充了一句，他说的的确是实情，和修家族十分神秘，就算他是CCG的特等搜查官，也根本了解不到太多的消息。
<br/>　　若有所思的点了点头，刘哔查看了一下时间，现在是早上七点多钟，如果自己速度快一些，大概能在八点左右赶到杭山。
<br/>　　这个国家的人，基本上都在八点左右这个时间出门，在此之前，街上的行人并不算多。
<br/>　　毕竟他现在的状态是被通缉，小心一些，也能避免很多麻烦。
<br/>　　万一要是引来了军队，那也是一件相当厌烦的事情。
<br/>　　击杀普通人，对他并没有什么好处，他必须猎杀能量体才可以获得积分。
<br/>　　“我会带着你儿子去，如果你的消息不准确，那你应该知道后果。”
<br/>　　沉吟了一声，刘哔蹲下身来，拍了拍丸手斋愤怒的脸蛋，随后抓起一边不知所措的男孩，从别墅中走了出去。
<br/>　　他不能判断出，丸手斋给他提供的是不是假情报，但至少也存在一定的真实性，这就够了。
<br/>　　有了这条线索，再想顺藤摸瓜的将和修家族给牵扯出来，就比一开始要容易许多了。
<br/>　　用手机规划了一下路线，刘哔挟着男童，捂着他的嘴巴离开了别墅区。
<br/>　　片刻之后，一阵轰鸣声响起，正在巡逻的护卫队们一惊，连忙向爆炸声传来的地方赶去。
<br/>　　……
<br/>　　13区，杭山山顶，白日庭内。
<br/>　　十几个身穿黑色制服的少年，正在一名老师的带领下，进行着体能训练。
<br/>　　这些少年的年龄相差不大，基本上都是十二三四的样子，但他们训练的内容，却足以让最为严苛的特种部队汗颜。
<br/>　　更加诡异的是，这些少年脸上的表情，仿佛一个模子里刻出来的一般，哪怕额头上全是汗水，却还是一副面瘫似的样子。
<br/>　　这种感觉，和有马贵将有些相似，但好像又差了一些什么东西。
<br/>　　整个大厅内，只有器材碰撞的声音，一个老师模样的中年人旁边看着，一言不发，时不时皱起眉头，用手中的钢鞭纠正着少年们的错误。
<br/>　　“你好，请问这里是白日庭吗？”
<br/>　　突然，一道平和的声音响起，众人纷纷停下手中的动作，看向声音传来的方向。
<br/>　　挠了挠头，刘哔站在门口，觉得有些诡异，正常来说，他们不应该质问自己是怎么进来的吗。
<br/>　　和他预先想好的剧本，有些不太一样。
<br/>　　“额，请问你们这里的负责人是谁？”
<br/>　　刘哔围着人群看了一圈，目光停留在老师模样的中年人身上。
<br/>　　作为这里唯一的成年人，多半也是负责人了，他进入的时候就查看过，整个别墅也只有眼前的这一点人。
<br/>　　但出乎他意料的是，众人仿佛是没有听到他的问题一般，依旧面无表情的盯着他，目光如同一群死人。
<br/>　　刘哔心里不免有些发毛。
<br/>　　这种诡异的处境，他还是第一次遇到，一时间颇有些尴尬。
<br/>　　“呃，我是来找你们负责人有些事情，请问负责人能出来谈谈吗？”
<br/>　　咬了咬牙，刘哔决定再问最后一次，如果这些人在不回答，他就要大开杀戒了。
<br/>　　他想，如果自己杀掉一些人，总能将负责人给逼出来吧。
<br/>　　但众人并没有给他机会。
<br/>　　话语刚刚落音，人群终于有了动作，中年男子向前走了几步，来到了刘哔面前。
<br/>　　“杀了有马贵将的那个喰种？”
<br/>　　两人互相对视，虽然是疑问句，但中年男子的语气压根没有半点起伏。
<br/>　　耸了耸肩，刘哔并没有否认，自己的资料，早就被大众所熟知了。
<br/>　　“宗太（旧多二福）也是你杀的？”
<br/>　　“……”
<br/>　　“伊丙入的死也和你有关系？”
<br/>　　“不不不，那可和我没什么关系，我记得，好像是宗太杀了他。”
<br/>　　听到中年男子将伊丙入的死算在自己头上，刘哔立即矢口反驳道。
<br/>　　那家伙的死，和他真没有什么关系，反正原著中就应该死去了，他也没改变什么。
<br/>　　不可置否的点了点头，中年男子并不信任刘哔的说法。
<br/>　　不过，这个问题并不重要。
<br/>　　或许单独拿出来看待，是个很严重的事情，但和有马，宗太放在一起，就不没那么重要了。
<br/>　　反正，这个人一定要死！
<br/>　　“你杀了我三名弟子。”目光中闪过一抹狠戾，中年男子的语气顿时充满了杀机。
