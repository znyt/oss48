　　再下面一条，让叶少阳有点吃惊：珞珈山方丈秋真禅师退位，由他三个弟子一起分管门中事务，纸上写了这三个弟子的名字：风扬，枚月，最后一个居然是腾永清！
<br/>　　“秋真禅师昨晚也在？”叶少阳问曲波。
<br/>　　“他这次没来，我本来也有些纳闷，打电话问了永清，他说是秋真禅师见各大门派掌教都退位让给年轻人，本来就萌生退意的他干脆也趁着这一波，把位子让了出来。三个弟子，分别管内务外务和修炼，将来的掌门会从三人中选一个。”
<br/>　　曲波跟腾永清的友谊，是不久之前，在星月奴的幻境之中结下的，那段时间两人一起在城墙上布防，比较投缘，出来就加了微信，保持着联系。
<br/>　　“我听永清说，本来靠资历和实力，他是没机会入选的，但秋真禅师考虑到他是捉鬼联盟成员，一方面是看你面子，一方面也是希望他能带着珞珈山，多跟你合作，争取能多立功劳，光耀山门……”
<br/>　　叶少阳笑了笑，内心也是暗爽的，四宝，吴嘉伟，腾永清现在都能掌控一个门派了，自己在法术界说话的分量也会越来越重。
<br/>　　他接着往下看，还有很多门派都换了一把手：
<br/>　　鸡足山的紫云法师退位，弟子陈楷涛即位；
<br/>　　三清山：天明真人退位，弟子马舒俊；
<br/>　　……
<br/>　　一排看下去，竟然有十来个门派都换了掌门，年轻弟子即位，叶少阳有点震惊了，“竟然有这么么多！”
<br/>　　“都是昨晚在的那帮人，连慈心师太都退位了，他们面子上过不去，一个个也都引咎退位了。”曲波道，“这些新任掌门，全都回了我们的信息，表示拥护掌门师兄你担任法术界盟主，听从号令。”
<br/>　　叶少阳缓缓点头，道：“曲师弟，你想办法给我弄一份完整名单，所有能联系到的门派都要，写上现任掌教的姓名和联系方式，我好心里有数。”
<br/>　　曲波答应下来，询问叶少阳下一步计划，叶少阳表示自己天亮就得走，去云台山跟道风汇合。曲波也没问什么事，只问他要不要联系云台山的方丈，叶少阳想了下还是不要了，关于太阿剑的事，他不想声张，免得节外生枝。
<br/>　　“这是龙虎山的账簿，上面记着历年山门收支的数目，请掌门师兄过目。”曲波将一本账簿递上来。
<br/>　　叶少阳拿过来，翻到中间，一张银行卡露出来，问曲波这是啥意思。
<br/>　　“这是掌门的工资卡啊，每个月固定打到这张卡里，给你平时零花的，密码123456，用龙虎山风景区的名义开的户，要是花完了你跟我说，一万以内，我直接能给你，超过三万，就得找长老会签名盖章了。”
<br/>　　一个月三万工资！擦，这龙虎山掌门可真是富啊。比茅山可强多了。
<br/>　　不过面子上还是推辞了一下，曲波直接把银行卡塞到他口袋里，“龙虎山家大业大，光是风景区的收入，每个月就能分到不少，大家都是按比例分红，你是掌门，比例高一点，我一个月也有近万呢，你就拿着吧，你拿了，否则大家都不好意思拿。”
<br/>　　叶少阳满心欢喜收下。他是还有点积蓄，但最近半年基本上没干任何挣钱的买卖，坐吃山空，心里早就着急了（茅山是不指望的，因为封闭，景区的人比龙虎山少的多，而且什么都在建设中，到处用钱，自己从没从茅山真正拿过钱），如今有这一个月三万的固定收入，倒是没有后顾之忧了。
<br/>　　怪不得张无生以前日子过得那么滋润，原来这掌门不是白当的。
<br/>　　“我先说一点啊，我对龙虎山，与茅山一视同仁，不偏不倚，但我只把自己当代理掌门看，将来还是要还位给张家人的。”叶少阳很严肃地告诉他，其潜台词就是：自己终究还是茅山弟子。
<br/>　　这骨子里的出身观念，是改变不了的。
<br/>　　曲波表示理解。
<br/>　　叶少阳翻了翻账本，还给了曲波，两人又聊了一会有关门派建设方面的事，曲波为他订了第二天去台州的机票，选了个中午的飞机，之后便告辞了。
<br/>　　叶少阳打开微信，邀请吴嘉伟和四宝一起视频，问了他们即位的情况，两人都有点无奈。
<br/>　　吴嘉伟还好，跟几个师叔商量好了，门中的事务还是交给他们几个打理，自己立刻就赶回石城。四宝是真有点委屈，毕竟当上方丈，凡事都要以身作则，肉是不能吃了，酒也不能喝了，这些都能忍，就是以后跟王旭雯约会要隐秘的多，尽量不要被人发现，不然影响五台山的名声。
<br/>　　叶少阳笑道：“你还真是死性不改啊，都当上方丈了，还不忘记跟妹子约会。我问你，你打算将来咋办，总不能一直地下联络吧。”
<br/>　　“目前来说吧，这个方丈的身份还有点用，至少有些事能帮你，等终极一战结束，我就退位呗，谁爱当这个方丈就谁来当，我还俗去跟雯雯结婚去。”
<br/>　　吴嘉伟道：“我也这么想的。”
<br/>　　“什么终极一战？”
<br/>　　“铲平太阴山呗。”四宝笑了笑，“少阳，我们终归要退休的，现在这生活虽然刺激，毕竟是玩命，总不能这样玩一辈子。把太阴山当做收官之战，是最好的选择了，到时候大家一起退休，像郭老那样，结婚生孩子，过正常人的生活。”
<br/>　　吴嘉伟跟着点头，问他什么时候展开行动。
<br/>　　叶少阳心中突然涌起一丝落寞的感觉，问他们：“你们做好准备了吗？你们不怕死？”
<br/>　　四宝道：“三叩九拜都过来了，这一仗不打，这辈子都心痒痒的。也别说什么为了天下苍生，咱们就是干这个的，这一票非得干了不可。”
<br/>　　叶少阳叹了口气道：“只怕是去了万一回不来……”
<br/>　　“所以要快点行动啊！”四宝催促道，“要是死在这一战中，也算死得其所吧，趁着我跟雯雯还在初级阶段，我死了她还能找别人，再拖下去，我舍不得她，她也舍不得我了。”
