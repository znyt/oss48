<br/>　　沁儿看到分身在月沁园门前，面带疑惑的问道。

<br/>　　“沁儿姑娘，我一直在等你！”分身长身而起，微笑着说道。

<br/>　　沁儿闻言一怔，随即笑着问道：“你不会是想要拜师吧？”

<br/>　　“嗯！”分身点了点头，对方怎么想无所谓，只要能见到玄天神王就行。
<br/>　　骆妤的脸色难看了下来，她一直认为苏莫另有所图，现在居然想要拜玄天神王为师，这要是成功了，身份地位，暴增十倍啊。

<br/>　　因为，他还有一个哥哥，他父亲除了培养她之外，只会关注她哥哥的修炼。
<br/>　　“不，我就只见你父亲！”
<br/>　　分身摇了摇头，他心中清楚，沁儿的父亲，不仅实力强大，更是宫主之子，最能代表宫主的意志。

<br/>　　“哼，你倒是胃口不小！”骆妤冷哼一声，她心中嗤笑，这段惊天真是异想天开。
<br/>　　她明白段惊天为何会有这种打算，玄天神王是宫中的三长老，又是宫主之子，攀上了玄天神王，那简直是得道升天了。
<br/>　　“这样啊！”
<br/>　　沁儿略一沉吟，道：“好吧，我带你去见父亲，不过，你能不能成功，就看你自己的了！”
<br/>　　“多谢沁儿姑娘~！”分身闻言一笑，心中松了一口气。

<br/>　　毕竟，只要本体有吞噬元神在，他就能吞噬到其它强大的元神。
