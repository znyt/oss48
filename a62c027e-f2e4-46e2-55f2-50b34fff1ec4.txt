　　这匹马像是听懂了叶少阳的话，撒欢地甩开四个蹄子跑了。
<br/>　　叶少阳发现另一匹马在看着逃走的那一匹，拍了拍它的脑袋说道：“别羡慕，回来我也给你放生。”
<br/>　　于是叶少阳让两个土匪在前面赶路，自己后面牵着马，一路下山，路上两个土匪愁眉苦脸，央求叶少阳放过他们，甚至愿意给他做马仔，被叶少阳往屁股上踢了一脚之后，立刻也就老实多了，也不敢再多说了。
<br/>　　一路上叶少阳想起什么事情就问他们，后来想起了那位孙司令，虽然自己完全不知道他是谁，但好歹人家也算是“救”过自己一次，想到当时三当家的听见孙司令名字时候的那种样子，顿时对这个孙司令产生了一些兴趣。
<br/>　　刘老头之前跟叶少阳说的，都是孙司令当土匪之前的情况，对于他当土匪之后的事，就不太知道了。被他抓住的这俩土匪，提起孙司令，也是敬佩得不得了，跟叶少阳说了很多关于孙司令的奇闻异事，其中最牛逼的就是怎么从一个败军之将到落草为寇，然后等于是白手起家，在百花山建起山寨，又怎么样在十年间，靠着武力征服了周围大大小小的山寨，成为江西境内最大的一股土匪。
<br/>　　不过他能成为江西三山五湖的总瓢把子，人强马壮是一方面，不过也不是全靠这个，毕竟江西这么大，假如远地方有不服从的，孙司令也不可能调兵来打，毕竟是土匪，不能像军队那样在街市地面上横行无忌。
<br/>　　孙司令之所以被所有土匪方奉为老大，主要还是因为做事仗义而公平，会保护一些弱小的势力，有一些山寨之间的矛盾后斗争，也会出来调停，态度不偏不倚，让双方都说不出什么来。久而久之，孙司令的威望在江湖上如日中天，没人敢不服从。
<br/>　　听两个土匪说了这么多关于孙司令的事，在叶少阳心中，这位土匪头子更是加深了传奇性，甚至产生一种想要一睹真容的念头，当然也只是想想罢了。
<br/>　　二十里山路，因为是下山，走的也快，不过很多地方根本就没有路，弯弯绕绕的，如果没有人带路的话，很难找到路上山。
<br/>　　叶少阳一路上各种强记，有些地方实在容易迷惑的，就丢下一张灵符，灵符上都画着简单的符文，只要是十二个时辰之内激活，在方圆几十米内都可以感知到所在，在可用的情况下，用来当路标也是再好不过。
<br/>　　在叶少阳不断踢屁股的催促之下，三人只用了两个小时左右就下了山，来到一条大路上。两个土匪表示，再往前走不远就是县城，叶少阳想着让这两个人是去自首的，这么绑着过去的话实在是不像，于是给两人松绑，这两土匪见识过叶少阳的手段，再加上有手枪在他袖子里藏着，根本不敢耍什么花招。
<br/>　　叶少阳牵着马，推着两人，往县城方向走，又走了有半个小时，到了城门下，有士兵在城门下把守，所有过往的人都要检查居民证，才给放行。
<br/>　　叶少阳三人走到城门下方，立刻就被拦住了，要检查居民证。
<br/>　　“同志……不对，那啥，伙计们，这两个人没有居民证，他们是二龙山上的土匪，过来自首的。”
<br/>　　这一队守城的士兵听他这么一说，立刻就傻了：从来只听说土匪来县城干坏事，还是头一回见到送上门自首的土匪，嫌活的腻歪了？
<br/>　　“你们真是土匪？”守兵中的一个头头睁大眼睛打量着叶少阳三人说道。
<br/>　　到这一步，也没什么好隐瞒的了，只能有什么说什么，争取宽大处理，两个土匪只好点头承认是土匪，并且把袖子挽起来，给他们看右胳膊上的纹身，两人的大臂上，纹着两头龙的形象。
<br/>　　“果然是二龙山的！”
<br/>　　这县城因为跟二龙山离得近，虽然县城有守兵把守，土匪们不敢光明正大进城，不过倒也经常骚扰周围的村庄，有时候也会摸进县城干一些坏事，因此县城对这般土匪深恶痛绝，知道他们的特征。
<br/>　　“可是，你们……为什么要来自首伏罪？”守兵们对两个土匪的这一反常行为完全不能理解。
<br/>　　“他们想做好人，愿意交代罪行和山上的底细。”叶少阳替他们解释道，两人连忙点头。
<br/>　　几个守兵更是不解，不过既然是送上门自首的，总不能不管，而且抓到土匪也算是功劳一件，当下找人把他们押送县衙。
<br/>　　叶少阳在一旁看着，心中十分满意这个结果，不料居然有士兵上来连他也想要一块绑。
<br/>　　叶少阳只好解释自己不是土匪，给他们看了胳膊，还有自己的居民证，表示自己是在路上遇到两个土匪，听他们说想要自首，正好进城，所以跟他们一起来了。
<br/>　　守兵们听完他的话，简直不知道说什么好，这种事情在他们生平简直就是头一遭。
<br/>　　叶少阳告别这些守兵之后，牵着马进了城，因为时间还早，自己既然来都来了，干脆在县城里逛了起来，身上也有些闲钱，于是买了一些东西，中午吃了一碗山西刀削面，感觉味道比一百年后的那些加盟店卖的要好吃得多。
<br/>　　在县城里一直呆到下午，叶少阳牵着马离开了县城，按照之前来的路，一路策马奔腾，又回到了二龙山上，因为担心遇到土匪，在半途中就把马给放了，然后摸黑上山，走到快到破庙附近时，见两人打着火把下山，叶少阳躲到附近的树林里，看着两人走近，也是土匪打扮。
<br/>　　两人一边走一边抱怨两个名字，叶少阳听了一会，明白他们抱怨的正是那两个被自己强行自首的土匪，因为下山采购，到现在还不回来，大当家的生怕耽误了晚上的婚礼，因此又派了两个人下山。
<br/>　　对这两人，叶少阳没做任何表示，如果自己把这两人再怎么办了，到了晚上他们不回去交差，大当家的肯定会怀疑了。
