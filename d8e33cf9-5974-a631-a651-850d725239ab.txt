　　洪荒之中，人族与巫族在大战，尸横遍地，血流成河，苍穹洒雨，乌云遮天。天地之间，劫煞之气遍布。
<br/>　　大地之下，数万里深处的一个地下空间中，一座三丈大小的黑色血池，正如同无底洞一般在吞噬洪荒大地之上的无尽鲜血，在这血池之上，还散发着无尽魔威，不断有精血被炼化，吸收，甚至一些劫气、杀气、煞气，它一样能够吸收。
<br/>　　里面时不时的甚至能够看到有怪物在不断翻滚，咆哮，还有厉鬼恶魂，在不断惨叫，可惜，无人能够听到。
<br/>　　太华山，劫魔殿中，天地间无尽劫气、煞气被牵引过来，不断被万劫魔尊吸收、炼化，殿内深处，一个强悍的气息正在酝酿之中，也许下一刻，便会如火山喷发一般爆发出来。
<br/>　　“哈哈哈！巫、妖、人三族大战，正好成全了本尊，哈哈！打吧，战吧，你们战得越欢，战得越惨，本尊越是高兴。本尊开始有点理解魔祖罗睺当初的行为了，哈哈！”
<br/>　　笑声中带着残忍，带着一股疯狂，一股渗人，更有一股坚决之意。
<br/>　　……
<br/>　　“轰！”
<br/>　　玄天再次惊天一掌拍下，镇压共工祖巫的黑水界，同时，他一个虚步，闪过帝江的空间震荡，对着他一拳轰出。拳头朴实无华，好似无任何奇异、华丽之处，但却蕴含着无上威力，带着惊天的力量爆发。
<br/>　　“嗡！”
<br/>　　玄天全身气势恢宏，如圣临尘，又如大海一般宽阔，无涯，浩瀚。带着一股神圣、浩然大气，震人心魂。
<br/>　　“轰隆隆！”
<br/>　　一阵阵巨大声响，代表着众人的战斗程度，道法显现，神通惊人，各种玄光飞舞，符文闪现，惊天动地。
<br/>　　武祖玄天五指一伸，将手一招，从五指缝隙之中射出五道长龙似的晶光，每一条晶光，都长达千百丈，散发出切割虚空，扭转时间的威严。
<br/>　　在晶光的周围，一声声禅唱出现着，一个又一个世界似乎在其中衍生而成，随后虚空中出现许许多多条天龙的虚影，这些天龙，似乎在咆哮，在怒吼，在飞舞。
<br/>　　“嗡！”……
<br/>　　一股奇怪的力量，立刻撕裂了玄天眼前的虚空，震散了九天之上的乌云，露出煌煌大日，破尽无尽乌云，照射下来。
<br/>　　而玄天身边的晶光，则横扫十方，化为千百条龙形，到处游荡，这些龙形虚影，口中喷射出金雷，张口之间，一个接着一个的大霹雳震荡着，将天地都震荡得摇动起来。
<br/>　　看到这一幕，无论是祖巫，还是人族，亦或是关注此战的大能们，都感觉道好像处在了世界崩灭的边缘。
<br/>　　万龙炼界大仙术！
<br/>　　传闻，远古年间，一万条天龙，摆出一个大阵，能够炼化一个世界，威力非凡。
<br/>　　而现在，玄天单手之间，就施展出了这样一个大神通，可见神通已经不可估量，用日月星辰的数量来衡量他的法力，都小了一点。
<br/>　　这是玄天用三十三种法则神通，凝聚出来的一种大仙术神通。
<br/>　　武祖玄天五指连连弹动，无限虚空，在他的之间衍生，顿时在无限空间的尽头，看见一个小小的世界，大有方圆十万八千里，宛若一国，山川河流，山石草木，种种生灵，光怪陆离，应有尽有。
<br/>　　“镇压！”
<br/>　　先天冷哼一声，抬手之间，穿过无尽虚空，向着时间祖巫烛九阴镇压下去。
<br/>　　在他手中，万条晶龙，呼啸着冲入虚空之中，围成一方大阵，向着烛九阴镇压下去。同时，那方小世界将烛九阴笼罩在其中，晶龙环绕世界，将之团团缠绕住，金雷震荡，口中更是喷出一股股龙炎，要将烛九阴连着整个世界一同炼化。
<br/>　　“该死！”
<br/>　　时间祖巫烛九阴一声怒吼，察觉到了自己的不妙，顿时时间法则震动，扭曲时光，想要令时光倒退，逃出牢笼。可是，玄天又怎会如他所愿呢。
<br/>　　“神通，封印！”
<br/>　　玄天一声冷喝，五指之间，流转出一种九彩氤氲的法则道力，化为一张虚幻的大网，一点一点将烛九阴笼罩，封印。
<br/>　　烛九阴怒吼，想要挣脱，可惜，他的神通已经被封印了，只能施展肉身力量，但凭他的力量，又不足以粉碎这万千条天龙的包围，只能任由这些天龙将他炼化。
<br/>　　“可恶，人族玄天，你将老二怎么了？我要你死！”
<br/>　　帝江见此，目眦尽裂，怒气升腾，火之祖巫祝融刚死，难道还要看着烛九阴死亡吗？他一声暴怒，挥手间爆碎空间，跨过无尽空间，想要前来解救烛九阴。
<br/>　　可惜，空间法则玄天一样会，他若存心将烛九阴流放到虚空之中，帝江一样救不到他。虽然能够看见，仿佛就在眼前，但实际上是隔着无尽虚空。
<br/>　　“风雨雷电，霜寒冰雪，四季天气，轮转不休！”
<br/>　　就在此时，天气祖巫奢比尸操控四季天气，风云雷电之力，向着玄天侵袭而来。
<br/>　　“哼！天气之力，小意思！”
<br/>　　玄天满脸不屑，看着向自己侵袭而来的各种天气变幻之力，他伸出一手，黑光衍生，黑洞显现。
<br/>　　“大吞噬术！”
<br/>　　在玄天的手中，出现一个黑洞，黑洞逆转，涌现出强烈的吞噬之力，将那些天气之力瞬间便吞噬一空。
<br/>　　“还给你！”
<br/>　　玄天又一甩手，黑洞方向逆转，顿时再次从其中传来一阵强烈的倒吐能力，之前被吞噬的那些天气之力，瞬间便被流放出来，化为一股巨大洪流，倒冲击在奢比尸身上。
<br/>　　“啊！”
<br/>　　奢比尸惨叫，被天气之力侵蚀，他的身体开始腐烂，虚弱。
<br/>　　“塔来！”
<br/>　　玄天大喝一声，传遍洪荒，震动苍穹。
<br/>　　“嗡！”
<br/>　　顿时，虚空一阵波动，从首阳山方向飞来一座高大的、闪烁着金光的九层宝塔，正是玄天的武塔。
<br/>　　“轰！”
<br/>　　武塔不偏不移，正好落在奢比尸上空，塔身巨震，化作一座高达数千丈的巨塔，犹如泰山压顶一般镇压下来。
<br/>　　奢比尸因为身体被虚弱的原因，来不及躲避，便被巨大的武塔镇压进去。
<br/>　　“轰隆隆！”
<br/>　　这时，金光闪闪的巨塔内传来声声轰鸣声，玄天知道，那是因为奢比尸在轰炸塔身，但他并没有多管，因为他知道奢比尸奈何不了这座武塔。
<br/>　　果然，瞬息之间，那武塔之上就闪现出各种神秘符文、各种神兽刻图，日月星辰刻图等，那些符文闪烁之间，使得宝塔变得更加神秘，更加浩瀚。
<br/>　　在那些符文之中，好像有着一股神秘的力量，能够将塔内的生物镇压封印，而且还能不断地削弱其实力。
<br/>　　果然不到一时三刻，那塔内就再无任何声响传出，玄天知道，那是因为因为奢比尸被镇压了。
<br/>　　他从奢比尸被镇压之后，便不再关注武塔了，而是将目光投向水之祖巫共工。
