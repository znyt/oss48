　　袁守诚见李承乾脸色发青的样子，知道他已经到了暴走的边缘，故而理智的摆摆手说道：“贫道没有其它意思，陛下多虑了！”
<br/>　　“话不说不明，朕最讨厌的就是把话说一半的人，袁守诚，你最好把话说清楚。”李承乾这个时候已经拼了，决定赌一把袁守城到底对自己的事情知道多少，如果这老道士知道的太多，那就不好意思，只能送他去见三清了。
<br/>　　看着李承乾眼中的杀意，袁守诚心中也是暗自心惊，不过脸上却没有表现出来。
<br/>　　在扶着木头栏杆站起来之后，缓缓说道：“陛下，当年杜相的事情您还记得吧？”
<br/>　　“你指的是什么？！”李承乾漠然问道。
<br/>　　袁守诚沉默了片刻，像是在考虑要不要说，最后长叹了一声说道：“陛下，贞观四年初杜相病重，按说本应神魂归位，然殿下展神医妙手力挽狂澜，救杜相于垂危，这一点贫道没有说错吧？！”
<br/>　　李承乾的脸色再次一变，袁守诚说了一大套话，表面上看似在提醒他杜如晦当初重病的事情，但是深知内情的他知道，这袁老道所说的一切并不是表面上那么简单。
<br/>　　而且袁守诚似乎并没有说完，顿了顿之后又继续说道：“还有，秦琼秦大将军的事情，陛下不会不知情吧？还有皇太后、长乐公主，您真的以为这些都是偶然？”
<br/>　　“够了，你到底想说什么？”李承乾已经没有耐心再听袁守诚扯蛋，被他点出来的四个人，每一个都是在很久之前就应该死去的人，结果到了现在只有老杜一个人死了，而其他三个人还都活的好好的。
<br/>　　这种事情李慕云以为只有天知地知自己知，最多崔钰也知道一点。
<br/>　　可是万万没想到，袁守诚这老道士竟然也知道，而且一个一个的全都点了出来，就差没有指着他的鼻子说：‘你丫是穿越来的’了。
<br/>　　“陛下，贫道没有别的意思，只是想提醒您，事出必有因，有因必有果，长乐公主的事情固然是我们的错，但是这里却涉及到一个惊天大阴谋。”似乎是为了表达自己所说一切的真实性，袁守诚深深吸了口气：“所以陛下若是想治吾等之罪，最好还是三思而行，莫要因小失大。”
<br/>　　李承乾沉默着，良久没有说话，好像是在判断袁守诚所说的东西里面到底有几分真几分假。
<br/>　　只是什么惊天阴谋之类的事情李承乾是不大相信的，在大唐李承乾相信除了自己，其他人就算是想要搞事情，也搞不出什么惊天大案来，所以他认为这只不过就是袁守诚的一个托词。
<br/>　　想到这里，李承乾冷冷的哼了一声：“袁守诚，不要找借口，阴谋之说朕天天都能听到，若你有真凭实据那就拿出来，否则就不要在朕面前信口开河。”
<br/>　　“陛下若是不信贫道也没有办法，只是这件事的确是另有隐情，非是像陛下想的那么简单。”袁守诚被李承乾步步进逼弄的一点办法都没有，有些话他实在没有办法说的更清楚，可是为了保住那些同门，他又不得不说。
<br/>　　长乐在李承乾心中的地位是什么样的，在大唐官场上几乎没人不知道，袁守诚这个经常与官员打交道的老道士自然也不可能不知道。他的心里有一种预感，如果自己不说出一点东西，李承乾暴走之下真的有可能将他们这些道门之人全部杀光。
<br/>　　而事实果然如袁守诚猜测的一样，李承乾并不相信他的含糊其词，甚至听了他的话之后，连眼都没有眨一下，只是无所谓的笑了笑：“既然你们什么都知道，又是打着为大唐，为朕好的名义，那么说出来又有何防，反正你们都已经对长乐说了，不如再跟朕说说如何？”
<br/>　　“咳咳……”一阵咳嗽声传来，接着是秦英老道士的声音传来：“陛下，可想听一个故事？一个跟大唐有关的事故？”
<br/>　　李承乾扭头看了一眼老的几乎连牙都快要掉光的秦英：“好啊。不过你最好能长话短说，朕的时间有限！”
<br/>　　秦英这老道士当初还给李承乾做过法事来着，不过李承乾是自家人知自家事，当时那种情况下一直把这老道当成一个骗子，只是碍于老头子的好意没有戳穿他罢了。
<br/>　　而现在事过境迁，当年的事情早就已经慢慢被淡忘，秦英这老道也不知道还能不能记起。
<br/>　　只是秦英这老道士也不知道是太老了还是故意装的，对李承乾的话好像根本没有听到一般，自顾自的说道：“相传上古之时，天地初开……。”
<br/>　　李承乾在小白和夜魅的陪同下，站在大理寺的监牢之中，听着秦英用他苍老的声音讲述着一个很久很久以前的故事。
<br/>　　从女娲补天到五色神石，李承乾本以为这老道还能讲出石头里跳出一个石猴之类，但事实让他并没有猜到这老道的思路。
<br/>　　五色神石被遗落凡间之后的事情变的与李承乾所知道美猴王没有任何关系，相反这色神石却是被地府中人得了去，精心炼化之后形成八块玉佩。
<br/>　　这八块玉佩其中的七块代表着北斗之数，另有一块玉佩便成了一块副佩，代表着辅助。
<br/>　　这八块玉佩不知怎么就流落到了凡间，为上古先民所得，后来辗转之下落入诸葛武侯之手。
<br/>　　听到这里，李承乾的脸色变的十分难看：“秦英，你是在调侃朕？！”
<br/>　　秦英老道士摇摇头：“非也，老道只是想让陛下知道，你的七块玉佩并非凡物。”
<br/>　　“那又如何？朕要知道的不是这几块玉的来历，而是你们的阴谋，算计长乐，让长乐身死，到底是为了什么！”李承乾说到最后，整个人几乎已经失控，如果不是小白和夜魅一直拉着他，估计他已经要冲上去殴打那个老的快要没有牙的老道士。
<br/>　　“陛下，难道您还不明白？事有反常必为妖！这八块玉佩是绝对不应该出现在这个世界上的，而它们既然出现了，那就说明……”袁守诚刚刚想要解释一下，可是话说了一半，外面大牢的牢门咣当一声再次被打开，一个人影风风火火的闯了进来。
