　　天下没有白吃的午餐，在自己使用系统的时候，肯定也要去为系统做些什么，这没问题，做任何事情都可以，只要系统能让他活着。
<br/>　　既然选择绑定自己，系统一定会有办法救活他，毕竟也没什么东西愿意浪费时间来哄骗一个死人。
<br/>　　事实上，他也没什么值得人哄骗的。
<br/>　　“宿主体内的病毒，系统已经择优净化，且不会对宿主产生任何负面影响，请宿主自行打开系统面板查看。”
<br/>　　听到病毒已经解决，刘哔心中一松，他最担心的，就是这件事情，只要不会变成丧尸，他就没什么别的要求了。
<br/>　　可是，当根据系统的提示，打开系统面板的时候，刘哔的眼神瞬间凝固了。
<br/>　　这是一个类似于游戏中的人物面板，旁边还有几个代表各种功能的按钮，看起来非常立体，也非常方便简洁。
<br/>　　但是现在刘哔并没有打开那些功能查看，只是盯着自己的属性面板，脸上的表情呆愣愣的。
<br/>　　宿主：刘哔
<br/>　　种族：丧尸
<br/>　　力量：20
<br/>　　精神：5
<br/>　　智力：7
<br/>　　敏捷：4
<br/>　　在他查看属性面板的同时，系统也在一旁介绍着，按照常理来说，普通人的各项指标都不会超过10，但是刘哔之前经过丧尸病毒的强化，力量自然要强大了许多。
<br/>　　可是，对这件事情刘哔并不感到兴奋，力量固然是增强了，但付出的代价，也太过于沉重了。
<br/>　　“系统，你不是说，你已经净化掉了我体内的丧尸病毒吗？可是，为什么我现在的种族还是丧尸？”刘哔声音微颤，下意识的舔了舔自己的牙龈，触到了一根尖锐的獠牙。
<br/>　　冰凉凉的，让脆嫩的舌头感受到一股冰冷，就和他此刻心情一样。
<br/>　　“只是净化掉了病毒中的有害部分，而对于可以提升宿主生存能力的部分，系统并没有权限清除。”
<br/>　　系统此时的声音，听起来有些冰冷，也让刘哔的心情沉入谷底。
<br/>　　“那么，我还会保持人类的外表吗？”稍稍冷静了一下，刘哔问出自己最关心的一个问题，一想到丧尸那满身腐肉的模样，他就忍不住有些反胃。
<br/>　　对于这个问题，系统似乎是不知道该作何解释，沉默了良久，方才回答。
<br/>　　“经系统的判定，丧尸的指甲和牙齿可以增强宿主的生存能力，且因为宿主体内保留了部分病毒，所以眼睛也会是红色，其余的身体特征，没有任何变化。”
<br/>　　停顿了一下，系统继续补充道：“由于丧尸的牙齿和指甲带有强烈的丧尸病毒，故此宿主在和人类接触的时候，最好要小心注意，以免引起一些不好的意外发生。”
<br/>　　听完系统的解释，刘哔长长的呼出一口气，还好，只要外表还是人类就好，反正多出来的那些特征，也不会对他造成多大影响。
<br/>　　瘦死的骆驼比不上一只活着的蚂蚁，再说有着一个丧尸的身份，也能在末世中给予他不少方便。
<br/>　　心情舒缓下来的刘哔，继续将心神沉入系统，仔细研究着那些功能，片刻之后，嘴角逐渐上翘，脸上露出了开心的笑容。
<br/>　　关于系统具体的作用，他已经了解的七七八八了，无非就是可以随机穿梭到某种特殊的位面，来完成系统指派的各种任务，从而获得积分和各种道具。
<br/>　　而积分的作用，用途也十分广泛，可以从商城里兑换各种商品，兑换的种类，从普通的枪械食物，到传说中的神器功法，应有尽有。
<br/>　　最让他感兴趣的，是技能一栏的介绍，关于技能，分为主动技能和被动技能两种，学习的种类和数量没有任何限制，可以通过各种手段来获得。
<br/>　　就如同，刘哔成为丧尸之后，自带的‘不死’技能。
<br/>　　根据系统的介绍，这属于被动技能的一种，功效是‘不死’技能的拥有者，只会在脑部死亡的情况下，才可以彻底死去。
<br/>　　这不光是只有刘哔，所有的丧尸都会有这样一个特效，但区别在于，那些丧尸们都没有系统啊！
<br/>　　对于他而言，这个技能简直就是逆天一般的存在，他之前在商城里面看到过，只要花费相应的积分，就可以恢复任何非致死的伤势。
<br/>　　如此一来，对于有了‘再生’能力的刘哔来说，只要大脑没有受损太过于严重，就不存在任何可以致死的伤势。
<br/>　　也就是说，哪怕是他被打的只剩下了半个脑袋，在有积分的前提下，依旧可以将自己的身体恢复如初，这样看来，这个技能的用处就实在是太大了。
<br/>　　收起系统面板，刘哔心情十分不错，目前为止，他唯一没搞明白的就是任务系统。
<br/>　　一直到现在，系统都没有主动给他下发什么任务，任务栏的地方，却只留着等待触发四个大字。
<br/>　　根据系统的回答，在每个世界中都隐藏着很多任务，奖励和难度也各不相同，而每个任务，都需要特殊的触发条件。
<br/>　　或许是要接触到什么人，或许是需要某种物品，又或者要到达某一处地方，反正所有的一切，都需要刘哔自己来进行摸索。
<br/>　　心中吐槽的一声，他有些埋怨系统给他说的不够清楚，不过对方不颁布任务的话，他也乐的轻松。
<br/>　　现在的当务之急，是要将这个世界的情况给搞明白，究竟来到了一个怎样的世界，心中挂念着自己的家人，他也想知道怎么才能从这个世界中离开。
<br/>　　关于这个方面，系统可是一点消息都没有透露给他。
<br/>　　想到这里，刘哔起身下床，穿戴好了自己的衣服，准备离开这个逼仄狭小的房间。
<br/>　　大概是处于古代，这里人的衣服都有些复古，不太熟悉的刘哔花了很大一会功夫，才将一套麻布粗衣套在了自己身上。
<br/>　　劣质的布料给刘哔微微带来一些不适，但这并不影响他愉悦的心情，临出门的时候，眼角余光瞥到了桌上的小圆镜，不由走上前去，拿起镜子仔细打量了一下。
<br/>　　容貌和先前没有多大变化，只是皮肤更加苍白了一些，眼睛里的红色，像是害了眼病一般，估计也不会有人将他和丧尸联系到一起。
<br/>　　确认一切都没有问题之后，刘哔将圆镜随手一扔，推门走了出去，随后，整个人都变的不好了。
