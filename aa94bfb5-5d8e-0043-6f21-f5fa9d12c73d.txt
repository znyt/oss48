　　包子想凑热闹，也跟着去了。
<br/>　　叶少阳要控制皇帝，跟吴嘉伟和碧清一起留下等着。
<br/>　　小马出去之后，外面的打斗声持续不断，过了有十分钟左右，包子突然飞奔而来，离老远就冲叶少阳喊道：“老大了不得了！小马哥不是那个道士的对手，被缠住了，老大快去帮忙！”
<br/>　　“道士？哪来的道士？”
<br/>　　“呃，就是那个什么将军啊，其实是个道士，端得厉害！”
<br/>　　叶少阳一听，觉得蹊跷，让碧清跟吴嘉伟留下看着皇帝，自己跟包子飞快地往外跑，心中十分惊奇。
<br/>　　虽然自己总是埋汰小马，但不得不说，如今的小马修为之深，在邪物中也早就进入了上层境界，在人间也是没什么敌手，虽然活有点糙，但能压住他的，那毕竟是很罕见的强者了。
<br/>　　除了皇城，叶少阳站在台阶上，老远就看到小马跟一个人打得起劲，附近倒是围了不少士兵，黑压压地将街道都堵死了，但两人对战之时，不断有气流扫过，让他们都不敢近身，只能远远地看热闹。
<br/>　　叶少阳定睛朝战场看去，跟小马战斗的是一个身穿金甲的将军，面白无须，看上去五十来岁，表情有些呆滞，明明是将军，手上却拿着一把拂尘，作法防守，在他头顶上空，悬着一块长条形状的白光，有胳膊那么长，两三根指头粗细，释放着咄咄逼人的白光，在这将军的操控下，不断朝小马打下去。
<br/>　　这白色的长条状能长能短、能弯能直，气势也柔中带刚，十分玄妙，小马虽然修为高深，但手段上根本不是对手，靠着手中的玄铁板砖硬扛着，被耍的团团转。
<br/>　　但对方显然很懂战术，并不着急利用优势，而是追着小马缠斗，积累着优势。
<br/>　　这手段……看上去很熟悉啊，叶少阳皱眉观察了一会，确定这老头用的是纯正的道门法术，心中疑惑，光顾着盯着老头看了，突然包子在边上喊道：“老大，你再不出手，小马哥就顶不住了！”
<br/>　　叶少阳猛然回过神来，这时候见那老者突然加快攻势，欺身而上，将手中的拂尘抛出去，双手结印，念起咒语：
<br/>　　“九天归命，一元化星！太上老君急急如律令！”
<br/>　　头顶上那个长条法器猛地一颤，竟然幻化出了数十道光影，将小马围在了中间，从叶少阳角度看上去，每一道光影，都悬停在了特殊的位置上，单看没什么讲究，但结合起来一看，正好就是一套天干降图。
<br/>　　在小马看来，自己前后左右到处都是光影，眼睛都要花了，自己手里就一块板砖，不知道该砸哪一个。
<br/>　　突然，所有光影在一瞬间收缩起来，宛如漫天星斗，却拖着长长的锋刃，从四面八方朝他射来。
<br/>　　“卧槽，这特么什么招啊！”小马哇哇乱叫起来。
<br/>　　“九天元辰尺！张师叔，快住手啊！”
<br/>　　叶少阳飞奔上前，一跃而起，直接跳上了一个士兵的肩膀上，一路踩过去，眼看着小马就要中招、自己却赶不上了，叶少阳情急中大叫一声：“张无生！！”
<br/>　　那老者作法的动作凝滞了下，转头看过来。
<br/>　　叶少阳将手中的木剑掷出去，看似随便这么一扔，击中了十二道星光中的其中一盏，万道白光顿时就全都凋零了，还是那一支白色的法器，当空落下，那老将军急忙上前抓住，转脸朝叶少阳看过来。
<br/>　　“好险啊卧槽！”
<br/>　　小马吓出了一声冷汗，也不再逞能，一溜烟跑到叶少阳身后来，按着他的肩膀，吐舌头说道：“这老东西太厉害了，小叶子咱俩一起上，肯定能弄得过他！”
<br/>　　叶少阳不理他，定定地看着老将军，道：“师叔，你怎么样子都变了？”
<br/>　　“你叫我什么？”老将军皱眉说道。
<br/>　　“叫你师叔啊。”叶少阳道，“你是龙虎山掌教张无生。”
<br/>　　“哈？”
<br/>　　小马惊到了，抓着叶少阳肩膀说道：“小叶子你这眼神，你欺负我没见过张无生吗，他哪里是！”
<br/>　　老将军也冲叶少阳微微一笑，“老夫叫刘德阳，不认识你说的什么张无生。你就是反贼头领？”
<br/>　　叶少阳冷笑，“你被人修改了记忆了，想不起来而已，你一手纯正的龙虎山道术，别人认不出，我可不会认错的！”
<br/>　　“胡言乱语。老夫是灵婆婆嫡传弟子，这一身法术都是她老人家教的，什么龙虎山，那种小地方也配谈什么法术！”
<br/>　　但愿你清醒之后还能想起这句话……叶少阳盯着他手中两件法器说道，“你手里这个拂尘和尺子哪里来的，叫什么名字？”
<br/>　　“师父所传，并无名字。”
<br/>　　“错啦，你手里的拂尘，叫玉清拂尘，柄上刻着一朵兰花，是你贴身法器，你左手里那把尺子更了不得，那是你龙虎山镇山之宝九天元辰尺，不信你看看，尺子上刻着‘裁天’二字，是你祖先张道陵刻的，张道陵当年创立五斗米教，自我感觉能成仙了，于是刻下这两个字，却犯了天禁，又苦修二十年，才真正悟道……”
<br/>　　这是法术界的逸闻，人人都知道的，至于那两件法器的特征，因为当年自己好奇，从他手中借过来看过，也是清楚得很。
<br/>　　张无生听了这番话，忍不住拿起两件法器看了看，表情就有些僵硬了，眉头皱起来，似乎陷入思考，片刻，他重新抬起头来，冷笑道：
<br/>　　“老夫这两件法器天下闻名，你略有见识，也不能说明什么，老夫一生记忆难道是假的不成？”
<br/>　　“你那记忆还真是假的。”叶少阳耸了耸肩。
<br/>　　小马在他身后说道：“小叶子，你跟他说这些没用，还得是老办法，你先擒住他，把他敲醒！”
<br/>　　“你是反贼头领？你又是什么邪物？”张无生狞笑，“你们以为这些鬼蜮伎俩，便能威慑朝纲？这些手段，老夫还不放在眼里！”
<br/>　　“张师叔，其实我一直都想跟你打一架。”叶少阳微微一笑，冲了过去。
