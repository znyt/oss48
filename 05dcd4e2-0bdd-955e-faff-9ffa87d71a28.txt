陆寒霆敛了一下俊眸，薄冷的嗓音溢出几分松软，“我没有忘，正因为我没有忘，所以你才能安然无恙的站在那里跟我通电话。”
<br/>
<br/>        这些年他的身边都没有过女人，因为他不会给那些女人接近他的机会，像夏妍妍今天这样又是追他车，又是给他疯狂打电话的早已经惹怒了他，她还能安然无恙的站在那里跟他通电话完全是因为…因为她是当年的那个女孩。
<br/>
<br/>        “陆总，那你承认吧，你一直没有忘，这些年你也一直对我念念不忘啊，你的心里是有我的位置的，你也喜欢我！”
<br/>
<br/>        陆寒霆抿了一下薄唇，他看着沐浴间的方向，“我不喜欢死缠烂打的女人，夏妍妍，给自己保留一点体面和尊严，这样我看着你的时候才不会出戏，你真的一点，一点都没有我记忆里的那个模样。”
<br/>
<br/>        夏妍妍的瞳仁瞬间收缩放大，她的心跳差点就停止了，他是不是已经察觉到了什么？
<br/>
<br/>        她当然不是他记忆里的那个女孩，当年的那个女孩其实一直都在他的身边，她是…夏夕绾啊！
<br/>
<br/>        夏妍妍的双唇止不住的哆嗦，她不敢再说话了，快自己露出破绽。
<br/>
<br/>        这时陆寒霆又道，“夏妍妍，如果你足够聪明的话，就应该利用那段救命之情提一个我会答应的要求，还有，我希望这是我们最后一次通电话，以后不要再给我打电话了，因为我的陆太太心眼很小，特别爱吃醋，她不喜欢我跟其他女人来往，我现在已婚，是一个有妇之夫，没有任何离婚或者是出轨的打算，不用再惦记我了，死了这条心吧。”
<br/>
<br/>        说完，陆寒霆直接挂断了电话。
<br/>
<br/>        ……
<br/>
<br/>        陆寒霆将手机丢在了床上，抬起手指扯开了衬衫的两颗纽扣，这时“嗒”一声，沐浴间的门开了，夏夕绾洗好澡出来了。
<br/>
<br/>        这里准备了睡衣，夏夕绾身上穿了一件香槟色的吊带睡裙，她绝色的脸蛋被里面的热气蒸蕴出了粉红，俏生生的像一朵水莲。
<br/>
<br/>        手里拿着毛巾擦拭着湿漉的长发，夏夕绾看向他，“陆先生，我洗好了，你去洗吧。”
<br/>
<br/>        陆寒霆的目光落在她的身上，上下打量了一眼，有点肆意，“那我去洗了，我很快的，陆太太，等我。”
<br/>
<br/>        他这是什么意思？
<br/>
<br/>        夏夕绾羞恼的将手里的毛巾用力的丢到了他的俊脸。
<br/>
<br/>        陆寒霆笑着进了沐浴间。
<br/>
<br/>        很快里面就传来了“哗啦啦”的水流声，夏夕绾坐在梳妆台前吹头发，这时男人低沉磁性的嗓音传来，“陆太太，帮我拿包烟。”
<br/>
<br/>        他都洗澡了还要烟干什么，在里面抽烟？
<br/>
<br/>        夏夕绾在床柜上拿到了一包烟，推门走进了沐浴间，磨砂玻璃门挡着，但隐约可以看到他高大挺拔的身躯。
<br/>
<br/>        他正弓着腰在洗头，细小的水珠溅到了他结实精健的肌肉上然后快速的弹开，限量极的美男沐浴图。
