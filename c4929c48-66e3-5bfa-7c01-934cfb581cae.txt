　　虚空之中，便是无数北邙学院的长老，一直一直都是以灵力，扫视着下方的人群，准备寻找几个资质极高的天才收为弟子来培养。
<br/>　　像是他们这等生死境强者，收徒之事，便是也都极为重要。
<br/>　　若是教出来的弟子，实力不济，败坏的可是作为师父的名声，自然没有人会贸然选择收徒。
<br/>　　“小有名气的天才又如何？在外面的天才，不过都是矮个子里拔高个的，在学院中，根本都排不上号。”
<br/>　　又一尊长老开口道，他的言语之中，极为不屑。
<br/>　　“他们二人的家族不错，从小培养罢了，要是看到一些有资质的天才，稍加培养，便是很快就能超越他们。”
<br/>　　“那是自然，我们北邙学院的资源，比起外面要雄厚无数倍。”
<br/>　　又一尊长老开口符合道。
<br/>　　他们如今挑选弟子，可不单单是看现在的实力。
<br/>　　毕竟来参加北邙学院考核的弟子，不少人的出身都不一样，同样的境界，若是一个依靠家族力量，一个自己修炼而成。
<br/>　　那自然是第二个的资质更加优秀，当然，实力一样是极为关键的部分。
<br/>　　一尊长老，看向楚风眠的方向，眼神中露出几分亮光道。
<br/>　　“这个小家伙倒是有些意思，能够以御风境的境界，居然就可以通过考核。”
<br/>　　“能够跨越六重境界，这等资质，也算不错了。”
<br/>　　“哦？不过可惜这个小家伙的实力还是太弱，这样的境界还是太低了一样。”
<br/>　　令一尊长老，也注意到了楚风眠的存在，扫视了一眼，摇了摇头道。
<br/>　　“要是他现在已经是踏入到了神力境的话，倒是可以多加培养，但是御风境，这样的人，能不能突破神力境都不一定。”
<br/>　　这两尊长老的扫视，楚风眠自然是察觉的到。
<br/>　　看到那两人没有发现什么异常，楚风眠终于是松了口气。
<br/>　　这些长老的实力，要远远超越他，他还远远无法跟这些生死境高阶的强者抗衡。
<br/>　　楚风眠可不想被人发现他身上的异常，于是便是隐藏了大部分的实力。
<br/>　　“等等，这个人是谁？”
<br/>　　一尊长老的眼睛，突然瞪得豆大，突然看向了平台上坐在楚风眠身边的寒月漓。
<br/>　　“好恐怖的寒意，我的灵识，在她的身边，居然都无法活动自如，这等寒意，简直恐怖至极！”
<br/>　　“真的，这一届的新人中，居然还有这样的人物？”
<br/>　　另外几尊长老，也同样以灵识探了过去，便是察觉到了其中的异常，一个个震惊到极点。
<br/>　　他们可都是生死境高阶的人物，居然他们都无法看穿寒月漓的隐藏。
<br/>　　寒月漓身边的寒意，足以将他们的灵识都给封锁。
<br/>　　“这等寒意！以神力境的境界，就足以影响我们的灵识，莫非此人是先天神体！”
<br/>　　一尊长老，猛地开口道。
<br/>　　只有先天神体的力量，才可以跨越境界，让他们无能为力，无法看穿。
<br/>　　“先天神体！”
<br/>　　这一句话，宛如一声惊雷，直接从无数长老之中响彻起来。
<br/>　　一道道灵识，纷纷的向着寒月漓看了过去，但是在寒月漓的身边一刻，都是被直接冻碎。
<br/>　　他们所能够看见的，便是宛如一座冰冷到极点的冰山。
<br/>　　“这样的先天神体！我一定要收为弟子才行！”
<br/>　　“不错！一个夜天君，如今便是在北邙学院中称王称霸，他的师尊也是地位水涨船高，如今又来一个先天神体，我必须要收她为弟子！”
<br/>　　几尊长老的心中，都是起了收徒之心。
<br/>　　先天神体，只要是稍加培养，便是会成为名震北域的超级强者。
<br/>　　“这女子，我要了！”
<br/>　　一声轻喝，突然响起，这是一道极为清澈的女声，突然之间，在虚空之中，一道冰蓝身影突然从中落下。
<br/>　　这道冰蓝身影，是一尊长相极为俊美的女子，脸若冰霜，令人看见，都不由的产生几分寒意。
<br/>　　“原来是冰璃长老。”
<br/>　　看到那女子出现，苏凌长老脸上都露出了几分恭敬之色。
<br/>　　这女子的实力，还要远超苏凌长老。
<br/>　　她的实力到底有着什么地步，楚风眠都看不清，但是至少也是要逼近生死境八重，九重的强者，甚至是生死境巅峰的强者，都有可能。
<br/>　　在北邙学院的无数长老之中，都是位高权重，地位极高的一尊大人物。
<br/>　　那些本来还想要争夺一下寒月漓的长老，看到这冰璃长老出面，便是都纷纷退去了，不敢争夺。
<br/>　　“玄寒之体，你的体制，正好适合跟我极为相似，跟我走吧。”
<br/>　　冰璃长老看了寒月漓一眼，便是大手一挥，直接将寒月漓带走。
<br/>　　“是。”
<br/>　　寒月漓恭敬的说了一句，便是直接跟随了冰璃长老离开。
<br/>　　寒月漓自己也清楚，这是属于一步登天的机会，便是直接跟随冰璃长老离开。
<br/>　　初入北邙学院中，便是就有着这样一座大靠山，楚风眠也是放心的点了点头。
<br/>　　寒月漓被带走，更是刺激了在场的无数武者。
<br/>　　在场无数武者，一个个便是将自己的实力催动到极限，也不在做任何的隐藏。
<br/>　　都想要跟寒月漓一样，被一尊长老看中带走，便是就可以从此平步青云。
<br/>　　“你，跟我走吧。”
<br/>　　又一尊长老下来，指着燕古开口道，便是将燕古带走。
<br/>　　然后剩下的，牧野战，也是被一尊长老带走。
<br/>　　这一届新人中，实力资质较强的几人，便是都被带走了，这倒是都在楚风眠的意料之中。
<br/>　　最终，虚空之中的长老，便是都纷纷离开。
<br/>　　一些没有收到弟子的长老，也不气馁，毕竟宁缺毋滥，没有合适的，等待便是。
<br/>　　楚风眠如今隐藏了自己大部分的资质，也没有被人带走。
<br/>　　突然拜师对楚风眠也不是一件好事，他的身上秘密是在太多，拜师的话，很有可能被人发现。
<br/>　　如今独自一人，倒是方便许多。
<br/>　　更何况，楚风眠本身也不需要什么老师，他有着前世的无数阅历，不需要任何功法剑术，他自己都懂得。
<br/>　　楚风眠现在就是缺少力量，并不缺少功法剑术，只要他的实力足够，一切都可以施展。
