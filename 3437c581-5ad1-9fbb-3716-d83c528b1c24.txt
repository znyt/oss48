　　第724章
<br/>　　这尊六手金佛有婴儿一臂大小，模样栩栩如生，重量自然也很重。
<br/>　　拿人钱财，与人消灾。
<br/>　　程大雷不想收，也不单单是因为不方便携带的缘故。这江湖，吃人一颗枣，就得给人还回去一棵树，没有吃白食的道理。今天不过就是吃了陈德山一顿饭，就得出手为他惹上一股贼。
<br/>　　这尊价值不可估量的六手金佛如果收下，还不知会惹出怎样的事情来。
<br/>　　陈德山欲言又止，脸上已经有些不好看。这程大雷分明是看不起自己么，他这种乡绅最是好个脸面。
<br/>　　“牛将军假若不收，老朽就撞死在地上，将这条命还给牛将军。”
<br/>　　程大雷停下筷子看着他，微微摇头。
<br/>　　便是这目光，就令陈德山心底冒起一股寒气。他本来怒发冲冠，须发倒竖，却也不知该如何收场。
<br/>　　陈忠近前一步，搀扶住自家老爷：“牛将军既然不愿收，就先保存在我们家中，牛将军什么时候想要，再给牛将军送过去。”
<br/>　　“喔，好，是这个道理。”陈德山捡了个台阶，算是重新坐下来。
<br/>　　陈忠清了清嗓子，道：“牛将军，俗话说救人救到底，送佛送到西，还有一桩事要请牛将军救命。”
<br/>　　程大雷没有说话，只是看着他。
<br/>　　“荆棘岭的贼人不会善罢甘休，他们若再来，一定会对陈家庄赶尽杀绝，还请牛将军救我全家老小一命。”陈德山知道现在不是要脸面的时候，于是干脆把话挑明白了。
<br/>　　酒桌上，刘发财和徐神机依旧大块朵颐，程大雷与陈德山之间的气氛却微妙起来。崔白玉漠然看着这一幕，心中大概在嘿嘿冷笑。
<br/>　　程大雷本身就不想管这件事，因为这件事不太好管。他虽然能喝退雷动一行人，但他们若是卷土重来？程大雷是护得了一时，护不了一世。荆棘岭也有百十号人，程大雷也不可能去把他们杀个干净。
<br/>　　纵然吃人家嘴软，拿人家手短，将人灭了干净。免不得荆棘岭群贼有些朋友弟子，他们回头想着报仇，也是一桩麻烦事。
<br/>　　只有千日做贼，没有千日防贼的道理，说来说去，没有一劳永逸的法子。
<br/>　　话说过来，这种事陈德山本就逃不过。他家大业大，仆役如云，自然就成了官府和山贼眼中的肥肉，所以今天没小白龙，也会有小黑龙，小王八之类的出现。
<br/>　　陈德山本来的心意，便是慷慨送出六手佛，让程大雷为其做事。程大雷的本事他看见了，但估计脑袋不是很好使，自己稍微奉承几句，他就二杆子精神大爆发，将命也舍了给他。
<br/>　　可他看错了，程大雷是忽悠别人给他卖命的主儿，这点心机他岂能看不出来。偏偏，程大雷还特别讨厌别人在他面前卖弄心机。
<br/>　　陈德山险些给程大雷跪下，却见程大雷依旧无动于衷。呆立在那里，久久无语，抬起头脸上已挂满老泪。
<br/>　　“你瞧瞧，说着说着怎么哭了起来。”刘发财道：“老丈，我们不过喝了你杯酒，不至于把你心疼成这样吧。”
<br/>　　刘发财也是惯走江湖的主儿，他早已知道，好人不能做，因为容易被人当做冤大头。
<br/>　　“好啦，好啦。”程大雷摆摆手，面向刘发财道：“好啦好啦，你就走一遭吧，看那荆棘岭是怎么会事。”
<br/>　　刘发财晃了晃膀子站起来，道：“备一匹快马，我就为你这老人家走一趟。”
<br/>　　快马奔驰而去，刘发财独自上路。陈德山完全是一头雾水，不明白程大雷的目的。
<br/>　　而程大雷依旧喝酒吃菜，宛若没有什么事发生。陈德山可放心不下，小心翼翼给程大雷倒满酒。
<br/>　　“那小白龙可不是个简单人物，杀人成性，贵属下一个人去没问题吧？我庄里也有些胆大的汉子，不若让他们陪着一起去。”
<br/>　　“不用麻烦，来，喝酒喝酒。”
<br/>　　程大雷一句话便遮过这件事，陈德山心里却不停打鼓，如同铁锅上的蚂蚁，度日如年的焦灼。可程大雷明显不想说，他也不可能打破砂锅问到底。
<br/>　　有一个多时辰，陈忠忽地从外面跑过来，口中高声道：“刘壮士回来了，刘壮士回来了。”
<br/>　　陈德山本就无心喝酒吃肉，腾地一下站了起来，踮起脚尖往外望。
<br/>　　刘发财脸上还挂着汗，可见他一来一去的匆忙。
<br/>　　陈德山上上下下将他打量一眼，仍旧不放心：“刘壮士，那小白龙没把你怎样吧？”
<br/>　　刘发财手里捧着一口刀，递到程大雷面前：“那人留下的，说悬刀在门口，就不会有山贼登门。”
<br/>　　“是白龙刀。”陈德山眼睛一亮，然后难以置信的看着刘发财。
<br/>　　这白龙刀就相当于小白龙签发的平安符，只有有了这柄刀，就意味着是荆棘岭的朋友。小白龙不仅不会过来生事，还会想办法保一方平安。
<br/>　　刘发财大马金刀的坐下，陈德山将酒与他斟满，目带崇敬道：“刘壮士如何做到的？”
<br/>　　程大雷撇撇嘴，还能如何做到，无非是报了程大雷的名字。白龙刀能保平安，但还是程大雷这三个字更能遮一方风雨。小白龙只要稍微懂些规矩，就不会再来寻衅。
<br/>　　一柄剑屠光荆棘岭是痛快，但这是赶尽杀绝的法子，有些坏规矩。比如今天，小白龙如果单单只是抢人，对山贼来说是稀松平常的事。可若他一言不合，就灭人全家，就是坏规矩。
<br/>　　行走江湖，坏规矩的事不是不能做，只是做了尽量不要让别人知道。大家哪怕杀人，多多少少也得找个由头，比如你今天为什么戴帽子之类的话。
<br/>　　“哈，还能怎么办的，我单人独骑到了荆棘岭，直接叫那小白龙出来说话，与其大战三百回合，打的是难解难分。”刘发财已经开始吹嘘起来。
<br/>　　“刘壮士还有这么好的武艺？”陈德山吃惊的睁大眼睛。
<br/>　　“嘿，我有什么武艺，只能说小白龙的武艺太一般。被我打败后，死皮赖脸要给我当徒弟，我嫌他天赋太差，不肯收而已。”刘发财无所谓道。
<br/>(本章完)
