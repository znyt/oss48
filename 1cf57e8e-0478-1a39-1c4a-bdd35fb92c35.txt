　　片刻后，慕清霄望着映入眼帘的日向家族驻地，星眸中闪过一丝怀念与歉意。
<br/>　　此时，他早已经换掉晓组织的服饰，取而代之的是一袭金色镶边的白衣，腰间挂着翡翠玉佩。
<br/>　　白衣翩翩，面如冠玉，唇若抹朱，剑眉星目，气质高贵。
<br/>　　日向家族的驻地，相比当年，也是小有变化，但总体位置并没有变。
<br/>　　慕清霄顺着记忆中的路线，很快就出现在驻地中央，一座类似四合院的建筑前。
<br/>　　视线穿过门槛，看向院落中，内部种植着一些花草，就算是站在门外，依旧能闻到其中浓郁的花香味。
<br/>　　而在花丛中央，一名年方二十左右的素衣女子。
<br/>　　女子手中提着花洒，黑发如瀑直达腰际，肌肤如玉，宛如百花仙子，完美无瑕的脸蛋，犹如上天的杰作，找不出一丝的瑕疵。
<br/>　　虽然穿着素以，却掩盖不了自她身躯上散发而出，端庄成熟的气质。
<br/>　　似乎是察觉到了什么，素以女子渐渐转过身来。
<br/>　　下一秒，当她看到站在门口处的身影后，娇躯猛的一颤，如遭雷击。
<br/>　　“咚咚......”
<br/>　　花洒掉落在地上，发出沉闷的声响，素衣女子却浑然不觉，玉手握着粉唇，透明如宝石一般的眸子，雾气弥漫。
<br/>　　渐渐的，珍珠一样的泪珠，自她宝石一样的眼眸中涌出，她却始终压抑着，不让自己发出哭泣声。
<br/>　　“丫头，我回来了。”
<br/>　　“呜呜......”
<br/>　　话语声刚落，雏田在也压抑不住，哪怕她在坚强，此时也压抑不住心中的委屈，泣不成声。
<br/>　　随后，美眸盯着向院中走来的青年，宛如梦境一般，同时，雏田心中升起一丝恐惧。
<br/>　　她已经不是当初懵懂的少女，倘若继续深陷其中，他再次离开，自己该怎么办？
<br/>　　可是随着青年来到她的身旁，熟悉的清香扑面而来，她在也忍不住，猛的扑进他的怀中，轻声抽泣起来。
<br/>　　抱着怀中颤抖的玉体，慕清霄没有一丝的杂念，只想紧紧的抱着她。
<br/>　　“你说过很快就回来的，骗子，骗子......”
<br/>　　脆弱的声音响起，雏田攥着粉拳，不断在他的胸膛上锤打起来，可见数十年的等待，对她来说是何等的煎熬。
<br/>　　闻言，慕清霄心脏疼的直抽，将她抱的更紧：“我以为很快就能回来的......”
<br/>　　慕清霄知道，再多的解释，相比数十年的等待，实在是太过苍白无力，只能不断的安慰她。
<br/>　　半个小时后，两人紧紧相拥，雏田则在发泄过后，乖乖的待在他怀中，贪婪的嗅着他的气味。
<br/>　　“这些年来，只要闭上眼睛，我就会想到你，以为你已经把我忘了，永远都不会回来了。”
<br/>　　慕清霄左手搂着她的细腰，右手轻轻抚摸着她的青丝，道：“如果我一直不回来，你打算怎么办？”
<br/>　　微微抬起臻首，雏田透明的眸子望着他。
<br/>　　“我不知道，可能会一直待在这里等你，也可能会离开木叶，但你终究是回来了，就像做梦一样。”
<br/>　　叹息一声，慕清霄一脸歉意，道：“雏田，跟我离开吧，去一个很远的地方，我们的家就在那里。”
<br/>　　“嗯，以后你都不准擅自离开我。”
<br/>　　雏田完美无瑕的脸蛋上，极为认真的说道，年龄虽然已经不小，性格也极其坚强，做事也沉稳。
<br/>　　但在慕清霄的面前，她依旧是当年，那个乖巧听话的小丫头。
<br/>　　“好，我答应你。”
<br/>　　说着，慕清霄在她粉唇上轻轻一吻，道：“暂时在木叶住一段时间，收拾收拾东西，将琐事处理好，我们就离开这里。”
<br/>　　微微点头，雏田道：“今晚你住哪里？”
<br/>　　闻言，慕清霄坏笑一声，道：“当然是住你屋里，你的屋子可不就是我的屋子吗？”
<br/>　　话语刚落，雏田桃腮微红，她可不像当初那么单纯，当然清楚慕清霄的意思，芳心羞涩不已。
<br/>　　“嗯，你先进去休息，我开始准备晚餐。”
<br/>　　望着她快步离去的背影，慕清霄微微一笑，道：“当年的小姑娘，已经完全长大了吗？”
<br/>　　若是以前，亲一下都羞得不行，现在提出住在一起，虽然羞涩，但反应却不是很大。
<br/>　　十年的时间，令人绝望，也能令人学会坚强。

<br/>　　随着时间的推移，夜幕降临，黑夜笼罩大地，木叶忍者村则依旧人气高昂，夜生活才刚开始。
<br/>　　日向家族驻地，偌大的客厅中，慕清霄坐在饭桌前，而坐在他对面的，正是漩涡鸣人与他的妻子日向花火。
<br/>　　左侧有两名小身影，紧紧盯着他，其中一名叫漩涡博人，另一名叫漩涡向日葵。
<br/>　　望着两道小身影，与对面的夫妻，慕清霄眼角抽了抽，一时间，客厅的气氛显得有些古怪。
<br/>　　就在这时，坐在对面，打量慕清霄已久的日向花火，道：“你就是姐姐一直提起的，宇智波清霄？”
<br/>　　“没错。”
<br/>　　听到慕清霄平淡的语气，日向花火气不打一处来，双手拍在桌上，满脸怒色。
<br/>　　“你个混蛋，居然还有脸回来，知不知道，姐姐等你等的多幸苦，每天都只待在屋内，甚至以泪洗面。”
<br/>　　“我的确亏欠她太多，小家伙是你们的儿女？”慕清霄并不生气，也没有生气的资格，望着两个小家伙说道。
<br/>　　旋窝鸣人微微点头，看向慕清霄的目光也极其的复杂，道：“嗯，旋窝博人，旋窝向日葵。”
<br/>　　“我这个做姨夫的也没什么好东西给他们。”
<br/>　　说着，慕清霄取出两瓶‘地心淬体乳’放在两人的面前，道：“小家伙，这个送给你们当礼物吧”
<br/>　　望着玉瓶中的碧绿色液体，漩涡鸣人道：“这是？”
<br/>　　“准备好木桶，盛满温水，滴上一些，易筋经的修炼心法你们都有，接下来的事情就不用我说了吧。”
<br/>　　拿过桌上的玉瓶，漩涡鸣人打开瓶盖，其中便有一股恐怖的力量席卷而来，令他为之一颤。
<br/>　　“这是一种有助于修炼的灵药，能够打牢他们的根基，改造他们的资质，没任何副作用，修炼过仙术的你，对这种气息不陌生吧？”
