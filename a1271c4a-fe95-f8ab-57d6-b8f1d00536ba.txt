        “成年了啊！”连星儿一脸坦荡。
<br/>
<br/>        “那我们特意过来庆祝你，一人一杯啤酒，你不喝，岂不是太不给我们面子了？”带头的女生故意使出了激将法。
<br/>
<br/>        其他四个女生纷纷随声应和：“就是！”
<br/>
<br/>        连星儿不满地皱起了眉头。
<br/>
<br/>        从小到大，还没有人敢逼她做她不愿意做的事情。
<br/>
<br/>        明司寒回来时，看到连星儿被五个女生围着，脸色一冷，径直走过去时，周遭的空气似乎也跟着他的情绪变得异常冷冽。
<br/>
<br/>        五个女生看到明司寒时，一个个不寒而栗。
<br/>
<br/>        带头的女生刚想开口说句什么的时候，但她的嘴巴才张开，一个字都未来得及吐出来，就只见明司寒将坐在沙发上的连星儿给拉起来，牵着连星儿离开了这间聒噪的包厢。
<br/>
<br/>        包厢门一关，连星儿便感觉耳朵得到了释放，这世界好像清净了不少。
<br/>
<br/>        “你不唱歌吗？”连星儿反问道。
<br/>
<br/>        明司寒嘴角微扬：“你不也一样，一进包厢，就坐在那儿玩手机。不点歌，也不吃东西。”
<br/>
<br/>        “因为太吵了啊！吃东西没胃口。”连星儿解释道。
<br/>
<br/>        明司寒微微一笑：“要不，我们出去散步。”
<br/>
<br/>        “好啊！”连星儿猛点头。
<br/>
<br/>        她其实有意识到，他一直都在关照她。
<br/>
<br/>        张子晋他们三个在包厢里唱得那么嗨，也就间接地证明了他们四个以往肯定也能唱。
<br/>
<br/>        连星儿不是不喜欢唱歌，只不过是不喜欢唱给那么多人听。
<br/>
<br/>        离开KTV城后，明司寒牵着连星儿的手，走在了满是霓虹灯光的街边。
<br/>
<br/>        明司寒知道，这暗中，一定有好几双眼睛正在盯着他。
<br/>
<br/>        想到这里，明司寒忍不住跟连星儿开了个玩笑：“你说，我如果把你拐跑了，你爹地派在你身边的那些保镖们算不算失职？”
<br/>
<br/>        “那你把我拐跑一次试试，让我爹地找不到我。”连星儿眼前一亮，像是遇到了一件很好玩的事情一样，想要雀雀欲试。
<br/>
<br/>        明司寒会心一笑，抬起手来，轻轻地刮了一下连星儿的小鼻尖：“我可不敢。万一，我被岳父大人讨厌了，以后还怎么把你这个小公主娶回家。”
<br/>
<br/>        “你想娶我？”连星儿笑不拢嘴地问。
<br/>
<br/>        明司寒不假思索地回答：“当然。”
<br/>
<br/>        “那我们明天就结婚！”连星儿高兴不已地说。
<br/>
<br/>        很明显，这丫头是那婚姻当过家家了。
<br/>
<br/>        明司寒忍不住揉了揉连星儿的头顶，笑道：“那你快长大啊！才十八岁，还这么小。”
<br/>
<br/>        “我要你陪我一起长大。”连星儿挽上明司寒的臂弯，笑着说道。
<br/>
<br/>        明司寒顺势微微俯身，吻了吻这小丫头的额头。
<br/>
<br/>        和明司寒在一起，连星儿发现自己确实没有什么烦恼。
<br/>
<br/>        他们在外逛了一圈，KTV城那边大家的余兴节目也已经结束，然后一起坐大巴车回到了临海大学。
<br/>
<br/>        翌日。
<br/>
<br/>        明司寒在课上到一半时，突然被班主任给叫了出去。
<br/>
<br/>        连星儿下意识地看向张子晋他们，他们三个微微耸肩，表示不知情。
<br/>
<br/>        明司寒去了校长办公室，才得知自己已被M国的HF大学计算机系研究生院提前录取了。
<br/>
<br/>        只是，令明司寒纳闷的是，他并没有去报考HF大学计算机系的研究生，怎么会突然收到提前录取的通知书。
<br/>
<br/>        校长一边推了推鼻梁上的老花眼镜，一边将录取通知书递给明司寒，和蔼可亲地说道：“你真的是我们学校的荣耀。”
<br/>
<br/>        “这到底是怎么回事？”明司寒依旧不解。
<br/>
<br/>        校长却答非所问：“你不是一直想考HF大学吗？怎么突然不需要你去考，就录取你了，你反倒不高兴了？”
<br/>
<br/>        “那倒没有不高兴，只是觉得很奇怪。”明司寒赧然一笑。
<br/>
<br/>        校长接着说道：“是M国那边的一个政客，好像是叫乔治先生，他给HF大学校长的推荐信。”
<br/>
<br/>        “乔治先生？我不认识什么乔治先生。”明司寒更加诧异了。
<br/>
<br/>        校长笑道：“不管怎么说，你能提前去读HF大学计算机系的研究生，对你来说也是件好事。那儿，全世界最优秀的老师。”
<br/>
<br/>        “嗯，对！谢谢校长！”明司寒接过录取通知书后，向校长微微颔首。
<br/>
<br/>        回到教室里时，明司寒的那张录取通知书倒是让张子晋他们乐开了花，但是，连星儿脸上却是笑容全无。
<br/>
<br/>        “星儿，你不高兴吗？”明司寒问道。
<br/>
<br/>        连星儿反问：“你要在那儿读几年？”
<br/>
<br/>        “我打算研硕连读，如果我成绩可以的话，打算读到博士后。”明司寒的俊脸上，挂着对自己未来计划的憧憬。
