　　见他处变不惊，涵素真人面露和煦之色，道：“阁下年纪轻轻就有如此成就，乃是前无古人啊。”
<br/>　　“涵素真人过誉了。”
<br/>　　至于其他应征长老的，则盯着慕清霄，仍旧有些不敢相信，如此年轻的练虚境强者！
<br/>　　见慕清霄面色平淡，丝毫不为自己赞赏的话语所动，涵素真人对其心性更加满意。
<br/>　　绝世天才他不是没见过，但像慕清霄如此妖孽的，他真是第一次见，哪怕是他自己，或者是紫胤真人。
<br/>　　在二十岁的时候，都只是刚接触修仙一途而已。
<br/>　　最重要的是，慕清霄才二十岁，在给他十年，或者二十年，能成长到何种地步，谁也不知道。
<br/>　　不管如何，都要将他留在天墉城，哪怕付出一些代价也在所不惜。
<br/>　　“今日之事，我希望各位长老能够烂在肚子里。”
<br/>　　说着，涵素真人瞥了一眼下方，参加应征的长老，话语中的意思，不言明了。
<br/>　　“是，掌门。”
<br/>　　满意的点了点头，涵素真人的目光再次落在慕清霄身上，道：“不知道慕长老，希望应征那一堂长老？”
<br/>　　话语刚落，殿内几名应征普通长老的，都纷纷露出羡慕之色。
<br/>　　以他们的成就，注定只可能是普通长老在，至于更高的职位，注定与他们无缘。
<br/>　　听闻涵素真人的话语，慕清霄知道，后者是在变相的拉拢他，正合他意。
<br/>　　“回涵素真人，在下在丹道上也颇有成就，所以想应征凝丹堂长老一职位。”
<br/>　　涵素真人怔了怔，不可思议，道：“难不成，你还是一位炼丹师？”
<br/>　　微微点头，慕清霄伸出右手，掌心就冒出一团黑色火焰，火焰没有散发出一丝温度，却显得极其妖异。
<br/>　　此时，涵素真人再也坐不住，自掌门座椅上站了起来，目光死死的盯着他掌心的神火，眼中尽是难以置信的神色。
<br/>　　“嘶嘶...！”
<br/>　　同时，殿内也响起一片倒吸凉气的声音，因为他们都知道，丹火意味着什么。
<br/>　　慕清霄微微一笑，来到古剑奇谭位面一个月的时间，他可没有白白浪费，平时除了游山玩水，并没有忘记收集资料。
<br/>　　‘古剑奇谭’位面的修炼体系，可比斗‘斗破苍穹’位面要高的多。
<br/>　　天墉城虽有剑仙紫胤真人，但比紫胤真人强的并非没有，只是不再人世间罢了，而是在虚无缥缈的天界。
<br/>　　但是，斗破位面却有一样东西远超古剑奇谭位面。
<br/>　　没错，那就是炼丹师。
<br/>　　在这个位面中，炼丹师如凤毛麟角，稀少无比，千万人当中，能有一个拥有炼丹师天赋，就已经是上天的眷顾。
<br/>　　正巧的是，天墉城也有这样一位炼丹师，正是凝丹堂的长老，实力也只有练虚境，却有着极高的地位。
<br/>　　凝丹长老，负责炼制内服丹药，外敷伤药，不论静思冥想，或者是动兵刃净化浊气，内外丹药都是绝佳辅助。
<br/>　　听闻凝丹长老只是一位七品炼丹师，但在天墉城内的地位却极其尊贵。
<br/>　　炼丹师分一到九品，九品之上还有‘天品’，至于更高的，慕清霄现在还没有接触的资格。
<br/>　　盯着慕清霄掌心的神火，涵素真人满脸震惊，道：“确实是丹火，阁下真是一名炼丹师，就是不知道，是几品炼丹师？”
<br/>　　“在下不才，炼丹多年，才成为七品炼丹师，说来有些惭愧。”
<br/>　　慕清霄微微摇头，一脸惭愧的说道，有《朱子率》在，他成为九品炼丹师只是时间问题，只是缺少实践与药材而已。
<br/>　　至于更高的‘天品’，也就是相当于斗破苍穹的帝品丹药，还为时过早。
<br/>　　但听闻他的话语，涵素真人脸上肌肉都抽搐起来，至于周围的普通长老，整个人都不好了。
<br/>　　甚至，脸色都变得有些难看，二十岁的炼丹师，二十岁的练虚境，难道是打娘胎里就在炼丹不成？
<br/>　　若是慕清霄不才，那他们这辈子，岂不是都活到狗身上去了？
<br/>　　“非常的好，七品炼丹师，果真是上天眷顾我天墉城啊！”
<br/>　　愣神过后，涵素真人倒吸一口凉气，满口赞誉，看向慕清霄的目光，犹如在看一件稀世珍宝。
<br/>　　“慕小友，你可愿意担任，我天墉城凝丹堂第十一代长老？”
<br/>　　此时，涵素真人也变得非常客气，目光也变得激动起来，二十岁的七品炼丹师啊，哪怕妖孽一词，都不足以用来形容他。
<br/>　　哪怕现在的凝丹堂，还虚真人也不过是七品炼丹师！
<br/>　　而且他也有些小心思，以慕清霄的资质，成就注定不会低，日后他修炼遇到壁障，可能需要他炼制丹药，助他突破。
<br/>　　“在下愿意。”
<br/>　　慕清霄微微行礼，涵素真人毕竟是天墉城掌门，虚礼还是要的。
<br/>　　行礼过后，慕清霄却是在思索，天墉城的还虚真人是现任的第十一代长老，难道要让自己把他踹下去？
<br/>　　成为凝丹堂的长老，好处可是无穷无尽的，天墉城会供应他炼丹的资源。
<br/>　　有了这些资源，他就能以最快的速度成为九品炼丹师。
<br/>　　炼丹过后，他还能从中抽取一些利润，有高级的灵药，他还能自己留下一些，种植起来，日后给妻子们使用。
<br/>　　最重要的是，有这一层身份，会给他在天墉城中，带来许多便利。
<br/>　　“恭喜掌门，恭喜慕长老。”
<br/>　　见慕清霄成为凝丹堂第十一代长老，其余普通长老纷纷开口恭喜，眼中在也没有一丝怀疑。
<br/>　　甚至，脸上都带有讨好之色，毕竟不管是慕清霄的身份，或者是资质，都有让他们讨好的资格。
<br/>　　“这是我天墉城的机缘啊。”
<br/>　　涵素真人语气虽平淡，可脸上的笑容怎么也掩饰不住。
<br/>　　实在是这种妖孽，万年难得一见啊，现在却加入了天墉城，怎能不喜？
<br/>　　“慕长老，这是天墉城凝丹长老的身份玉佩，今日起，你便与还虚真人，同为凝丹堂第十一代长老。”
<br/>　　说话的同时，涵素真人右手一掷，一块巴掌大小的玉佩就向慕清霄飞射而来。
<br/>　　慕清霄手一挥就将玉佩抓在手中，玉佩通体碧绿，正面刻着‘凝丹’两字，反面则刻着天墉城三个大字。
