　　半个时辰后，两人耳鬓厮磨，亲密一番后，便开始穿戴衣物。
<br/>　　午餐过后，慕清霄就下定决心，打算立刻离开，否则，坠入温柔乡中，越是舍不得离去。
<br/>　　“这就要离开了吗？”
<br/>　　莲藕般的玉臂，环在他的腰间，臻首贴在他的背上，贪婪的嗅着他的气息，俏脸上尽是不舍。
<br/>　　“嗯。”
<br/>　　说着，慕清霄将手腕上的吞天蟒取了下来，道：“彩鳞就留在你身边，我不在的时候，由她来守护你。”
<br/>　　接过七彩吞天蟒，小医仙俏脸一喜，没有一丝的惧怕，经过几日的相处，她早就喜欢上，这可爱的小家伙。
<br/>　　有彩鳞在，小医仙的安全也有保障。

<br/>　　这些花瓣，自然就是能增强灵魂力量的超级灵药，有了药尘的先例，恢复天火尊者的灵魂，想必也不再话下。
<br/>　　这时，一道虚幻的灵魂，自纳戒中缓缓升起。
<br/>　　见到天火尊者，慕清霄道：“前辈，晚辈需要离开一些日子，这些灵药，完全足够前辈修复灵魂。”
<br/>　　听闻能够修复灵魂，天火尊者瞳孔一缩，手一卷就将灵药拿到面前。
<br/>　　片刻后，天火尊者声音有些颤抖，道：“这......这是何等灵药，前所未闻，老夫的灵魂，真的精神了一些。”
<br/>　　震惊过后，天火尊者连忙将灵药收进纳戒中，生怕慕清霄反悔似得。
<br/>　　“前辈，晚辈承诺的事情已经兑现，不过，在我离开的期间，可否请前辈保护她们一段时间？”
<br/>　　“你是说那两个小女娃？没问题，只要老夫灵魂修复，保护她们一段时间又何妨。”天火尊者满口答应，没有丝毫的犹豫。
<br/>　　他很喜欢慕清霄的性格，如此宝贵的超级灵药，二话不说就拿给他。
<br/>　　天火尊者本以为，灵魂想要完全修复，恐怕需要很长一段时间，却未曾想到，幸福来的如此突然。
<br/>　　如此一来，慕清霄总算放下心来，彩鳞实力虽强，但也只是斗宗，有天火尊者这层保障，他也能安心离开。
<br/>　　将事情处理完毕，慕清霄刚想准备离开，脑海中就响起一道声音。
<br/>　　“你真的要离开，本王可没答应你，说要保护她。”
<br/>　　“彩鳞，仙儿与你好歹也是姐妹，而且，迦南学院的天焚炼气塔，对你来说，也有不少溢出。”
<br/>　　沉默片刻，彩鳞温柔的声音再次响起：“记得，早点回来。”
<br/>　　“遵命，我的女王大人。”
<br/>　　说罢，慕清霄就毅然转身，离开住处，周围空间一阵扭曲，身影就自原地消失。
<br/>　　“系统，我需要暂时离开本位面。”
<br/>　　“宿主并未完成全部任务，离开本位面，需要支付双倍掠夺积分。”脑海中，久违的响起系统的提示声。
<br/>　　闻言，慕清霄深吸了口气，道：“说吧，需要多少？”
<br/>　　“宿主想暂离本位面，需要支付两万掠夺积分。”
<br/>　　两万掠夺积分，说多不多，说少也不少，但相比起完成任务后，获得的巨额积分，物超所值。
<br/>　　“扣吧。”
<br/>　　“叮，扣除成功，宿主可随时离开本位面。”

<br/>　　片刻后，一只三四丈高的白猿，极其狼狈的逃了出来，望着山洞，满是愤怒，脸上却挂着一副生无可恋的模样。
<br/>　　三番两次的抢老子的山洞，灵药都被你们取走了，现在连山洞都不放过，真是欺猿太甚。
