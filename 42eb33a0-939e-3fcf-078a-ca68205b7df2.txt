　　狠人大帝，也就是荒主，她此刻的状态并不是很好，被无数仙金组成的锁链封锁，或者说，是她自己将自己封印了！
<br/>　　这乍一听是匪夷所思，可这就是事实！
<br/>　　女帝如今的状态绝对不算太好，当然，也不算太差。她的道果此刻并没有在身上，所以导致她的神魂时而清晰，时而模糊。她这是害怕自己在这段时间内做出什么自己不愿做的事，所以将自己封锁了。
<br/>　　混沌道君定眼一看，当然，看到的不仅是女帝的风华绝代，还有她的修为以及她的……一生！
<br/>　　混沌道君一眼看去，便能算出女帝的一生，从小到大，一世，两世，三世……女帝的一生，在道君眼中便没了秘密，他想知道什么，便能够知道什么。
<br/>　　看完后，道君也不得不在心中感慨。女帝无愧狠人之名，以一介凡体便能够走到如今这地步，真是恐怖。她如今的修为，已是半步大罗金仙了，也就是此界中的半步红尘仙。只待道果圆满，便可晋升有道果的大罗金仙。
<br/>　　半步大罗金仙，看似不怎么样，在洪荒更是多如牛毛。但那是洪荒，这里可是遮天界，一个大道不全，长生物质匮乏的低级神魔世界，是一个已经经受过不知多少纪元的世界，是一个资源消耗得差不多的世界。能够在这样一个世界中逆活数世，更是达到如此境界，那就恐怖了。
<br/>　　池塘里怎么可能养的出大鱼？这是世界限制了她，若是这样的人出生在洪荒，那成就，就算不成圣，也能达到混元大罗金仙了。
<br/>　　如她这样的人在遮天界还有一个，无始大帝。这样的人才，就应该利用起来，浪费是可耻的。
<br/>　　要不要将这样的人送到洪荒？道君如此想到，觉得很有趣，若是狠人大帝，无始大帝这等人到了洪荒，想必洪荒会热闹无比，更加有趣。只是，洪荒世界外的保护罩能让其他世界之人进入吗？道君有些不确定。
<br/>　　他想了很多，这些都只是在瞬息之间发生，他面色上并没有出现什么变化。道君的目光依旧看着女帝。
<br/>　　这时，女帝似有察觉，她那一直模糊茫然的目光突然泛起异样光彩，也许是一瞬，也许是永恒。女帝的目光一下子变得凌厉起来，天地为之震动，道则为之颤抖，好似一尊无上恐怖的存在在觉醒！恐怖的气场涌向混沌道君，但到了他身边，便自然而然的分散开来。
<br/>　　仅仅只是一瞬，天地便再次恢复平静，好似什么也没有发生过。
<br/>　　女帝恢复原样，只是之前那双茫然的眼睛已不复茫然，也没有变得璀璨夺目，而是变得平平淡淡，普普通通，双眼中存不下任何东西。那漂亮得不像话的身姿，出尘而淡雅，明明就在眼前，却又好似身在仙界。
<br/>　　这才是狠人大帝！
<br/>　　真正的风华绝代，她看是这般漂亮，风姿绝世，可是一旦凌厉起来，这天地都要毁灭，古今鲜有人可阻挡。
<br/>　　她淡淡的盯着混沌道君，尽管他看不透他的实力，但她知道他比她强，她眼中也并没有露出丝毫的情绪波动，依旧那般淡然，那般漠视。
<br/>　　“不为成仙，只为在红尘仙等你归来！”蓦然，混沌道君开口说了这么一句。
<br/>　　女帝目光微微波动，闪烁几下，口中喃喃道：“归来……哥哥……”紧接着，她再次恢复淡然。
<br/>　　“道友可曾想过，你要等的他，若不是‘他’要该怎么办？岁月悠悠，世间终会出现两朵相似的花，但也仅仅只是相似而已，你心中的‘他’未必会真正归来！”道君淡漠开口道。
<br/>　　女帝目光顿时变得凌厉起来，空间破碎，乾坤震动，她的目光好似两道锐利的剑气，洞穿虚空，直直朝着道君射来，恐怖无比！
<br/>　　道君微微一笑，也不见他有什么动作，那两道凌厉的目光便化为虚无，空间瞬间恢复平静。女帝此刻目光微凝，眉头一皱，想要说些什么，但终极没有开口。
<br/>　　“道友可曾想过亲自复活你所要等之人？”道君继续说道，女帝将目光投向他，让他继续说。
<br/>　　道君也不恼，继续说道：“想必道友也尝试过，可是不能。那道友你有没有想过，不是不能复活，而是其他问题，比如……你的实力还不够强大。”
<br/>　　女帝目光闪烁，显然她也知道这些。道君不理会她，继续说道：“当实力强大到一定程度时，可是无所不能的，什么复活死人，那是再简单不过的事情了。”
<br/>　　道君说到这里，明显一顿，紧接着，他单手向着虚空一划，划开一道口子，他不停，继续向着虚空一抓。
<br/>　　顿时，天地风云变幻，道则颤动，乾坤震荡。道君目光看向虚空，便垂下一道光幕，将四周虚空遮蔽，天道不能进，天机不能显。
<br/>　　一道略显虚幻的岁月长河散发着岁月的气息，于虚空之中延伸出来，渐渐变为凝实。大河滔滔，岁月悠悠，贯穿古今未来，无数人的影子在里面挣扎。
<br/>　　在这条长河里，可以看到过去，现在，未来，无数古人，无数今人，无数未来之人。
<br/>　　只要是没有跳出时间长河的，所有的一切，无论是人还是物，俱在其中挣扎。
<br/>　　岁月长河刚一出现，虚空便震动不已，好似是经受不住这般强大的力量，要倒塌毁灭。还好道君已将这片虚空定住，同时也遮蔽了天机，阻断了天道，否则此刻便是天下震动，大劫降临了。
<br/>　　岁月长河乃逆天之物，不得显现于人前，否则必遭天谴，除非是能够抵抗天道的，否则不得轻易唤出岁月长河。
<br/>　　女帝目光凝重，看向虚空中的岁月长河，眼前这人到底有多恐怖，竟能够召唤出传说中的岁月长河，而且看他的样子还毫不费力，这怎么可能？
<br/>　　不论她相不相信，混沌道君的确是唤出了岁月长河，他目光一凝，伸手一拉，岁月逆转，一道道场景在虚空之中浮现，就如同银幕上的电影在倒退一般。
<br/>　　时间在道君面前，被他随意拉伸，就如同看电影一般，既能快进，也能暂停，更能倒退。
<br/>　　岁月逆流，场景后退，两人看见了万古青天一株莲的青帝，看见了一生大败的乱古大帝，看见了仙路尽头谁为峰，一见无始道成空的无始大帝，看见了狠人大帝的数世，开创度神诀的南岭天帝，混沌成帝的不灭大帝，吞尽诸天的吞天大帝。
<br/>　　最后，他们看到了一个少年，少年登临祭坛，眼中满是泪水。
<br/>　　“我要死了，谁能……帮我照顾妹妹？”
<br/>　　“神血、妖血、佛血……都已浇灌在它的身上，马上就要轮到我了。死不要紧，可谁能帮我照顾妹妹，她还小，我放心不下。”
<br/>　　带着遗憾，带着不甘，带着哀求，少年终将陨落。
<br/>　　“轰！”
<br/>　　看到这里，女帝身上爆发出无与伦比的恐怖气势，震动诸天。道君挥手，将岁月长河散去，同时将女帝散发的毁灭气势也散去。
<br/>　　微微一笑，道君淡然道：“本道君能够轻易复活他，可，道友你不想靠自己复活他吗？”
