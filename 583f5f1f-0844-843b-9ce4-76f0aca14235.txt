　　上封天地，下定山水。一个牛逼的地师，本事不亚于任何一个道法通天的法师。在百年后的当代，地师一门行踪更为诡秘，叶少阳也只是听过几个名字而已。四宝在风水学上的造诣，其实应该勉强算得上一名地师，不过也是初级的那种。
<br/>　　“那个紫狂真人现在何在？”叶少阳问。
<br/>　　“大概二十年前殁了。”道渊真人说。
<br/>　　三足金蟾叹道：“若是紫狂真人在，又怎会让你们如此对我……”
<br/>　　叶少阳踢了他一脚说道：“你就拉倒吧，你也就是随便被人家拉来的一个精怪，给你个神位，你个小人得志的，就在这作威作福，祸害一方！”
<br/>　　在听了三足金蟾的讲述之后，叶少阳算是明白了，之前他还以为这三足金蟾有多大的背景，心中多少有点敬畏，原来只是那位紫狂真人在这个山水局中的组成部分，用来吞吐死气而已。说白了，只要是妖物，这活换谁都能干，给他保举神位，估计也只是想拴住他长期在这里呆着，没想到这蛤蟆精一点都不安稳，还真把自己当成一方霸主，开起了后宫。
<br/>　　叶少阳真是越想越来气，不过却已经没有杀了他的心思了，倒不是别的，而是认定他是一个小角色，实在没有必要为了他去让毛小方和道渊真人受到牵连，而且三足金蟾已经写了罪状，又有萧逸云亲自押送，从中作梗，肯定不会让他好过。
<br/>　　“对了，古墓不是一直都被镇压吗，百鬼夜行是怎么回事？”叶少阳想起这件事，问道。
<br/>　　“那是……炼尸缸里的鬼魂跑了出来，兴许是，不过炼尸缸的势力范围，并不能离开那个山谷。”三足金蟾老实地回答。
<br/>　　“一直都是这样吗？”
<br/>　　“不是，以前一直都很太平，也是最近才发生这种事，本王……”
<br/>　　叶少阳一脚踢过去，“你就一个山神，别本王本王的，有事说事！”
<br/>　　三足金蟾心中愤恨，但也不敢多言，只能继续回答道：“我也不知道为什么会这样，那古墓我不敢进去，好在一直相安无事，我已经想办法知会了少主人，他应该不日就赶来了。”
<br/>　　“谁是你少主人？”
<br/>　　“紫狂真人的隔代传人，妙心真人。当年紫狂真人走之前，特意给我一枚玉符，声明如果山水阵出了问题，让我捏碎玉，他即刻会感觉到，就算他不在人间，玉符的母符也会有人接管。大概三五年前，妙心真人来过一次，检查了古墓，她是紫狂真人隔代传人，我承蒙真人封赐神位，她自然是我少主人了。古墓的阵法出问题之后，我自然就通知她了。”
<br/>　　叶少阳又问了一些与之相关的问题，三足金蟾也回答不上来，叶少阳于是结束了问话，跟萧逸云说了几句，让萧逸云送三足金蟾去阴司报道。
<br/>　　萧逸云走之后，叶少阳转头看去，这才发现了在一旁呆若木鸡的吴巧，叶少阳冲她笑了笑，道：“你干的不错，之前多亏你了。”
<br/>　　吴巧听见他的问话，这才回过神来，望着他怔怔地说道：“五通神被带去哪里了？”
<br/>　　“去阴司接受惩罚去了。”
<br/>　　“阴司……”吴巧一张脸吓的煞白，“刚那个白脸的是谁？”
<br/>　　“你一个人类，不必知道这么多。总之……没事了，你的贞洁也保住了。”
<br/>　　吴巧脸上一红，有些忐忑地说道：“五通神不会再回来了吧？”
<br/>　　“当然不会。他要在地府受点罪了。”叶少阳笑笑，“你可以回家了。”
<br/>　　“回家……我怎么跟我父亲说，还有，你们怎么跟镇上的人交待五通神这件事？”
<br/>　　叶少阳挠了挠头，这个问题，确实之前没有想到：五通神被抓走了，吴巧现在回去，镇上的人一定会起疑，到时候发现五通神不在，说不定会逼问她发生了什么事。
<br/>　　实话当然是不能说的，叶少阳跟大伙商量了一下，决定编造一个五通神被阴司来者抓走的谎话，理由当然是他在人间干坏事，霸占人家姑娘，阴司忍不住了，所以把他带走……把责任推给阴司。
<br/>　　反正也没有别的人证，只要吴巧一口咬定，别人也不能说什么，何况她一个弱女子，没人会怀疑她能把五通神怎么样，自然也不会为难他。
<br/>　　叶少阳教吴巧怎么样去说，跟她对好了口供，吴巧牢记于心之后，叶少阳表示让她自己回家去。吴巧有点不敢，叶少阳正好也要跟毛小方和道渊真人回镇上去商量别的事，于是送她回去，从五通神的房间刚出来，看到外面院子里的地上躺着一个人，叶少阳一看，是那个陈老汉，附身趴在地上。
<br/>　　叶少阳上前查看，陈老汉只是昏过去，忙问这是怎么回事，美华解释，之前叶少阳等人在房间里斗法的时候，京东了陈老汉，过来查看动静，正好美华赶来，干脆打昏了他。
<br/>　　叶少阳计上心头，让美华附身一次，在陈老汉的梦境中化身地府的阴神，告诉他五通神因犯仙家戒律，已经被革职送往阴司查办，他是这里的庙祝，他说话，镇上的人更会相信，正好也与吴巧说的吻合，一定会让人深信不疑。
<br/>　　于是留下美华作法，其余人离开了蛤蟆峰，一路赶往镇子。快到镇上的时候，叶少阳担心镇上的人看到吴巧跟他们在一起，引起怀疑，让吴巧先回家，照计划行事，自己三人前往客栈，分开行走。
<br/>　　到了镇上，叶少阳突然想起一件事，带着两人来到镇口，询问当地人，找到了进入镇子的毕竟之路，在路边砍了两根柳条，交叉在一起，打了一个法结，然后插在路边一个显眼的位置。
<br/>　　“你弄这个是给谁看？”道渊真人问道。
<br/>　　“那个什么妙心真人，如果她真的要来的话，必然从这里到镇上，看见这个，肯定去会找我们。”叶少阳用柳条打的这个法结，是法术界的一种通用信号：在一个地方插上这东西，表示这一带有法师占据，正在办事，后来的法师看到这个，就要赶快前去相见。
