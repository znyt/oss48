　　“凭什么？”火牙宗主，脸上尽是不屑，甚至是不以为意。
<br/>　　“就凭你这个乳臭未干的丫头？”
<br/>　　“当然不是。”廖灵儿冷声说着，看向身后一众武者。
<br/>　　“诸位，现在可看清火牙宗的真面目了？”
<br/>　　“这个外表光鲜的火牙地域第一宗门，不过是个邪修集聚的分部。”
<br/>　　廖灵儿指着火牙宗主，“这个人，看似一代强者。”
<br/>　　“这个宗门，看似举行盛事，为了提升火牙地域各武者的实力，实则，根本是个幌子。”
<br/>　　“举行火牙盛事，不过是他们更快地击杀火牙地域的强者和天骄的手段。”
<br/>　　“每次火牙盛事，多为危险难度，动辄死伤过半。”
<br/>　　“每次武者进入火牙大阵之内，看似殒命于大阵，实则，大阵之内早有禁制，会随意将过半武者送入这片山洞之内。”
<br/>　　“这群邪修，便是在这山洞之内，将这些武者击杀，以武道冰晶吸收他们的武道力量。”
<br/>　　周遭武者闻言，脸色发黑。
<br/>　　他们没想到，火牙宗的手段，如此高明。
<br/>　　每次火牙盛事举行，来自各地的天骄和强者，足有数万人。
<br/>　　过半武者，则是一万有余，近两万。
<br/>　　火牙宗此举，等于每隔一段时间便一网打尽，比邪修在外单独一个个击杀快多了。
<br/>　　偏偏火牙宗打着火牙盛事的名号，又已说明了火牙盛事的危险。
<br/>　　这样一来，即便每次火牙盛事死伤过半，也无人会怀疑火牙宗分毫。
<br/>　　火牙宗，仍旧是那个高高在上，人人敬仰的大势力。
<br/>　　火牙宗，确实好算计。
<br/>　　“还有。”廖灵儿继续冷声道，“诸位当真以为，火牙大阵是随火焰颜色而改变难度？是凭借运气？”
<br/>　　“火牙盛事，每逢赤色火焰，便是难度最低。”
<br/>　　“参加者，几乎九成以上都安然出来。”
<br/>　　“但这般情况，数次盛事都难遇一次。”
<br/>　　“你们以为这是火牙宗大发慈悲？”
<br/>　　廖灵儿冷笑一声，“大阵，早就在火牙宗主掌控中；难易与否，也只是他一念之间。”
<br/>　　“特地数次盛事，便有一次难度最低，不过是要掩人耳目罢了。”
<br/>　　“火牙分部，统领周遭数百地域的邪修分部。”
<br/>　　“每隔数年时间，这些分部，就会将已经装满武道力量的武道冰晶，统一送来此地。”
<br/>　　“故每隔数年，就会有大批邪修，从四面八方赶来，上交这些武道冰晶。”
<br/>　　“如此多的邪修赶来，如何能无声无息，不让人发现？”
<br/>　　“很简单。”廖灵儿冷声道，“便是借着火牙盛事，无数武者赶来之时。”
<br/>　　“各分部邪修，乔装成正常武者，进入火牙宗。”
<br/>　　“我们各个武者在宗门广场等待盛事之时，各地邪修，早已进入这山洞之内。”
<br/>　　火牙宗的高明算计，似乎超过了所有人的想象。
<br/>　　每次举行火牙盛事，不过是火牙分部本身收集各天骄和强者的武道力量的途径。
<br/>　　而数次盛事之后，其余各地邪修分部，也存够了足够多的已装满武道力量的武道冰晶，便一并上交来火牙分部。
<br/>　　数次盛事之后，出现一次最低难度，不过是让各地邪修能随无数正常武者潜入火牙分部。
<br/>　　“哈哈哈哈。”这时，传来一阵阴冷大笑。
<br/>　　火牙宗主冷笑着，“你把所有人带来这里，就是要揭穿我们火牙宗的真面目，让他们给你报仇？”
<br/>　　“不错。”廖灵儿冷眼看着火牙宗主，“火牙大阵之内，本就有禁制。”
<br/>　　“我不过是以空间手段，强行触发了这一禁制，送所有人进来。”
<br/>　　廖灵儿，自然是没有传奇强者那等手段的，她不过是个圣王境九重武者。
<br/>　　她只是以自己的武魂手段，触发火牙大阵之内的禁制罢了。
<br/>　　那时，火牙大阵在全力操控九牙火兽，本身大阵的力量已无比虚弱，这才让她找到了机会。
<br/>　　“说起来，还是前辈你给了我一并下品圣器，我才能那么快刻画好我的手段阵法。”廖灵儿喜悦说着，回过头。
<br/>　　“嗯？前辈呢？”
<br/>　　廖灵儿本想看向萧逸，但她却蓦地发现，这位前辈，不知何时起，早已不见了踪影。
<br/>　　她还来不及多想，已注意到身后一众武者面露冷色。
<br/>　　廖灵儿笑笑。
<br/>　　火牙宗主，同样笑笑，“不错，不愧是那个废物的妹妹，这般空间手段，倒也不俗。”
<br/>　　“只是。”火牙宗主阴测测道，“你费尽心思，送所有人进来这里替你报仇，你可曾想过，他们愿意帮你报仇吗？”
<br/>　　“嗯？”廖灵儿脸色一惊，她忽然发现，周遭武者的冷色，根本不是看向火牙宗主，而是看向她。
<br/>　　“你们…”廖灵儿脸色一变。
<br/>　　火牙城少城主、三大家族的强者等等，尽皆脸色难看到极点。
<br/>　　“廖姑娘，我们不知晓你和火牙宗的恩怨，你何必这般害我们。”
<br/>　　“你和火牙宗的恩怨，与我们何干？”
<br/>　　“你们。”廖灵儿沉声道，“火牙宗的邪修，难道不该杀？”
<br/>　　“睁大你们的眼睛看清楚，那一块块武道冰晶，可是沾满了我们火牙地域武者的鲜血。”
<br/>　　不错，山洞一侧，堆满了无数武道冰晶。
<br/>　　这是火牙宗与周遭数百地域存了数年才存起来的武道冰晶。
<br/>　　这些武道冰晶，已装满了一位位天骄，一位位强者的武道力量。
<br/>　　这一块块武道冰晶，堆积成山，何止一万之数？
<br/>　　“我们拿什么帮你报仇？”一众武者，虽脸色难看，但也脸色冰冷。
<br/>　　“你们怕什么？”廖灵儿气急，“你们是各大势力的强者，各大势力的天骄，火牙宗敢动你们不成？”
<br/>　　“你说呢？”为首的火牙城主府强者与三大家族强者，一片灰丧之色。
<br/>　　“如你所说。”火牙宗主冷笑一声，“他们不过是各大势力的少家主和一些长老强者。”
<br/>　　“若是他们的家主、城主在此，我自是忌惮。”
<br/>　　“即便如此。”廖灵儿同样冷笑，“若他们悉数在你火牙宗死了，后果…”
<br/>　　“我还有得选择吗？”火牙宗主，打断了廖灵儿的话。
<br/>　　对，他还有得选择吗？
<br/>　　这过万武者，已见到了他们所有邪修。
<br/>　　火牙宗，除了杀光这里的所有武者，还有别的选择吗？
<br/>　　没了。
<br/>　　廖灵儿一愣，下一秒，脸色一灰。
<br/>　　“臭丫头，你还太嫩了。”火牙宗主不屑一笑。
<br/>　　对，廖灵儿想得太理所当然了。
<br/>　　她想到如何揭开火牙宗的真面目，却没想到如何应付火牙宗的一众邪修。
<br/>　　她甚至没有发现，火牙宗主从一开始就没有解释半句，更没有半分脸色变化。
<br/>　　因为，从一开始，火牙宗主就已下了决定，今日这里的武者，一个都活不了。
<br/>　　“也罢。”火牙宗主冷笑一声，“反正是要送这些武道冰晶回总府的。”
<br/>　　“走之前，多添一万天骄和武者的武道力量，倒也是好事…”
<br/>　　火牙宗主这般说着，但话语，忽然停顿。
<br/>　　他的目光，看向山洞一侧。
<br/>　　不，准确来说，是看向山洞一侧的一个黑袍武者身上。
<br/>　　“你做什么？你是何人？”火牙宗主脸色一惊。
<br/>　　轮不得火牙宗主不惊，因为，黑袍武者，就在那堆庞大的武道冰晶之旁。
<br/>　　“哦？你们聊完了吗？”黑袍武者淡漠地看了眼火牙宗主。
<br/>　　淡漠的语气，几乎让人心头一颤。
<br/>　　“你到底是什么人？”火牙宗主，脸色大变。
<br/>　　以他的实力，竟然一直没有发现这黑袍武者何时出现在那堆武道冰晶之旁？
<br/>　　“我？呵。”黑袍武者淡漠一笑，手中火光一闪，随后指尖一点。
<br/>　　哗…
<br/>　　身上黑袍，轻轻一扯，就此甩掉。
<br/>　　黑袍之下，是一个一身黑色劲装，戴着面具的年轻人。
<br/>　　“猎妖师，易霄。”萧逸的语气，一如既往地淡漠。
<br/>　　但听在火牙宗主以及周遭武者耳中，却不亚于一句让人内心震颤的话语。
<br/>　　嗖…几乎是一瞬间，火牙宗主身影一闪，瞬间来到了廖灵儿身旁。
<br/>　　“原来是你。”火牙宗主盯着萧逸，双眼一眯，“我没记错的话，这丫头原本一直在你身旁吧。”
<br/>　　火牙宗主，扣住了廖灵儿的咽喉。
<br/>　　“立刻给我退离武道冰晶之旁，否则，这丫头…”
<br/>　　“她的性命，与我何干？”萧逸淡漠地打断了火牙宗主的话语。
<br/>　　“这些武道冰晶，你没机会带回邪君府了。”
<br/>　　嘭…手中一道紫炎凝聚。
