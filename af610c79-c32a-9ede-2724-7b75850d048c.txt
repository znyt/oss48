        &lt;h3&gt;第三卷第二百五十六章  送资料&lt;/h3&gt;
<br/>
<br/>        江俞轩开着车到了张函和张倩楠出差的地方，他打电话给张倩楠，等着门卫放行，要不然他连大门都进不去。
<br/>
<br/>        下车做了登记，然后将车开进院子停好车拿起文件袋，文件袋里是夏颜的一些资料，他也是找人花了一番功夫得到的。
<br/>
<br/>        有可能这些资料对现在的夏颜起不了太大的作用，但是能让夏颜收敛一些也好，最起码给陈子昂和西城分公司一点缓冲的时间。
<br/>
<br/>        江俞轩拿着资料敲开了张倩楠办公室的门。
<br/>
<br/>        张函正在办公桌前忙碌着，张倩楠打开门，看见门口站着的江俞轩，波澜不惊的说了一句：“你来了。”
<br/>
<br/>        “嗯，来看看你们。”
<br/>
<br/>        “坐吧，我把手中的这个文件写完，马上就好。”张函头也不抬的对着门口的江俞轩说。
<br/>
<br/>        “好，你们忙你们的。”江俞轩在沙发上坐下，他第一次来他们的办公室。
<br/>
<br/>        办公室很大，有内外套间，只有张倩楠和张函俩个人，办公设备简简单单，张函和张倩楠的办公桌上都堆着厚厚的档案。
<br/>
<br/>        张倩楠给江俞轩倒了一杯茶又回到了办公桌前，似乎江俞轩来和她没有关系。
<br/>
<br/>        办公室只听见张函敲击键盘的声音。
<br/>
<br/>        “俞轩，好久不见。”张函站起来走到江俞轩的面前。
<br/>
<br/>        张函本身很高，现在更是站在江俞轩的面前，江俞轩感觉他要昂着头看着张函，这让他心里有点不舒服。
<br/>
<br/>        江俞轩站起来：“好久不见，你好像瘦了很多？”
<br/>
<br/>        张函拉了一张凳子坐在了江俞轩的对面：“坐吧，你是来找我的？”
<br/>
<br/>        “我是来找倩楠，顺便看看你。”江俞轩看了一眼依然在忙碌的张倩楠。
<br/>
<br/>        “最近确实很忙，我们两人从上班到现在都没有休息，委屈你了。”张函微微一笑，拿起了江俞轩放在茶几上的文件袋。
<br/>
<br/>        江俞轩的神情一滞。
<br/>
<br/>        张函打开文件袋看了看：“找这些资料费了一些番手脚吧？”
<br/>
<br/>        “我是找别人帮忙的。”江俞轩低头喝了一口茶。
<br/>
<br/>        “陆家知道我和子昂有孩子的事情了，可能也会牵扯到你，你注意点。”张函从弥青云的口中知道了夏颜和冯睿去找了蔡少芬。
<br/>
<br/>        “和我没有多大关系吧。”
<br/>
<br/>        “你能调查这么多，能和你没有关系？”张函盯着江俞轩看。
<br/>
<br/>        “我是为你着想，陈子昂不是你深爱的人吗？我前几天还见了夏颜，感觉夏颜和陆家走得很近，而且最近好像有意识的针对陈子昂。”江俞轩辩解着。
<br/>
<br/>        张倩楠停下敲击键盘的手，静静的放在键盘上。
<br/>
<br/>        电脑上是他们搜集的关系图，有了这些图，思路就清晰了很多，而张函的工作也能找到着手的点。
<br/>
<br/>        听到江俞轩的话，她抬起手，想敲一下回车键，但是抬起的指头颤抖了一下，她看了一眼张函，张函清瘦的脸庞棱角分明，表情平静。
<br/>
<br/>        张倩楠垂下眼眸，睫毛轻轻的颤抖着。
<br/>
<br/>        “这份资料有可能对夏颜的作用不大，也有可能会激起一些浪花，如果夏颜舍弃亲情或者是友情，那么夏颜会高枕无忧。”张函分析着。
<br/>
<br/>        “倩楠，你看看这份资料。”张函将资料递给张倩楠。
<br/>
<br/>        张倩楠打开看了看：“我会将这份资料递给相关的人员。”
<br/>
<br/>        “最近还好吧？”张函问江俞轩。
<br/>
<br/>        “你是问我呢，还是问陈子昂？如果是问我，我一切都很好，如果是问陈子昂，我也是好久没有看到她了，不知道她近况如何，不过听文文说，他们母子都很好，好像在搞什么研发。”江俞轩就像是在汇报工作一样。
<br/>
<br/>        “你们都要好好的，只要你们都好好的，我也就安心了。”张函深吸一口气。
<br/>
<br/>        “你也要好好的，注意身体，工作再忙都抵不上一个好身体，再说，你有个健康的身体才能守护你要守护的人。”江俞轩想起杨总的体检，叹了口气。
<br/>
<br/>        “其实，我很羡慕子昂有你们这一帮好兄弟、好朋友，携手与共，不离不弃，不像我总感觉是孤军奋战。”张函笑着说。
<br/>
<br/>        “你怎么会是孤军奋战呢，你还有我和倩楠，还有千千万万个站在你身后的人，当然还有你的孩子和你的爱人。”江俞轩感觉张函有些忧伤，便出言安慰。
<br/>
<br/>        “是啊，函哥，你还有我们啊，你怎么会是孤军奋战呢？难道我这一段时间陪着你加班加点都白瞎了吗？”张倩楠走过来坐在江俞轩的身边。
<br/>
<br/>        “俞轩，这一段时间都很忙，倩楠也没有联系你，是不是怪我这个大哥了啊？”
<br/>
<br/>        “没有，没有，我知道你们忙，所以我也没有敢来打扰你们的工作。”
<br/>
<br/>        “你上次去北京事情办完了吗？”张倩楠问江俞轩。
<br/>
<br/>        “没有，上一次本来想去公司辞职的，我的精力还是有限，但是这边事情紧急，我第二天一大早就回来了。”江俞轩撒了个谎。
<br/>
<br/>        “这么说，你还是要身兼数职在陈子昂的公司呆下去了？”张倩楠拿起茶壶给江俞轩和张函各添了一点茶水。
<br/>
<br/>        “嗯，西城这边目前我还兼职着。但是大多数事情还是陈子霖在干，我最近事情也特别多。”江俞轩解释着，似乎他一直在解释着什么，回避着什么，这种感觉不太好，江俞轩皱了一下眉头。
<br/>
<br/>        沉默了一会，江俞轩说：“倩楠，我准备在北京买套房，你喜欢哪里的地方？”
<br/>
<br/>        “买房干什么？”张倩楠淡淡的问。
<br/>
<br/>        “我们住啊，你不可能一直住在单位或者是娘家吧？”江俞轩看了一眼张函，又看了一眼张倩楠。
<br/>
<br/>        “目前我还没有想要住哪里，最近事情多，你看看我到处跑，一时三刻怕也固定不下来，而且你也一样，你在西城，我在北京，也不太好，暂时就不用买房了。”张倩楠笑着对江俞轩说。
<br/>
<br/>        “那随你吧，你什么时候想好了什么时候告诉我。”江俞轩心里一松。
<br/>
<br/>        “好！”张倩楠答应得干脆利索。
<br/>
<br/>        “我还有事情，我就不打扰你们工作了，倩楠想回家了给我打电话，我来接你。”江俞轩站起来向张函和张倩楠告别。
<br/>
<br/>        江俞轩走后，张函对张倩楠说：“你怎么回事？”
<br/>
<br/>        “你的那一番话提醒我了，我不想欺骗大家。我和江俞轩还没有到很爱的那种地步，先就这样吧，我们继续工作，我陪你去甸城，我们没有必要把时间浪费在无谓的事情上，我们的工作不仅仅是为了保护陈子昂，你来看看这些错综复杂的关系网，你看看这些可怕的数据。”张倩楠转身走到电脑前，将她做好的数据图打开让张函看。
<br/>
<br/>        “哥，我们张家一直是守护者，我们需要守护着这片净土，保护他们的清白，维护着他们的安宁，虽然我和你都是不起眼的小人物，哥，我们抬头能看到我们飞扬的红旗，低头能看到我们衣服上的标志。哥，我会和你一起努力！”
<br/>
<br/>        “倩楠！”
<br/>
<br/>        “哥，我姓张，我叫张倩楠，我是张家的孩子！我们生下来就有我们的责任和担当！”
