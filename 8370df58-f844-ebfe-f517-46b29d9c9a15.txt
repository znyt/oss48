　　叶少阳贴了一张感召符在鬼魂的额头上，闭眼念咒，用罡气感知……过了一会，睁开眼睛，对一脸好奇的毛小方说道：“都是半魂鬼，丢失了人魂。”
<br/>　　“怪不得没有神识。”毛小方恍然。
<br/>　　鬼魂有三魂七魄，每一道魂魄都有自己的作用，少了哪一道，都会有不同的表现，其中人魂主的是神识，人魂丢失，如果是活人，不影响身体机能，但是会成为痴呆，就算变成鬼也一样，没有了人魂，就失去了一切神识，只是囫囵地凭着一些本能做一些行动，这样的鬼魂，特别容易被邪修的法师操控，被当成傀儡。
<br/>　　毛小方当下就提出了这种可能。
<br/>　　叶少阳看着山谷中数不清的鬼魂，可以用漫山遍野来形容了，深吸一口气说道：“不会有什么邪修法师，有这么大的能力，杀这么多人吧？”
<br/>　　“我看也不会，不管在哪个时代，如果有这么大规模的屠杀灭魂，就算人间的官府不管，法术界绝不可能不知道，阴司也不可能放任不管。”毛小方摇了摇头，“这件事，诡异得很啊，贫道我也是头一回遇到。”
<br/>　　见叶少阳沉吟不语，想到什么，对叶少阳说道：“少阳子不必为刚才的事自责了，半魂鬼，没法超度的，碎魂重生，反而是一种归宿。”
<br/>　　叶少阳往前走了几步，又回到山谷的阴气范围，蹲在地上，看着那些从地上冒出来的“怨气”，现在已经知道，这其实不是怨气，而是真实的鬼魂，心中更是多了几分疑惑，提议跟毛小方再从山谷里走一遍，注意观察地下，寻找阴巢所在，结果两人刚走了不到几十米，突然之间感到风动，抬头一看，弥漫在山谷中的阴气好像漩涡一样朝着中间集结，两人还没明白怎么回事，一阵撕心裂肺的惨叫声，便在山谷之中弥漫开来。
<br/>　　这种惨叫，有点像是人在经受酷刑之中发出的声音，还夹杂着痛苦的哭泣声，听上去极为刺耳。如果两人不是心理素质极好的法师，换成普通人，这会儿只怕早就吓尿了。
<br/>　　饶是如此，两人还是感到了相当的震惊，朝左右看去，那些原本在空中漂浮的鬼魂，突然之前都停了下来，脸庞缓缓转到了他们这边，仿佛已经发现了他。
<br/>　　那些马赛克一般模糊的脸庞，出现了扭曲和愤怒的表情，一双双眼睛都闪烁着红光。
<br/>　　“这……不妙啊少阳子！”
<br/>　　“跑！”
<br/>　　叶少阳吐出这个字，立刻转身就跑。这时候那些失去人魂的鬼魂，仿佛锁定了目标，一个个发出尖叫，朝着二人扑了过来，堵住去路。
<br/>　　“先不要伤了他们。”叶少阳说完，一扬手，打出了六张灵符，在空中旋转发光，朝着谷口的方向推出去，所产生的灵力，将附近的鬼魂都推了出去。
<br/>　　叶少阳和毛小方一起跟着符阵后面朝山谷外撤退。
<br/>　　他们怕的根本就不是这些鬼魂，就算是目前叶少阳只有真人的实力，这些最低级的鬼魂对他也没有什么杀伤能力，但是这些没有了人魂的亡灵，居然会在同一时间发现自己，并且对自己展开进攻，绝对是受到了不知道是什么力量的驱使。
<br/>　　事后叶少阳想起来，触发点多半是自己杀了那个鬼魂，可能是让那股力量觉察到了自己的存在，才对自己发动了袭击。
<br/>　　眼看着就快到谷口了，突然之间，那股集结了的阴云从侧面飘过来，挡住了谷口，六道灵符撞击在阴云上面，立刻就被毁了——不是燃烧，而是像被什么东西腐蚀融化了。
<br/>　　叶少阳和毛小方猛然站住，望着这团漂浮在前方的阴云，不管是阴气还是什么，就算是被什么操控，都不可能产生太强大的力量，所以，叶少阳清楚，这块阴云之中，一定是藏着什么东西！
<br/>　　叶少阳也不知道里面有什么，正因为不知道，才更加让人感到恐惧。
<br/>　　他跟毛小方互相看了一眼，毛小方从自己的褡裢里提出来一把金钱剑，对叶少阳点点头，然后冲了过去。
<br/>　　既然对方想要阻拦，那就只有冲出去，不然在这里继续被困，局面只会越来越坏。
<br/>　　“天地无极，自有神妙，金钱索魂，洞悉无形，急急如律令！”
<br/>　　在贴近那团阴云的时候，毛小方念动咒语，一转手中的金钱剑，上面的铜钱立刻发出哗啦啦的声响，灵光乍现。
<br/>　　毛小方手持金钱剑，冲进了阴云之中。
<br/>　　叶少阳也立刻跟了进去。
<br/>　　进入阴云之后，眼前顿时一片漆黑，身边狂风呼啸，吹的人站立不住。
<br/>　　“跟我走！”黑暗中毛小方拉了叶少阳一把，金钱剑一抖，在狂风中劈开了一条路，飞奔过去，就在这时，一道树藤般的东西从上而降，击打在金钱剑上，毛小方只觉得手腕一麻，发出一声闷哼，踉跄后退。
<br/>　　这时候叶少阳打出了八张灵符，燃烧起来，组成旋灯火，绕着两人旋转，旋灯火的亮光，将附近照亮，两人这才看清，在前方不远地方，立着一个“人”，看上去更像是怪物：浑身像是被扒了皮一样，鲜血淋漓的，两只手极长，像个蛤蟆似的趴在地上，摇头晃脑，脸上像是被人砸了一锤子，已经烂得没有五官了，全是骨头碴子，中间黑乎乎的一片，好像爬着什么虫子，在裂开的脸庞中间爬进爬出。
<br/>　　叶少阳好久都没看过这么恶心的画面了，一时间有点反胃，情不自禁地骂了一声：“****！”
<br/>　　毛小方深吸一口气，说道：“这是什么鬼东西！”
<br/>　　“不知道。像是某种僵尸，看上去挺难对付的。”
<br/>　　叶少阳话音刚落，对面那具尸体突然腾空而起，扑了过来。
<br/>　　毛小方挥动金钱剑，砍了过去，结果那东西根本就没想硬拼，在空中来了个急停，口中吐出一道长长的舌头，瞬间长到巨大，对着金钱剑舔了一下。
<br/>　　“嘶……”
<br/>　　一阵腐蚀的声音响起，毛小低头看了一眼，几乎不敢相信：金钱剑的一面已经萎缩了，铜钱融化……
