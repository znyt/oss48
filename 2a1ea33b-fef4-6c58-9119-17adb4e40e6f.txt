　　“这上九天的洞府，已经被我定下了，王辉你就不用去了。”
<br/>　　突然一道声音，就在身边传来，这声音极为狂傲，似乎是丝毫不将王辉放在眼中。
<br/>　　顺着声音，只见几尊武者，突然走了出来，为首的一人，看起来是一尊中年男子，在他的背后，也跟着几名年轻人。
<br/>　　这几名年轻人身上的境界，都不算高，但是身上的气息却是一个比起一个恐怖，显然都是来前往这归墟城来，准备进入归墟天的天才。
<br/>　　“蔡寒，是你？”
<br/>　　王辉听到这声音，也是脸色猛然一变，变得冰冷许多。
<br/>　　“这洞府，我的这两位朋友已经定下了。”
<br/>　　“那就让他们滚。”
<br/>　　蔡寒还未开口，在他身后的一尊年轻男子，猛然站了出来。
<br/>　　这年轻男子负手而立，双眼便是扫视了楚风眠跟寒月漓一眼，眼神中露出几分不屑，没有将楚风眠二人，给放在眼中。
<br/>　　这种狂傲，便是展现的淋漓尽致。
<br/>　　可是这年轻男子，也并非是没有这狂傲的资本，此人的境界在生死境二重左右，但是这一身灵力，却是深不可测。
<br/>　　比起许多生死境巅峰的武者，都要灵力雄厚的多，几乎是一尊生死境无敌的人物。
<br/>　　“你的谁？”
<br/>　　楚风眠看着这年轻男子，平静的开口道。
<br/>　　“既然是来到这归墟城的，应该并非是无名之辈吧。”
<br/>　　“哼，告诉你也无妨，我们是南域玄天宗的弟子，既然知道了我们的身份，还不速速退去，将这洞府让出来，我们少不了你的好处！”
<br/>　　这年轻男子自傲的开口道。
<br/>　　南域，在九域之中，已经是足以称得上是中游，比起北域，西域，都是要强大几分。
<br/>　　而玄天宗，正是南域之中最大的宗门，在整个九域，玄天宗虽然是跟七大宗门相比，还稍逊几分，但是这几年的势头，已经是要逼近到七大宗门。
<br/>　　所以这些玄天宗弟子，除非是碰到七大宗门的弟子，还会注意一二，除此之外，任何人他们都不会放在眼中。
<br/>　　“玄天宗的人？没听说过，如此狂傲，我还以为是七大宗门的人。”
<br/>　　楚风眠撇了撇嘴，冰冷的开口道。
<br/>　　“滚，今日我心情好，不想出手教训你们。”
<br/>　　楚风眠的话一出，站在楚风眠身边的王辉脸色猛然变色了。
<br/>　　南域，本来就是要超越北域，本来在听到这年轻男子，是玄天宗弟子的时候，他的心中便是萌生退意，想要劝解楚风眠后退一步。
<br/>　　却没想到楚风眠居然如此嚣张的开口了。
<br/>　　这一句话说出来，那年轻男子的脸庞，都是不断的抽动着。
<br/>　　心中的怒火，一下子爆发出来。
<br/>　　“狂妄小子，好大的口气？不将我们玄天宗放在眼中？”
<br/>　　年轻男子先是一惊，随后猛然一步上前，死死的盯着开口道。
<br/>　　“跪下！”
<br/>　　“现在跪下，我可以饶你一命！”
<br/>　　听到这年轻男子的话，楚风眠的心中都是极为无语，这种狂傲，已经是一种自大行为了。
<br/>　　显然是这年轻男子，平日里威风惯了，这种高高在上的狂傲，好像是俯视众生，将楚风眠当做了一只蝼蚁了。
<br/>　　“话这么大，也不怕闪了舌头，跪下？你以为你是谁？”
<br/>　　楚风眠脸色露出几分阴冷，冰冷的开口道。
<br/>　　“一个默默无闻的小人物罢了，真以为自己已经是九域至尊？就算是九域至尊，也不敢让我跪下，你算是个什么东西！”
<br/>　　“嗯？”
<br/>　　听到这话。
<br/>　　不止是那年轻男子，他背后的另外两人，也都是脸色一寒。
<br/>　　其中一尊鹰鼻男子，更是阴阳怪气的开口道。
<br/>　　“木师兄，看来我们玄天宗还是太多低调了，一个蝼蚁，居然都敢来挑衅我们的威严，不如今天就给他一个校训，给那些废物一个警告。”
<br/>　　“一个蝼蚁？口气倒是不小，不过也就是一个能逞些口舌之勇的废物罢了。”
<br/>　　楚风眠冰冷的开口道。
<br/>　　这些玄天宗弟子，一上来比起一个狂妄。
<br/>　　跟这些人，没有道理可讲，不给这些人一个教训，这件事根本无法解决。
<br/>　　既然如此，楚风眠也不愿意在多浪费任何口舌。
<br/>　　“给我跪下！”
<br/>　　楚风眠直接一爪，猛然冲着那鹰鼻男子抓了过去，凌力如风，席卷而去。
<br/>　　“废物一个，也敢出手？给我死！”
<br/>　　鹰鼻男子冷喝一声，也是突然出手了。
<br/>　　在刚刚，他便是已经在积蓄这灵力，所以在这突然出手，一出手便是全力。

<br/>　　一拳打出，便是漫天拳风，足有千万道之多，令人想躲都不可能。
<br/>　　“诸天十拳！这是诸天十拳之一的满天拳！一拳打出，如同满天星辰轰击，根本无法抵挡！”

<br/>　　“玄天宗的人？再跟谁起了冲突？”
