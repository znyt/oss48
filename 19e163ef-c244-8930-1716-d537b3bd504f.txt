　　第332章
<br/>　　三天后的一个黄昏，严笛再次出现在城外。
<br/>　　“嗨，我又来了，没想到吧。”
<br/>　　严笛猛地回头，看到程大雷出现在自己身后，他张大的嘴巴很久没有合上。
<br/>　　“惊不惊喜，意不意外。”程大雷笑笑：“看把你高兴的。”
<br/>　　“你，你是个魔鬼！”
<br/>　　“不许这么说！”程大雷板起脸：“组织又有新任务交给你了……三天后我们还在这里见面。”
<br/>　　“三天之后又三天，到底什么时候才能结束，老大，我想让人知道，我是个好人。”
<br/>　　“好啦，好啦，我都清楚了。记得我上次对你说的，一名合格的间谍，要学会控制自己的情绪，你太容易冲动了。”
<br/>　　“这次什么任务？”渐渐严笛已习惯程大雷的说话风格。
<br/>　　“这一次……”程大雷顿了顿：“我需要琴川关的防御布置……”
<br/>　　琴川关每天都有人巡逻，城门更是重点防守区域，三队人马每天轮流警戒，每队都有一百人。他们的防守区域，换岗时间都属于绝对机密，至少，是不该被程大雷知道的机密。
<br/>　　严笛不想背叛莫鸣米，但最后鬼使神差，还是把自己知道的一切告诉了程大雷。
<br/>　　之后他失魂落魄的回到琴川关，城门的卫兵知道严军师这几日每天傍晚都会出去，大家自然也不会怀疑军师有什么问题。
<br/>　　“严军师，您看上去起色不是太好，是不是最近太忙了？”
<br/>　　“嗯呐。”严笛随口道了一声，摆摆手，踱着步走远了。
<br/>　　卫兵看着严笛的背影，轻轻叹了口气。将军生病，琴川关上下的事情都是有严军师处理……军师实在是太辛苦了。
<br/>　　严笛回到自己院子，刚才卫兵的话还响在耳边，他下意识站在铜镜前，看到镜中自己的模样时，自己也被吓了一跳。
<br/>　　两腮凹陷，颧骨高耸，头发干枯，下巴冒出稀疏杂乱的胡渣。
<br/>　　自己……何时变成了这般模样。严笛倒抽一口凉气，才想起那如梦魇一般出现的程大雷。不知不觉间，自己的精神已承受了太多压力。
<br/>　　严笛是个体面人，他向来是很注意仪表的人。今次，不知不觉间，他竟然憔悴至此，怪不得卫兵说自己气色不好呢。
<br/>　　不能再这样下去了。
<br/>　　他让书童进来，为自己束发，重新打好发髻，又仔细洁面刮脸，让书童取了一件干净袍子。
<br/>　　严笛这才觉得神清气爽，整个人也精神起来。想那程大雷也未必有什么可怕的，下次再见面，自己绝对不能被他气势所压住。
<br/>　　“军师，将军请你去府上吃酒。”
<br/>　　这时莫鸣米差人来请莫鸣米。
<br/>　　严笛理了理袍子，随着来人去到将军府。今次，只是私宴，酒席上只有严笛和莫鸣米二人。
<br/>　　莫鸣米注意到严笛的新袍子，道：“军师今天怎么换了新衣服？”
<br/>　　“啊……将军，我们刚遭大败，士气低迷，我们万不可也低落，该给手下人做个表率。”
<br/>　　“恩，军师言之有理。我这几日，当真是有些颓废，多谢军师提醒，以后绝不可如此了。”莫鸣米拍拍手，紧跟着有仆人送来一件新袍。
<br/>　　“这几日，我差人与军师做的，今日才做好，正准备送给军师。”
<br/>　　严笛感激得急忙跪倒：“将军大恩，属下结草衔环，百死莫报。”
<br/>　　“好啦，起来吧。”莫鸣米将严笛扶起：“今日只是私宴，只是随便饮些酒说些话而已。”
<br/>　　酒席宴间，莫鸣米难免长吁短叹，心中的苦水也向严笛倒了倒。
<br/>　　“我算什么……”莫鸣米酒喝得半醉：“我也只不过是人家手里一条狗而已，人家不想用我的时候，一脚就可以把我踹死。”
<br/>　　“将军，您醉了，要保重身体……”
<br/>　　“嗨，那程大雷是好欺负的么，从他第一天来到这儿，我就知道迟早要出事。”
<br/>　　“军师，你年纪也大了，找机会也该成个家，也好有人与你传宗接代。”
<br/>　　莫鸣米酒喝得越多，话也越来越稠。严笛听在耳中，轻叹口气，莫鸣米这一方枭雄，看似风光，其实背后却也有许多不为人知的苦楚。
<br/>　　夜里的琴川关静悄悄，只有更梆的声音偶尔会响起。琴川关矗立此地多年，敢不听莫鸣米话的有，但敢来攻打琴川关的却一个都无。
<br/>　　长此以往，城内的防守自然松懈下来，也就是最近因为程大雷的出现，卫兵才开始变得警惕。
<br/>　　城外，一支人马悄悄靠近，步行，脸上带着黑色面罩。走路时不发出任何声音，宛若夜行的鬼魅。
<br/>　　程大雷身上背剑，手扣在城墙上，冲左右看了看，突然起身。
<br/>　　城高六丈，大概也就是二十米，程大雷身子在城墙上一攀一点，又是三四米的距离，不断接近城头。
<br/>　　下面的人都看着他，这样的高度已经突破大家认知。不知程大雷能否顺利登上城头，如果他真的可以做到，此战便先胜了一半。
<br/>　　程大雷攀上了城墙，再借力，身子已出现在城头上。
<br/>　　正好有两个卫兵在城墙上巡逻，此刻恰在程大雷面前经过。
<br/>　　程大雷的剑割过一人喉咙，又瞬间捅入另外一人张大的嘴巴里，连杀二人，过程中没有发出任何声音。
<br/>　　程大雷轻松口气，将绳子顺了下去。赵子龙领着几个手脚快的攀上城头，继续接人上来。约有一炷香时间，城头大概已有五六十人。
<br/>　　人人身穿黑衣，黑布蒙面，悄悄摸出了兵器。
<br/>　　在程大雷手势的指挥下，众人潜下了城。
<br/>　　城门十几个人打着哈欠。
<br/>　　“这天儿是越来越冷了。”
<br/>　　“再熬一回儿，就有人来替我们了，大家再撑一撑。”
<br/>　　在下一瞬，一队人从黑幕里跳了出来，那人还没反应过来，已被一剑捅穿心脏。
<br/>　　此刻再不用隐藏身形，程大雷大吼一声『杀』，继而又杀了一人。
<br/>　　这些卫兵还不知发生了什么事，就看到一群山鬼般的怪物跃了出来，如狼似虎，杀人如麻。
<br/>　　战斗结束得很快，短短时间，城门下就已倒下十几具尸体。
<br/>　　赵子龙打开了城门。
<br/>　　城外蛤蟆寨的山贼野兽一般冲了进来。
<br/>
