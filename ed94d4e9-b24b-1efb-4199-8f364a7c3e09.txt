　　林峰拿着雪碧应付一群前来给他举杯敬酒的人。
<br/>　　“林峰小兄弟，我火王还是第一次对一个00后如此服气啊！！”
<br/>　　一个大嗓门传来林峰就看到一个不穿上衣的家伙走了过来，这个男人的皮肤居然有点火红色，说话声音很响亮，中气十足。
<br/>　　“自我介绍一下，步爬惹，人送外号，火王！八阶异能者！！”
<br/>　　林峰闻言原来是火系异能者啊，而且似乎还是基地内第一的火系异能者。
<br/>　　步爬惹离开后，又来了个女人。
<br/>　　这个女人长得一般，带着个黑框眼镜，慢条斯理道：“小女子水王水蓝有幸见过林峰小兄弟！话说回来，林峰小兄弟还真是俊俏呢！！”
<br/>　　水王？
<br/>　　看来此人是基地内的第一水系异能者，也是八阶。
<br/>　　火王水王随后，又来了个雷王。
<br/>　　雷王实力也是八阶。
<br/>　　是个七十多岁的老头子。
<br/>　　“老夫雷声见过林峰小兄弟，还真是英雄出少年！！！佩服！佩服的紧啊！！”
<br/>　　火王水王雷王这几个都是B级异能者，乃是基地内的栋梁之才，火王和水王也都加入了军部，成为军部的高手，雷王虽然是天宫的人，但是俗话说得好，伸手不打笑脸人，林峰也算给他们一些薄面，他们三人过后，林峰应付了一些巴结之人，穿过人群来到了陈勾旁边。
<br/>　　还是和陈勾呆着舒服。
<br/>　　“林峰，我就喜欢你一本正经胡说八道，恭喜你，你算是获得了优先交配权了，瞧瞧那群大屁股刚才兴奋的样子。”
<br/>　　陈勾羡慕极了指了指远处站着的几个拿着雪碧的高挑女人。这群女人就好像是末世之前，一群出席在酒宴之上，穿着高档晚礼服和高档珠宝，拿着红酒杯的贵妇。
<br/>　　不过如今虽然没有高档晚礼服也没有高档珠宝，但这群女人依然不简单。
<br/>　　林峰扫过去，的确，这群女人实力都不低，身材特别火辣，看到林峰目光，都是对林峰抛了个媚眼。
<br/>　　“小姐姐们，我也可以的！我是林峰同学还是他兄弟！我叫陈勾！你们要寂寞了随时联系我啊！！”
<br/>　　陈勾嘿嘿笑道。
<br/>　　他这话一出，真的还有那么几个模特走了过来。
<br/>　　陈勾顿时跑了过去。
<br/>　　“林峰，等下见啊！”
<br/>　　林峰看他那猴急的样子也懒得管，食色性也。
<br/>　　本来他林峰也很好色的，也不知道怎么回事，随之实力的变强，他现在只对提高实力有兴趣，可能是这人生丰富了后，一些欲望变得淡然了，林峰现在心中最大的欲望，就是提高视力，以至于对于苏云的美腿都没有以前那么的依恋了。
<br/>　　“我怎么感觉我实力再强一些都要变成无欲无求的和尚了，这可不行，虽然说无欲无求，但这人活着就是要率性而为。”
<br/>　　林峰心中嘀咕着。
<br/>　　这时候一个人朝他走了过来。
<br/>　　“来一根？”
<br/>　　这个人手里拿着一包辣条就好像末世之前一些人见面给对方发烟一样，不过这个人年纪不大，也就二十一岁一看是个大学生，似乎也不抽烟，所以发的是辣条。
<br/>　　“是你？周凯？”
<br/>　　林峰记得他，那个铠甲异能。
<br/>　　实力也足足八阶。
<br/>　　“铠王的辣条，我得接着。”
<br/>　　林峰笑着把对方抵过来的辣条吃了。
<br/>　　“铠王不敢当，都是其余人给我的美誉罢了，更何况是在战神面前，我这个铠王算个什么。”
<br/>　　周凯很低调也很温和，他长得说实话很帅气，不过比起林峰还是差了点的。
<br/>　　“之前看很多人邀请你加入他们的势力，怎么，考虑好了加入哪一个势力了没。”
<br/>　　林峰看着宴席，每个人都吃吃喝喝闲聊着，就好像现在不是末世，是和平时期，但是林峰知道，这是暴风雨来之前的宁静罢了，也算是给人们一次短暂的喘息机会，要不了多久，他们要发动攻击，对巨树之森总攻，这一次，不再是防御战，而是主动出击，到那时肯定要损失很多人。
<br/>　　“考虑好了。”
<br/>　　周凯笑了笑然后看着林峰。
<br/>　　“我听说战神有个自己的据点，我想跟着你干。”
<br/>　　林峰意外的看着他，“跟着我？”
<br/>　　‘嗯，怎么样。不知道战神看不看得上。”周凯笑道。
<br/>　　“这自然欢迎了。”林峰笑道，这可是八阶异能者，很强大的。人送外号铠王，最主要的是人品很好。
<br/>　　“那以后你就是我老大了。”
<br/>　　周凯笑了笑，不过很快笑容就消失了，而是拿起了雪碧喝了一口，似乎就好像喝酒消愁一样。
<br/>　　“实际上，我之前答应过我的同学们，以后什么势力都不加入，只和他们在一起，但是，他们都死了，等我打败那些怪物，再回去找他们的时候，只看到了一地的尸体，我想要保护他们，但到头来，他们依然死了。”
<br/>　　林峰愣了，没想到还有这种事情。
<br/>　　“你也不必自责，你已经尽力了。”
<br/>　　周凯把雪碧喝完。
<br/>　　又拿起了一瓶AD钙奶。
<br/>　　“我也是这样安慰我的，从那个时候开始，我就下定决心，我要把这群怪物全部杀完！见一个杀一个！所以我在等！在等一个足够强大的人带着我一起去杀怪物！我知道，双拳难敌四手！林峰，我希望你不会让我失望！！”
<br/>　　林峰笑了笑，伸出一只手，“跟着我干，我保证你杀的够爽！”
<br/>　　“好！”
<br/>　　周凯点了点头，伸手抓住了林峰的手，两人四目相对，眼中都充满了强大的斗志和不屈的意志。
<br/>　　宴席就这样愉快的结束，而林峰又收获了一员大将，铠王。
<br/>　　至此，林峰手下八阶高手如云。
<br/>　　盾王、锤王、铠王、斧王，外加陈勾那些原班人马。
<br/>　　宴席结束后，陈勾搂着两个模特走了过来，嘴巴里叼着根九五至尊的香烟。
<br/>　　“林峰，你先回去吧，我带小姐姐们去体会下人生。”
<br/>　　林峰看了眼那陈勾的咸猪手都按在人家小姐姐的臀部上，摇了摇头，“这个东西拿着。”
<br/>　　说着林峰丢给了对方一盒东西。
<br/>　　“六味地黄丸？我靠，还是林峰你最懂我！！”
<br/>　　陈勾收入囊中，这可是补肾的好东西。
<br/>　　“走，小姐姐，我们骑马去！！”
<br/>　　说着就和几个小姐姐走了。
<br/>　　“我们回去吧，别管那小子。”
<br/>　　林峰对陈勾自然放心的很。
<br/>　　“林峰！”
<br/>　　这个时候，陆勇和周华还有一个穿着作战服的中年人走了过来。
