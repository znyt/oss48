　　这一道身影逼近的一刻，楚风眠才看出来，这是一名年轻男子，身上的境界，在八劫古帝。
<br/>　　这一名年轻男子，见到楚风眠的一刻，不由分说，率先出手，一掌直指楚风眠的胸口轰击而来。
<br/>　　“找死？”
<br/>　　楚风眠眼神冰冷无比。
<br/>　　以楚风眠的实力，九劫古帝他都足以轰杀，现在这区区一名八劫古帝，居然敢对楚风眠动手。
<br/>　　楚风眠同样是一掌，凌空轰击而出，跟这年轻男子的一掌猛然对拼。
<br/>　　“大日金刚掌？佛道武者，难道是净土星域的武者？”
<br/>　　两股力量凌空碰撞的一刻，楚风眠眼神之中突然流露出几分惊奇。
<br/>　　这一掌之中蕴含的力量，正是佛门之力。
<br/>　　楚风眠本以为对他如此仇视的，直接不由分说出手对付楚风眠的是，肯定是三大势力一方的武者。
<br/>　　可没想到，居然是佛门的人，现在的佛门，都在净土星域之中，楚风眠跟净土星域，可没有任何的仇怨。
<br/>　　不过既然是敢对楚风眠出手，楚风眠也不管是不是净土星域的武者了，猛然之间手心之中灵力狂霸。
<br/>　　“轰！”
<br/>　　狠狠的一掌轰击在了这年轻男子的胸口之上。
<br/>　　只见这年轻男子的身躯，被打飞了出去足足数百米，却是稳住了身形，旋即再度向着楚风眠冲了过来。
<br/>　　“什么？”
<br/>　　楚风眠惊讶无比，刚刚那一掌，虽然不能算是楚风眠的全力，但是一掌轰击而去的一刻，也足以是一位九劫古帝都要吃一个亏。
<br/>　　对于八劫古帝来说，承受刚刚那一掌的力量，只怕是会被直接轰杀，哪怕是八劫古帝之中的妖孽，承受了那一掌的力量，都要重伤。
<br/>　　可现在这年轻男子，看起来居然只是受到些轻伤？甚至还在反击楚风眠。
<br/>　　“轰隆！”
<br/>　　在楚风眠惊讶之际，这年轻男子的身形再度逼近过来，依然是一掌轰击而去。
<br/>　　在年轻男子的一掌之下，突然间涌现出了无数的火焰。
<br/>　　这火焰，呈现出一种纯粹的红色，业火，佛门之中的独特火焰。
<br/>　　据说佛道巅峰的佛祖所打出的业火，威力不亚于纯阳天火。
<br/>　　猛然之间，这火焰手掌，便是向着楚风眠轰击而来。
<br/>　　“找死！”
<br/>　　楚风眠眼神中蕴含杀意，这一次索性也不在留手了，凝聚灵力，位于他的掌心之中。
<br/>　　群龙咆哮，楚风眠直接施展出始祖第一式来，一掌轰击而去，火焰手掌被当即打碎。
<br/>　　旋即楚风眠的身躯一闪，来到了这年轻男子的面前，一掌轰击在了这年轻男子的胸口之上。
<br/>　　一掌轰击，将这年轻男子胸口处，打出了一个血洞来。
<br/>　　只是，楚风眠突然眉头一皱。
<br/>　　在这年轻男子的胸口处，居然是一滴鲜血，都没有流出。
<br/>　　“此人不是活人？”
<br/>　　楚风眠眼神之中惊讶的刹那，那年轻男子再度出手了。
<br/>　　被打破胸口，就算是古帝，也需要恢复重塑，可这年轻男子，居然是毫无影响。
<br/>　　逼近楚风眠的一刻，再度出手对着楚风眠轰击而来。
<br/>　　这施展出的，都是纯粹的佛门武技。
<br/>　　“破！”
<br/>　　楚风眠手指一点，一下子将这年轻男子的身躯打碎。
<br/>　　只见这年轻男子的身躯破碎，掉落在地上，却是无比沉重，楚风眠一摸，居然是一种金属一般的触感。
<br/>　　“这是……傀儡？”
<br/>　　楚风眠看着地上那年轻男子的身躯，突然想到了什么。
<br/>　　这年轻男子，居然不是活人，而是一种傀儡。
<br/>　　这年轻男子，的确是在出手的时候，从来没有跟楚风眠交谈过，看起来应该是没有任何的意识。
<br/>　　不过虽然没有意识，但是这年轻男子的一身武道，却是保留了下来。
<br/>　　这一身武技，精粹无比，远远超越了傀儡的级别。
<br/>　　像是一般的傀儡，虽然是力量极大，肉身强横，但是始终却不是武者的对手。
<br/>　　最大的区别，便是武道的掌控，玄妙的武道，足以一力降十会，所以只有低级武者，会惧怕傀儡。
<br/>　　对生死境之上的武者来说，傀儡就不值一提了。
<br/>　　但是这年轻男子，虽然是一尊傀儡，但是却保留了武道，在加上傀儡的巨力，跟坚固的肉身。
<br/>　　这年轻男子，只怕是比起九劫古帝，还要难缠。
<br/>　　要是让一名九劫古帝，被这年轻男子缠上，只怕都要耗费不少的手段，才能解决。
<br/>　　“佛寺之中，怎么会出现这样的傀儡？以活人炼制傀儡，这不是佛门的做法。”
<br/>　　楚风眠观察着地上破碎的傀儡，不由的自言自语道。
<br/>　　佛门秘术，虽然诡异，但多都是洗脑，控制之术，足以是让武者阪依佛门，为佛门做事。
<br/>　　可是这傀儡，以活人炼制的傀儡，显然并非是佛门的做法。
<br/>　　反倒是像，魔道的做法。
<br/>　　楚风眠仔细观察着，突然间这破碎的傀儡，开始逐渐的散开，上面金属般的硬块，逐渐散去，变成了正常一具尸体的样子。
<br/>　　在这尸体之上，楚风眠看见其中几道魔纹，一下子坚定了楚风眠的猜测。
<br/>　　“这应该是魔门之中的魔人之道。”
<br/>　　魔人之道，据说是古老魔道之中，最为恶毒的秘术之一，以活人为根本，加以许多手段炼制，可以炼化为一种魔人傀儡。
<br/>　　这种魔人傀儡，被抹除了意识，一直听从于主人的命令。
<br/>　　但是却继承了其中武者生前的武道，力量，在加上傀儡赐予的坚固肉身，是一种极为恐怖的存在。
<br/>　　曾经六道魔门统领九域的时候，就秘密抓捕了不少的强者，将其炼制为魔人傀儡。
<br/>　　以至于后来，六道魔门被九域武者，联手推翻毁灭。
<br/>　　只是，要是单单是这魔人傀儡，楚风眠倒是并不惊讶，在许多古老魔道遗迹之中，都能寻找到魔人傀儡的影子。
<br/>　　但是这里，这一片荒原，却是在一处佛寺之内。
<br/>　　在佛寺之内，居然是有魔人傀儡的存在，而且这被炼化的年轻男子，更是一名佛门武者。
<br/>　　“这荒原之中，有着几分古怪。”
<br/>　　楚风眠身形一动，再度是从这荒原之中飞起，不过这一次，楚风眠却是在注意着周围的一举一动。
