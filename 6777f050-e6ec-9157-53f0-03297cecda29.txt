　　“每，每什么？”薛仁贵知道六十万里的概念，但却理解不了那个每秒是什么意思。
<br/>　　“每秒，就是把一个时辰分成七千两百等份，每一份叫就一秒。”白大褂略显无奈的解释道。
<br/>　　老薛揉着下巴在房间里转了几圈，扭头看了看他的亲卫：“你能理解么？”
<br/>　　亲卫把头摇的跟拨浪鼓一样，事实上他早就已经听懵逼了，后来根本就没听那白大褂在说什么，薛仁贵恨其不争的瞪了他一眼，又转头看着白大褂：“那一秒到底是多长时间？”
<br/>　　白大褂也有些无奈，与几个同伴对视一眼，想了想说道：“理论上说，正常人眨一下眼睛是0.2秒，所以将军可以把一秒理解为以最快的速度眨五下眼睛。”
<br/>　　薛仁贵连一秒的概念都理解不了，现在又整出一个0.2秒，这简直就要了他的老命了。
<br/>　　白大褂似乎也看出了薛仁贵的迷茫，深深吸了一口气，用一种极端无奈的语气说道：“将军如果实在理解不上去，那就记住，其实只需要一眨眼的时间，您发出的信息就会被长安那边接收便可。”
<br/>　　一眨眼？薛仁贵把眼睛眨了眨：“你的意思就我刚刚这么几下，我说的话就被送到长安了？”
<br/>　　“对，可以这样理解。”白大褂终于有些不耐了。
<br/>　　薛仁贵这个时候也忍不住了，上去一把薅住白大褂的衣领：“胡说八道！火车已经是我大唐最快的东西了，送信回长安也要半月之久，你现在竟然告诉老子一眨眼就能把消息送回长安，莫非是欺负俺不成！”
<br/>　　什么叫秀才遇到兵？什么叫对牛弹琴？白大褂发现自己今天所犯的最大错误就是不应该搭理这帮兵痞，如果不理他们，最多也就是和这些兵痞互不相干，可现在一搭理他倒道，这货不感激自己不说，反而还要打人。
<br/>　　而薛仁贵也发现这天是真的聊不下去了，科学院那帮书呆子整天就拿一堆新名词来忽悠人，还特么一眨眼消息就送到长安去了，若是让他再说下去，岂不是会告诉自己人可以到月亮上去！
<br/>　　想到这里，薛仁贵气鼓鼓的把白大褂推到一边，没好气的说道：“你也就不是老子的兵，如果是老子的兵，老子非得抽你三十鞭子不可。”
<br/>　　白大褂经过开始的震惊这个时候已经镇定下来，瞪着薛仁贵寸不不让：“薛仁贵将军，我会将你刚刚行为如实向陛下汇报！”
<br/>　　“报就报，是你先看老子好欺负，故意耍俺！”薛仁贵似乎是觉得打一个书呆子不显自己的能耐，故而哼了一声转身便要离开。
<br/>　　而就在这个时候，房间中那台无线电收发机信号再次闪了起来，原本同仇敌忾眼着薛仁贵的科学院研究员瞬间顾不上他，一个两个又把注意力放到了记录上面。
<br/>　　薛仁贵虽然想走，但潜意识中却隐隐有一种期待，希望刚刚那个白大褂没有骗自己，所以在走到门口的时候，他又停了下来，回头看着那些书呆子在那里忙活。
<br/>　　又是近一刻钟的等待，灯不闪了，白大褂拿着一张纸，鼻子不是鼻子脸不是脸的走来过，将那纸往老薛面前一递：“这是长安的回复，薛将军看完把字签了。”
<br/>　　但是薛仁贵却没有理会那白大褂，一把将他手里的纸抢过来，放到眼前一看，只见上面写着：贵部要求已经收到，手套将随下一趟军列发出，望接收。
<br/>　　落款竟然是李承乾三个字。
<br/>　　这下薛仁贵也有些不大确定那白大褂是不是在骗自己了，目光在纸上停留许久，终于下定决定，如果过几天真的收到了那一批手套，那么自己大不了学廉颇负荆请罪，若是下次军列到了没有手套，那么就只能不好意思了，非把丫打的他妈妈都认不出来不可。
<br/>　　想到这里，薛仁贵拿着笔在纸上签了自己的名字，重新交给白大褂，带着亲卫头也不回的离开了试验场。
<br/>　　……
<br/>　　而此时的长安城也陷入一场动荡，无数科学院的研究员兴奋的奔走相告，将无线电试验成功的消息分享给自己的好友。
<br/>　　被紧急招集到一起的大佬们则是面面相觑，一个两个露出不可置信的表情，他们再怎么也想不到，远在数万里之外的大食竟然可以在转瞬间把消息送达长安。
<br/>　　但事实证却在向他们证明着一切都是真的，两方往来的电报被他们传来传去，那上面每一个字他们都认识，可却怎么也无法接受这个现实。
<br/>　　信息传递在古代一直是让人头痛的难题，无数人都在想着如何提高信息传递的速度，就好像当初的八百里加急，再到后来的信鸽，再到后来的火车。
<br/>　　可是现在竟然有一种可以转瞬间把消息传递到任何地方的东西出现，这简直就超过了正常人所能理解的范围。
<br/>　　甚至就连太上皇李世民的定力也有些接受不了这样的消息，拿着那几封往来的电报，看了又看，最后不大确定的看着李承乾问道：“小子，你确定这不是假的？不是下面那些人在骗你？”
<br/>　　李承乾信心十足的说道：“父皇，这是科学，是做不得假的，再说若是有假，手套运到薛仁贵那边，他立刻就会发现其中的猫腻，科学院的人不可能考虑不到这样的后果。”
<br/>　　无线电终于搞成了，有了这东西，以后对大唐各地的控制力无疑会强许多。
<br/>　　铁路可以快速将部队运到各处，无线电又可能快速传递消息，这对那些存有二心的人是一种无形的震慑，想要搞事情就必须先考虑如何应付朝庭的大军。
<br/>　　李世民再次将低头看了看手上的几张纸，深深吸了一口气，脸上写满纠结。
<br/>　　如果他当初出海的时候能有这东西，何必与那些大食人纠缠那么长时间，只要将无线电这东西打开，调动军队什么的还不是跟玩儿一样，而且速度也是敌人完全无法想像的。
<br/>　　只是可惜，这东西研究出来的时间晚了些，可惜，可惜了。
