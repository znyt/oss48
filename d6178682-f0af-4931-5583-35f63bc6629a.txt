　　傍晚时分，慕清霄跟在萧炎身后，来到萧战的房间外，轻轻敲响房门后，屋内就传来一道沉稳而温和的声音。
<br/>　　“进来。”
<br/>　　坐在桌前，萧战望着手中的薄纸，眼中含有微怒之色，因为上面是后辈的资源分配信息，却没有萧炎的份。
<br/>　　“炎儿，这么晚了还不睡，有什么事情？”
<br/>　　将薄纸藏起，萧战顿时抬起头来，望着萧炎幼嫩的脸颊，语气温柔，可当他看到身后的慕清霄后，眼中却有些困惑。
<br/>　　“父亲，孩儿找您有事商议。”
<br/>　　“嗯？”
<br/>　　萧炎看了一眼慕清霄，道：“父亲可还记得，孩儿近几年体内斗之气消失的事情？”
<br/>　　萧战闻言，沉声道：“自然记得，就是因为这件事情......”
<br/>　　慕清霄站在一旁沉默不语，看着父子两人的交谈，萧战眼中的溺爱不加以掩饰，可见萧炎确实碰到个好父亲啊。
<br/>　　“这位是我的师兄，因为我修炼的功法出了问题，所以体内的斗之气也出现了问题，师门派师兄来辅导我修炼一年的时间，这一年他会暂住在家族中。”
<br/>　　经过师徒三人一番计划，最后还是想出了这个天衣无缝的计划。
<br/>　　“师兄？”回过神来，萧战困惑的点了点头。
<br/>　　难怪面前的青年会来萧家，而且在议事大厅的时候，视线一直望着炎儿，原来是他的师兄，但炎儿何时拜他人为师，怎不曾听他提起过？
<br/>　　“只是，让面前的青年辅导炎儿修行，真的没问题吗？”萧战暗自想道。
<br/>　　貌似看出了萧战的猜疑，萧炎道：“父亲，别看师兄年轻，他可是一名斗皇强者！”
<br/>　　话语刚落，卧室中的气氛就显得压抑起来。
<br/>　　萧战的脸色瞬间凝固，暗自吞咽了口唾沫，简直难以置信，面前这名年纪不大的青年，居然是一名斗皇强者，斗皇可是站在加玛帝国巅峰的存在啊。
<br/>　　想到这里，萧战浑身一震，究竟是何等恐怖的存在，才能教出如此妖孽的徒弟，有如此恐怖的势力支持，自己儿子前途无量啊！
<br/>　　“尊驾...”
<br/>　　见萧战有些拘谨，慕清霄摇头一笑，道：“萧炎是我的师弟，萧叔叔叫我清霄就行，暂住在萧家的日子，有劳萧叔叔了。”
<br/>　　“哈哈，既然是炎儿的师兄，那就将这里当作自己家就行了，不必客气......”
<br/>　　......
<br/>　　......
<br/>　　萧家日子一如既往平静，震惊之余，萧战性格也是稳重，并没有将慕清霄的信息公布出来，也没有说明关系，而是随便找了个理由搪塞过去。
<br/>　　谁也不知道，一个小小的萧家，居然隐藏着两尊斗皇强者。
<br/>　　接下来的日子，药尘开始给两人辅导炼药的知识，慕清霄虽是斗皇级别，但未曾接触过炼药，对此自然是一窍不通，一切都得从基础开始学起。
<br/>　　好在慕清霄不仅在修炼上资质逆天，炼药领域亦是如此，往往一点就通，有九霄琉璃焱的他，练药的时候更是得心应手。
<br/>　　萧炎学习炼药术的时候，修炼也不能落下。
<br/>　　而在萧炎修炼的时候，药尘就教慕清霄更高等的炼药术知识，将自己徒儿的资质看在眼中，药尘心中比谁都高兴。
<br/>　　就在这时，房间外传来苍老的声音：“三少爷，族长请你们去大厅！”
<br/>　　药尘点了点头，瞬间钻入戒指中，慕清霄剑眉皱了皱，神色古怪的看了萧炎一眼，道：“这是你们的家务事，我就不参合了，你自己过去吧。”
<br/>　　萧炎被他盯的有些发毛，尴尬的摸了摸幼嫩的脸颊，暗道：“难道这家伙有什么特殊癖好不成？”
<br/>　　“嗯，我这就过去。”
<br/>　　随口应了一声，萧炎就走出房间，对着房外的一名青衫老者微笑道：“走吧，墨管家。”
<br/>　　望着少年稚嫩的脸庞，青衫老者和善的点了点头，转身的霎那，浑浊的老眼中，掠过一抹不易察觉的惋惜，暗道：“唉，以三少爷往日的天赋，恐怕早就该成为一名出色的斗者了吧，真是可惜。”
<br/>　　让他古怪的是，现在三少爷整天躲在房间里，若是跟女子还好说，但跟一个男子躲在房间中，难免让人往歪处想啊。
<br/>　　想到这里，老管家一哆嗦，顿时起了一身鸡皮疙瘩。
<br/>　　萧炎离去后，慕清霄也消失于卧室中，转眼已经来到萧家迎客大厅的屋顶。
<br/>　　此时，屋顶除了他，还隐藏着一名老者，正是暗中保护古薰儿的斗皇，慕清霄向他微微一笑，便不再理会他。
<br/>　　凌影却是汗毛炸立，目光凝重的盯着慕清霄，后者来到屋顶的时候，他才反应过来，可见隐匿身形的方式何等恐怖，比他高明的多啊！
<br/>　　也就是说，慕清霄的目标如果是他，他现在就算不死，恐怕也是重伤，但很快发现，后者目标不是他，紧绷的心弦也松懈下来。
<br/>　　此刻，凌影心中的震惊不亚于超级大地震，小小年纪就是斗皇强者，到底是那个超然势力培养的弟子，为何会出现在这里，到底有什么目的？
<br/>　　慕清霄并不理会凌影，目光放在大厅中，大厅中人数不少，多半是萧族中人，坐在最上面的则是萧战与萧家长老们，另一批人的袍服袖口上印着云彩银剑图案。
<br/>　　这些日子，他可不光是跟药尘学习炼丹，对于加玛帝国的形势也做过不少了解，云彩银剑图案，代表的就是加玛帝国一霸的云岚宗。
<br/>　　看到云岚宗来者，慕清霄就知道事情如他所料，顿时也就失去了兴趣，离去是不禁瞥了一眼坐在萧炎身旁的古薰儿。
<br/>　　思来想去后，慕清霄决定暂时不夺走金帝焚天炎，对付眼前的斗皇简单，夺走异火也简单，但古族可是斗气大陆的超然势力。
<br/>　　其中斗皇强者多如狗，斗宗斗尊满地走，更有他现在遥不可及的斗圣，绝对不是他现在能够招惹的。
<br/>　　慕清霄是个理智的人，自信是好事，但盲目的自信就是愚蠢，若是现在夺走金帝焚天炎，必然会遭到古族的追杀，光是想想就觉得头疼。
<br/>　　微微摇头，慕清霄就转身离去，倒是隐藏在屋顶的凌影若有所思。
