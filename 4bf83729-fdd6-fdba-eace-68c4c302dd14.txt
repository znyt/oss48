　　进一步说，假如新的天选之子上位，而捉鬼联盟又不能为他们所用，那就一定会遭到忌惮，进而被针对……
<br/>　　“你们也是这么想？”叶少阳转头看小九他们。
<br/>　　道风懒得开口回答这种问题，今天他已经说了太多话，实在不符合他人狠话不多的性格，因此决定不再说话了。
<br/>　　小九点头道：“少阳，军师说的对，我们已经没退路了。”
<br/>　　“好吧，我自己……说实话，根本不在乎什么天选之子，但是为了大家，我愿意顶下去。”
<br/>　　小九欢喜地笑起来。
<br/>　　这场谈话终于结束了，大家都说了很多话，都觉得很累，但是有些事情，却是还等着叶少阳去做。
<br/>　　叶少阳默默朝陈悦走过去。
<br/>　　“老大……”
<br/>　　瓜瓜挡在陈悦面前，眼中含着泪水，嗫嚅道：“老大，纵然她不是我娘，我也答应过她，要保护她的……”
<br/>　　林三生喝道：“瓜瓜，只有你一个人重情重义吗，不杀了她，少阳总有一天会因她而死！”
<br/>　　瓜瓜呆住，难过地转过头，抬头望着陈悦，喃喃道：“你会杀了老大吗？”
<br/>　　陈悦没有作声。
<br/>　　瓜瓜以为她是默认了，大颗的泪水从眼角流下来，用力摇着头，“为什么，为什么要让我经历这种事……”
<br/>　　突然抬起头，对叶少阳说道：“老大，我永远都是你小弟，我自然是不可能看着你死，不过我也不能失信于人，毕竟我也叫了她这么久的娘，你要杀她，我不拦你，你动手吧。”
<br/>　　说完转头看着陈悦，轻轻一笑，“娘，我们认识不久，也许你觉得我很幼稚，但我没有娘，在第一次叫你娘的时候，我就把你当成亲娘了，但叶少阳是我老大，我不能看着他有一点不好，哪怕她要你的命，我也没办法，我只有……陪你一起死了！”
<br/>　　说完伸手朝天灵盖拍过去。
<br/>　　叶少阳和陈悦都吓了一跳，出手阻止，但是瓜瓜出手太突然了，两人反应都慢了半拍，突然一道青光打在瓜瓜身上，将他震飞出去。
<br/>　　道风出手了。
<br/>　　叶少阳急忙冲过去，把瓜瓜拉起来，心疼地骂道：“你有病是吧，这么喜欢玩自杀！”
<br/>　　瓜瓜情绪爆发，扑在他怀里哭起来，“我……我不想你死，但又不能眼看着我娘被你杀了，我好矛盾，我不知道怎么做……”
<br/>　　没等叶少阳开口，身后响起陈悦的声音，“叶少阳，他们说的没错，从你的角度，你必须杀了我。”
<br/>　　叶少阳转过头，四目相对，叶少阳道：“你未必就是天选之子。”
<br/>　　“我不知道是不是，也有可能，我真的是。”
<br/>　　“如果有一天，你觉醒了，而我却是沉睡的左手或是转世鬼童，你……会杀了我吗？”
<br/>　　陈悦静静地看着他，突然摇摇头，道：我跟你一样，一个人想当什么，是他自己选的，只有弱者才屈服命运。”
<br/>　　这句话说得好！
<br/>　　叶少阳微微低头，过了一会儿，抬起头，长出了一口气，同时也吐出了三个字：“你走吧。”
<br/>　　陈悦一下怔住，有点不敢相信自己的耳朵，道：“你让我走？”
<br/>　　“走吧。”
<br/>　　“你不怕？”
<br/>　　“有点吧。”叶少阳笑了笑，“但至少目前，你还是我朋友，我没法对你下手。”
<br/>　　陈悦心中感然不已，怔在当场。
<br/>　　“少阳——”林三生声色俱厉，但叶少阳做了一个坚决的举动，强行让他闭上了嘴：他把七星龙泉剑拔出来，猛地一下反插在桌子上，收伏剑柄，大声喝道：“今天你们谁对她动手，就是不把我当兄弟，以后也一样，我要她活着，哪怕她就是天选之子，我也要用正常的手段打败她，你们难道不相信我有这本事！我什么时候让你们失望过！”
<br/>　　林三生等人一下子都怔住了，道风也沉着脸不做声。
<br/>　　“你在为自己增加难度。”林三生嗫嚅了一句，但只是一句抱怨，没再阻拦了。
<br/>　　“少阳，如果我真是天选之子，只要你还是你，我一定不会对你下手，并且一定把这所谓的名头让给你。不管你信不信。”
<br/>　　“我信。”叶少阳听懂了她那句话：只要你还是你。意思是让自己坚守信念，不要因为觉醒而失去了理智。
<br/>　　陈悦笑着点点头，经过他身边时，在耳边低声说道：“如果你刚才要杀我，我不会反抗，因为死在朋友的手中，总比死在敌人手中好，不过，谢谢你，我没看错人。”
<br/>　　叶少阳心有所感，道：“我希望有一天我们不要有兵戎相见的那一天。”
<br/>　　陈悦从他身边走过，朝宫殿大门走去。
<br/>　　小九犹豫了下，打开了封锁宫门的结界。
<br/>　　叶少阳看着陈悦走掉的身影，心中突然想到，现在的她，还是一文不名的一个普通道士，但是下次看到她的时候，她会是什么样子呢？
<br/>　　或许，会是那个君临天下、万人俯首的天选之子？
<br/>　　“我去送送她……”瓜瓜回头看了叶少阳一眼，跟着陈悦钻出了宫门。陈悦也很高兴地拉住了他的手。两人离去。
<br/>　　叶少阳长出了一口气，在椅子上坐下，转头看着大伙，道：“你们想说什么就说吧，不然骂我一顿。”
<br/>　　他当然知道，他们是为了自己好，内心也很感动：他们是想不声不响地杀了陈悦，为自己清除障碍，同时也让自己毫不知情，这样道心就不会有影响。
<br/>　　如果不是自己至亲之人，谁会如此用心良苦，为自己默默去做这些？
<br/>　　“我就知道你重情义，才想帮你把事办了，没想到还是这结果，罢了，当我白操心一场。”
<br/>　　林三生拂袖要走，叶少阳一把拉住他，说道：“军师，我是重情义，如果不是重情义，我怎么会有你们这班兄弟？你想想，假如把陈悦换成你，或者你们中的一个，你觉得我能下手吗？”
<br/>　　林三生听他这么说，软下来，叹道：“就算你不杀她，至少也该困住她，不该让她这么走了，是，我知道你相信她，可她万一真的觉醒，很可能会变成另一个人，别跟我说这不可能，到时候你如何面对？”
