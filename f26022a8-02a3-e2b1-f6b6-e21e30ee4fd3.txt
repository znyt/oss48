        &lt;h3&gt;第三卷第一百七十四章  扑空&lt;/h3&gt;
<br/>
<br/>        凌辰并不知道宁雅去了韩国，也不知道宁雅在这短短的一段时间里就已经开始行动。
<br/>
<br/>        他还一直以为宁雅在陈子昂的公司。
<br/>
<br/>        他和刘诚之飞回山西，迅速处理了公司的事情和家族的事情，又准备踏上了去北京的旅程。
<br/>
<br/>        看了看年后的工作安排，他对刘诚之说：“你确定要去甸城？”
<br/>
<br/>        刘诚之：“其实这两年你和我都明白，甸城的业绩不怎么样，但是厂家经过调查，发现那边的销量很好，这里面一定是有问题的，厂家为什么迟迟不动，肯定也是有原因的。”
<br/>
<br/>        凌辰看着刘诚之那有点稚气未脱年轻的脸，才发现他不但是记忆力好且对事物的分析有着独到的见解，看事情能看到核心处。
<br/>
<br/>        “你既然知道有原因，为什么你还要去冒那个险？”
<br/>
<br/>        “你和董亚子去，肯定会引起何少华他们的怀疑，尤其是何家还向董家提亲了，董家要是答应了何家的提亲，看着你们俩人一起过去会怀疑的，不过去也会怀疑，不如让我这个名不见经传的人过去，反而他们会放松警惕，你们在别的地方动作越大，我就越轻松。”刘诚之并没有回答凌辰的问话，而是和凌辰讨论问题。
<br/>
<br/>        “诚之？”
<br/>
<br/>        “我大概到2月底就可以过去，游游山玩玩水。顺便看看甸城的情况，按照董家的意思，肯定是要一击而中，董家是不会想将女儿嫁给何少华的，如果要嫁，总代的位置轮不上你来坐，人家来个亲上加亲多好？”
<br/>
<br/>        “你这分析来分析去，就是不想让董亚子嫁给何少华啊。”
<br/>
<br/>        “难道你希望董亚子嫁给何少华？”刘诚之反问凌辰。
<br/>
<br/>        “这个说真的，何少华配不上董亚子，容貌就不说了，何少华长得也不差，只是太过阴柔，带着副眼镜，给我的感觉就是阴冷的眼镜蛇。”
<br/>
<br/>        “所以嘛，别让那花朵一样的人让眼镜蛇给毒害了。”刘诚之笑嘻嘻的说：“怜香惜玉这种事情人品好的男人都会做。”
<br/>
<br/>        “还说不喜欢董亚子，都怜香惜玉了。”凌辰对刘诚之嗤之以鼻。“都不敢承认自己喜欢，懦夫！”
<br/>
<br/>        “说真的，我对董亚子没有那么多感情，就这么几天的时光我不可能会有感情的，只是看在她是一个女孩子的份上不多计较，要说是懦夫嘛，这个名号非你莫属。”刘诚之看着凌辰一本正经的说。
<br/>
<br/>        “说得和真的似的，你和董亚子碰出来的火花，就差没有四溅了。”
<br/>
<br/>        “这个不是我们现在讨论的主题，董亚子是喜欢你的，说不定要是宁雅已经有男朋友了，董亚子就投入你的怀抱了。”
<br/>
<br/>        “这个.......我应该不会，因为我不爱董亚子。”
<br/>
<br/>        “犹豫就是有可能，在强大的利益面前一切皆有可能。你和何家必有一战，凌总，做好战斗的准备吧。我明天就回杭州了，我也有好久没有回去了，再不回去老爷子就该唠叨我了。”
<br/>
<br/>        “好，我明天也去北京，看看宁雅的情况。”
<br/>
<br/>        俩个人结束了谈话，凌辰陷入了沉思，刘诚之看着年轻，但是思维缜密，考虑事情很周到，他的方法不是不可以，把刘诚之放到甸城确实不妥，但是他好像没有更好的选择。
<br/>
<br/>        董总明显是不会将董亚子嫁给何少华的，无非就是想借他们的手来拒绝何少华然后打击何少华，何少华已经将他当作眼中钉肉中刺，无论他拿不拿到总代，何少华都不会善罢甘休的。像刘诚之说的他和何少华之间总有一战，那么就让刘诚之帮他一把吧，再说，刘诚之目前还是公司的员工。
<br/>
<br/>        .......
<br/>
<br/>        第二天，俩个人一同到机场，刘诚之对凌辰说：“凌总，加油，祝你马到功成，追妻到手！”
<br/>
<br/>        凌辰：“你也快了，董亚子送我们的时候可是说过：年后要去杭州，你是不去也得去的。”
<br/>
<br/>        刘诚之：“你这老总当得也太憋屈了吧？竟然要下属牺牲色，相？”
<br/>
<br/>        凌辰：“你有色吗？你有相吗？话说，说谎话是会长长鼻子的。”
<br/>
<br/>        刘诚之：“话说，咱俩不相上下，难怪你的鼻子那么长，原来你说过的情话都是谎话！”
<br/>
<br/>        凌辰：“你这胆子是越来越肥了啊。”
<br/>
<br/>        刘诚之：“没办法，小雅姐借我的。”
<br/>
<br/>        凌辰：“我这心里慌得厉害，你说宁雅要真的有男朋友不见我怎么办？”
<br/>
<br/>        刘诚之：“去陈子昂公司要回你的款，打道回府，然后去贵州陪着董亚子过一个别样的新年。”
<br/>
<br/>        凌辰：“我为什么要去贵州？回去陪我老娘不香吗？”
<br/>
<br/>        刘诚之：“哈哈，那我们就都回去陪老娘吧。”
<br/>
<br/>        .......
<br/>
<br/>        两个多小时的时间对凌辰来说就是度日如年：小雅，跨过千山万水，还是你最美。
<br/>
<br/>        小雅，没有你的消息，我是度日如年，就如那一叶浮萍，找不到我的根。
<br/>
<br/>        小雅，你有男朋友了是真的吗？
<br/>
<br/>        小雅，是我的错，是我不该只是暖昧，是我没有胆量追向你。
<br/>
<br/>        小雅，你知道吗？我只是想再努力一下，有了足够的资本向你开口，我以为你会在原地等我，我以为我每天的问候能留住你。
<br/>
<br/>        凌辰在颠簸的飞机上胡思乱想，宁雅在韩国风淡云轻。
<br/>
<br/>        北京的天气雾蒙蒙的一片，走出机场大楼，凌辰不禁打了一个寒颤，室内温暖如春，室外寒风凛冽。
<br/>
<br/>        他首先到达了酒店，放好行李，忐忑的拿起电话。
<br/>
<br/>        电话响了很久，都无人接听，凌辰在房间里走来走去，走来走去，记得上次到北京还是住的这个酒店，那个时候想约宁雅出来，宁雅说她在加班，第二天他就走了，然后过两天他就接到了宁雅那娇媚的电话。
<br/>
<br/>        他的心还是在隐隐作痛，这一次他就是来做个了结，无论结果如何他都要接受。
<br/>
<br/>        中午饭他也没有心思出去吃，在酒店坐卧不宁，拿着两个手机，边工作边等着看看宁雅回不回信息。
<br/>
<br/>        一直到下午，他终于是忍不住，他给陈子昂打了个电话。
<br/>
<br/>        “凌总！”
<br/>
<br/>        “陈总，在北京吗？”
<br/>
<br/>        “在山西呢。”陈子昂到山西出差了。
<br/>
<br/>        “哈哈哈，我今天刚从山西到北京。”
<br/>
<br/>        “我明天早上回去，那明天中午一起吃饭？”
<br/>
<br/>        “宁雅在公司吗？我打电话给她，一直无人接听。”
<br/>
<br/>        “宁雅去韩国了。”
<br/>
<br/>        “什么时候去的？”难怪电话无法接通。
<br/>
<br/>        “应该是今天走的。”
<br/>
<br/>        “她去韩国干什么？”
<br/>
<br/>        “和朋友一起出去考察市场了，可能到年后才能回来。”陈子昂在电话那头客气的解释着。
<br/>
<br/>        “哦，我知道了，谢谢陈总。”凌辰听说宁雅去韩国了自认为宁雅和男朋友一起走的，便连再见也不给陈子昂说，就挂了电话。
<br/>
<br/>        心，疼的无法呼吸！
<br/>
<br/>        凌辰回到了山西，陪老娘过新年不香吗？只是这个新年心已经不知所踪。
