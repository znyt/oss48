　　第101章
<br/>　　“没见过，滚开！”呼尔勒大吼。
<br/>　　石桥少年眨眨眼睛，没有说话，却也没有动怒。
<br/>　　“喂，喂。”程大雷又圈马跑回来，气喘吁吁道：“我就是你要找的那个眉目如画，风姿潇洒的美公子。”
<br/>　　少年打量程大雷一眼，抿嘴一笑，给了程大雷一个鼓励的眼神。
<br/>　　你妹喔！
<br/>　　这种明知道对方是个傻叉，却又努力不拆穿，还表达鼓励的眼神，究竟是想证明你的修养，还是表达我的愚蠢！
<br/>　　少年打量程大雷的同时，程大雷也打量着对方。见少年十五六岁年纪，身材修长，面若敷粉，着一身洗得发白的麻布长袍。眉目带笑，阳光得紧，似乎四方寒雪也在他阳光的笑容下融化。
<br/>　　“该不会你就是那个眉目如画，风姿潇洒的美公子？”程大雷试探着问。
<br/>　　少年笑，笑而不语。
<br/>　　“赵云？”
<br/>　　少年笑，这次摇了摇头。
<br/>　　“那你究竟是谁？”
<br/>　　“我的名字平平无奇，料说出来阁下也未必听说过。”少年谦逊一笑，很平常道出一句话：“吾乃常山赵子龙。”
<br/>　　嘶！
<br/>　　程大雷倒抽一口凉气，果然平平无奇！
<br/>　　有马蹄响，徐神机一行人拍马赶到。几人齐声大吼：“大当家不必惊慌，我们来救你！”
<br/>　　少年眼前一亮，冲那边挥挥手，迈步就要过去。
<br/>　　“小子，去死吧。”
<br/>　　呼尔勒骑在马上，一斧头劈了过来。
<br/>　　“抱歉，你挡路了。”
<br/>　　少年眉毛一挑，紧盯着奔过来的斧芒，他蹂身避过，手往桥上积雪中抓了一把。
<br/>　　扑索索雪粉荡漾，宛若琉璃世界被敲碎洒下乱玉，好一条亮银枪从雪地中抓出来。枪身一甩，宛若雪蟒裹着惊雷。
<br/>　　马下拆战三合，不分胜败。
<br/>　　“子龙，上马！”
<br/>　　程大雷从马上下来，在马屁股上重重一拍。
<br/>　　少年一个垫步，一跃站在马上，脚立在马鞍上，枪挑如龙。
<br/>　　程大雷长长松口气，爆喝：“子龙，杀人！”
<br/>　　少年没搭理他。
<br/>　　刘悲大喊：“子龙，杀人！”
<br/>　　少年星眸爆寒，有这句话在，他宛若变了一个人。枪在掌心，打了个螺旋，一手勒着马缰，一手持枪扎向呼尔勒咽喉要害。
<br/>　　“阁下的首级我收下了。”
<br/>　　咦！
<br/>　　若说程大雷不是那眉目如画，风姿潇洒的美公子，程大雷也没有不服……反正也差不了多少嘛。可若说刘悲是……瞧他苦逼的表情，这得眼睛瞎到什么程度。
<br/>　　嘶！
<br/>　　程大雷倒抽一口凉气，后槽牙有些疼。
<br/>　　二人大战车轮相仿，转了圈厮杀，雪粉四荡，乱琼碎玉一片。
<br/>　　此刻秦蛮、林少羽已经加入战团，和赤虎、野德等人厮杀，在很短时间内，便夺回自己兵器。
<br/>　　长枪横扫，梅花四绽。
<br/>　　秦蛮窝了一肚子火，此刻才施展开来，在十余骑的包围圈中，不落下风。在于林少羽的合力厮杀下，将一个个人砸落马下。
<br/>　　但最精彩的战斗还在少年和呼尔勒之间，呼尔勒渐露败相。他没想到这小小年纪的少年，竟有如此了得的枪法，且骑术绝不在自己之下。
<br/>　　姓名：赵子龙（初出茅庐的顶级武者）
<br/>　　年龄：15
<br/>　　技能：百鸟朝凤枪
<br/>　　隐藏属性：龙胆
<br/>　　啧啧！
<br/>　　程大雷连连赞叹，才十五岁，已经是顶级人才，这要再过几年那还了得。
<br/>　　如此，胜负已分。
<br/>　　赵子龙双腿控马，一手抓着枪柄，一手抓着枪杆，一揽一抛，一枪扎了出去。虽只一枪，却若万千枪同时绽放，万千枪又汇成一枪，电闪雷鸣般扎了过去。
<br/>　　呼尔勒右肩被洞穿，跌落马下。
<br/>　　但他仍旧强悍，换左手握斧头，转了圈乱挥。
<br/>　　程大雷此刻喘匀了气，也恢复些体力。他拔出剑，瞅个空隙，一剑刺呼尔勒小腿。
<br/>　　呼尔勒跌坐在地，眼瞪着程大雷，忍着剧痛挥斧头砸过来。
<br/>　　“狗贼，去死！”
<br/>　　程大雷一脚将他踹倒在雪地中，鲜血浸湿地上积雪，他呼哧呼哧喘气，紧握着手中的斧头。
<br/>　　程大雷洞穿他的手腕，拔出来后小臂上出现个血洞。
<br/>　　“狗贼……”
<br/>　　一剑穿过他的肺叶。
<br/>　　“狗……”看着头顶明晃晃的剑尖，剩下的字吞进口中。
<br/>　　“喂，你相信正义么？”程大雷问。
<br/>　　咦，这是什么鬼！
<br/>　　“你相信正义么？”程大雷再问。
<br/>　　呼尔勒张大嘴巴，呼哧呼哧喘气，血沫从嘴里喷出来。
<br/>　　“难道你在城里这么久，没听到过一个传说么？”程大雷双手握剑，脸上仿佛出现圣洁的光泽。
<br/>　　“善信的人有福了，世间最暴力的恶当有最暴力的善来了解，当正义审判的日子来临时，当死的死，当活的活，所有罪恶绝无藏匿之地”
<br/>　　“所以……”
<br/>　　程大雷略顿了顿：“你相信正义么？”
<br/>　　呼尔勒咬着牙，努力恢复气息，突然抓紧斧头，照着程大雷挥过来。
<br/>　　程大雷一剑切断他右手。
<br/>　　秦蛮和林少羽的战斗已快分出胜负，赵子龙加入战斗后，形势更加明显。一一有人被挑落马下，秦蛮出手极重，他手下过的人不死也得重伤。
<br/>　　见呼尔勒这边形势危险，赤虎等人没命冲锋，却是冲不过秦蛮三人立下的铁墙。
<br/>　　“小王子！”
<br/>　　“小王子！”
<br/>　　赤虎等人绝眦欲裂，没命大喊。
<br/>　　程大雷充耳不闻，剑指着呼尔勒眼睛，呼尔勒的眼中依旧充满愤怒，恨不得生吞程大雷的肉。
<br/>　　“你看这天……”程大雷以剑指天，悠长叹道：“这天多高啊。”
<br/>　　“你看这地……”程大雷以剑指地：“这地上的雪多白啊。”
<br/>　　“你再看这斧头……”程大雷将剑还鞘，把地上的斧头抓起来。
<br/>　　呼尔勒已说不出话来，只能用极微弱的声音说：“斧头，我的斧头……我的风暴战斧……”
<br/>　　程大雷双手握住斧柄，血水顺着斧刃流淌在地上，砸落在雪中。
<br/>　　“这他妈是我的斧头！”
<br/>　　程大雷大声喊出这句台词，斧头狠狠挥下，在呼尔勒死不瞑目间，劈断他的脑袋。
<br/>　　血从腔子里喷出，既落在雪山，也落在程大雷衣上。
<br/>　　天是高的，雪是白的，这斧头是我的。
<br/>　　这些都是真理，不应该被置疑。
<br/>　　
<br/>(本章完)
