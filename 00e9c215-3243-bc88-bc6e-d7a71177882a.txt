　　第243章
<br/>　　程大雷口中说的自然是林少羽，林少羽也被官兵抓了，只是二人不在一个牢房，程大雷也不知道他的情况。
<br/>　　“他今夜也会被放出来，算时间当快到了。”青叶佛知道程大雷说的是谁。
<br/>　　“林冲，秦蛮，你们去接一接，快去快回。”程大雷。
<br/>　　林冲和秦蛮迫不及待去向城门方向，程大雷在原地等待着。
<br/>　　“施主，你只有一个时辰时间。”
<br/>　　一个时辰的不设防，一个时辰后就是满城通缉，然后传告天下，人人见之可杀。
<br/>　　时间一点一滴过去，程大雷心中略有些着急。不知林少羽可否平安归来，说实话，帝国在程大雷心里没有任何信用可言。
<br/>　　这时候秦蛮大步回来，跑得气喘吁吁。
<br/>　　可只有秦蛮一个，林少羽和林冲都不见踪影。
<br/>　　果然出事了么？
<br/>　　“大当家，少羽走了。”
<br/>　　“走了？”程大雷愣了愣。
<br/>　　“从城门出来后，他便和林冲一起走了。”秦蛮道：“临走前他让我转告大当家，说他累了，想找个地方歇歇，把一些事情想明白。”
<br/>　　“累了么，找个山清水秀的地方歇歇也好。”程大雷如此说着，可心里还是有些失落。
<br/>　　但他很快打点起精神，看着身边的阿喜，阿苦，飞天蛤蟆等人大声笑了笑。
<br/>　　“诸位弟兄，我就有走了，江湖路远，咱们改日再会。”
<br/>　　“大哥，你不带我们一起走么？”阿喜立刻道，声音亮得像钟。
<br/>　　“西北太远了，你们更适合生活在这里，阿苦，我在长安打拼一年，钱在那里你都知道，分给弟兄们。西北那边什么情况我还摸不清，等我混好了，兄弟们可以去投奔我。”
<br/>　　众人生于此，长于此，有些已经生儿育女，他们未必舍得下这里。可是，今日分别想起曾经程大雷的恩情，他们心里都涌出不舍。
<br/>　　“大哥，那我们以后去那里找你？”飞天蛤蟆道。
<br/>　　程大雷看着他，忽然笑了笑，道：“日后你们若听到有个以蛤蟆为号的人，那大概就是我。”
<br/>　　众人围着程大雷，依依惜别之情难以言说，程大雷挥挥手，带着秦蛮、徐神机、徐灵儿转身向西北方向。
<br/>　　“走啦。”
<br/>　　……
<br/>　　赤日炎炎，满山的绿树都卷了叶，林间蝉鸣越噪。
<br/>　　程大雷一行四人从远处慢慢走过来，因为天气太热，都走得汗流浃背。
<br/>　　远处的山坳停着一哨人马，约有五六十众，手中都拎着腰刀，大枪，铁锤之类的。
<br/>　　一见程大雷过来，人群嗡地涌了过来，为首是个芝麻眼，大饼脸的男人，还没看见程大雷就大声喊着。
<br/>　　“程大哥，程大哥，我可接住你们了。”
<br/>　　葫芦山四当家李大眼，与程大雷倒也认识，关键这人生得太有特点，长着一双芝麻眼，可外号偏偏叫『大眼』。
<br/>　　“老四呐，你怎么在这儿？”
<br/>　　“还不是等程大哥，我们大当家撒出去十几波人，守在葫芦山各条路上，嘿，要说还是我运气好，把程大哥给接着了。”
<br/>　　“走，上山看看。”
<br/>　　葫芦山的老大名叫范桐，程大雷守着长安城，葫芦山守着长安城出口，两人算是强强联合，互相合作，当然也称得上勾结。比如上次官兵攻打葫芦山，如果不是程大雷通风报信，出谋划策，范桐恐怕已经被官兵砍了脑袋。
<br/>　　而程大雷这次出事，范桐也是出了不少力，程大雷的斧头如今还在葫芦山搁着，包括那个真正的牛三斤，也被转移到葫芦山囚禁。
<br/>　　几人还没上山，就见一个足有三百斤的胖子迎过来，脸上肥肉横生。
<br/>　　“老程，老程，你可不够意思，有多长时间没来看我了。”
<br/>　　“废话，我出得来么，出事了，被关起来了。”
<br/>　　“听说了，听说了，你不是把那姓杨的杀了么，嘿，真有你的。”
<br/>　　山上早已摆好酒宴，葫芦山的几位当家都在，酒席宴间，诸人推杯换盏，满嘴粗言秽语。程大雷从长安走到这里，吃得还真的不好，于是也是狼吞虎咽。
<br/>　　“老程，长安是没法待了，这次打算往哪儿去？”
<br/>　　“去西北看看。”
<br/>　　“嘿，那鸟不拉屎的地方可比不上咱们这里，不如你干脆别走了，就留在我葫芦山。”
<br/>　　“我若留在你这里，朝廷也就盯上你了，我还是离你远点。”
<br/>　　“那好吧，兄弟你多在我这里住几天，我给西北道上传下话去，让他们一路上接着些，别让老程你路上受委屈。”
<br/>　　范桐强留程大雷在葫芦山住了三天。每日大宴小宴，大酒小酒，差点将程大雷喝成醉鸭。
<br/>　　即便这样，范桐仍不让程大雷走，要不是程大雷坚持，看这架势要将程大雷留到明年开春。
<br/>　　葫芦山为四人准备了四匹好马，都是好脚力的走马，冲刺不行，但都有好耐力，正适合走这样的远路。
<br/>　　程大雷拿回自己的斧头，还有四人一路上的干粮，方便花销的散碎银两。
<br/>　　秦蛮有一套黑色锦衣，徐神机有月白书生袍，尤其是对徐神机，范桐十分赞赏。
<br/>　　“听说啦，听说啦，神机妙算徐神机么，大事小事都是您拿主意，这套衣服正适合你。”
<br/>　　徐灵儿有葫芦山夫人们送的首饰、衣服，这里说的夫人们，不是山寨各个当家的夫人，而是范桐一个人的夫人们。
<br/>　　四人全身上下焕然一新，程大雷背着大斧，腰间佩剑，秦蛮一身黑衣，徐神机月白长衫，捂着脸还真有几分仙风道骨，徐灵儿锦缎流苏，腰上的首饰箱叮叮当当。
<br/>　　程大雷在葫芦山得到坐骑和新皮肤后，重新踏上行程，目标西北琴川。
<br/>　　在他们走后两三日后，有三个人从葫芦山被丢了出来。
<br/>　　“他爹，咱们要不咱们还是去南方吧，哪里就算要饭也比云水关吃得好。”
<br/>　　“南方？”满脸大胡子的汉子摇摇头，坚定道：“我是帝国奉的武将，吃朝廷俸禄，就要替朝廷守土，咱们，回云水关。”
<br/>　　“你都三年没拿过俸禄了。”女人委屈得埋怨，却无可奈何抱着孩子跟在男人身后，嘴里不停数落着什么。
<br/>　　“我当初就不该嫁给你，我要不嫁给你，就不会来长安，我不来长安……”
<br/>　　
<br/>(本章完)
