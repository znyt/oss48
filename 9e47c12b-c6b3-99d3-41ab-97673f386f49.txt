
    这天晚上，尽管已经喝懵逼了的王惊蛰，脑袋却异常的清醒，躺在床上瞪着两只眼睛看着天花板，始终都没有睡着。<br/>　　<br/>　　    真应了那句话，其实酒是不醉人的<br/>　　<br/>　　    能让王惊蛰醉的，只有小草姑娘，他忽然想到那日在渭河上初见，仔细回味，才发现他吃了一条渭河鲤鱼后现在却想不起来是什么味道了，他记忆最深的，就是泛舟河上的一个姑娘了。<br/>　　<br/>　　    隔天清晨，天还蒙蒙亮，顶着两只黑眼圈的王惊蛰被王令歌开车送往白云机场，路上，他始终抿嘴低头不语。<br/>　　<br/>　　    王令歌问道“要不我跟你去一趟”<br/>　　<br/>　　    “不用”王惊蛰干脆利索的拒绝了。<br/>　　<br/>　　    “你似乎也没什么朋友了吧这一点跟向叔挺像的，他当年身边就是狐朋有狗有两三个，每遇到事的时候想摇旗呐喊都喊不出来”<br/>　　<br/>　　    “还有一点我也挺像他的，那就是凡事还是靠自己”王惊蛰没要王令歌的好意，尽管他也知道自己张嘴对方可能真就直接把车扔在机场跟他去滕州了，但自己的事真的没有必要麻烦外人，借力借惯了，你以后走路岂不是都得需要人扶着了<br/>　　<br/>　　    半个多小时后，车子抵达白云机场，取完登机牌到安检口，王惊蛰朝着等在外面的王令歌摆了摆手。<br/>　　<br/>　　    “昨晚没喝好，有空再聚”王惊蛰随即转身走过了安检。<br/>　　<br/>　　    王令歌望着他的背影，直到人影消失，才说道“跟我叔，太像了”<br/>　　<br/>　　    早上七点半，航班飞往遥墙机场，十点多落地，王惊蛰轻装简行的从机场出来，直接打了一辆车赶往滕州，中午左右到达后，就没急着往唐梁山方向走了，而是找了个地方落脚。<br/>　　<br/>　　    凉绣给了他三天的时间，如今一天还没过去，他也不怕对方在这三天里再扯什么幺蛾子。<br/>　　<br/>　　    “唐梁山”午后，随意扒拉了两口饭的王惊蛰找了一张滕州地图，将唐梁山给圈了出来，然后搜肠刮肚的回忆着关于墨家的一些点滴。<br/>　　<br/>　　    先说这唐梁山，此山历史倒是挺久的了，不过可能除了滕州本地人外，这唐梁山外人可能连听都没有听说过，哪怕就连一些滕州的青年可能都不太知道，因为这山实在是太没有什么声名了，论底蕴没有啥名胜古迹，论景色也说不上风景秀丽，就是一座占地几百亩，高有三百多米的山头。<br/>　　<br/>　　    越不出名的地方，自然对于隐居就越是有好处了，谁也不曾知晓，历代墨家巨子就是隐居在唐梁山上的草庐中的。<br/>　　<br/>　　    关于墨家王惊蛰了解的底细也不是很多，除了墨家思想外，他大概也就知道墨家最擅长机关术数，奇门八卦了，因为墨家相比儒家来讲，确实低调，声名不显了很多，因为千年来墨家人始终都是出于被打压和排挤的状态，毕竟他们的理论和各大王朝都比较推崇的儒家思想大相径庭。<br/>　　<br/>　　    午后，王惊蛰从住处出来，上了一辆通往唐梁山方向的中巴车，在车上颠簸了两个小时后就到了唐梁山附近的一处镇子上。<br/>　　<br/>　　    王惊蛰到了镇子上，打了一辆三轮蹦蹦，往几十里地外的唐梁山去了。<br/>　　<br/>　　    工欲善其事必先利其器，得能尽量做到知根知底才是。<br/>　　<br/>　　    三轮子行驶在坑洼不停的村道上，颠的王惊蛰屁股都木了，他靠在座椅上拿出烟自己点了一根，给前面开着三轮的老头递了一根。<br/>　　<br/>　　    老头接过烟，咬在嘴上，操着半生不熟的普通话说道“小哥啊，去唐梁山干什么”<br/>　　<br/>　　    “没事，去逛逛，听说那风景不错”王惊蛰吧嗒着烟嘴说道。<br/>　　<br/>　　    老头使劲的抽了口烟，挺吃惊的问道“去那看风景您这真是闲着了，那就是座石头山，上面长着几棵树没啥看的”<br/>　　<br/>　　    “呵呵，随便看看”王惊蛰淡淡的回了一句。<br/>　　<br/>　　    二十多分钟后，三轮蹦子到了唐梁山附近，王惊蛰从车上下来，递给老头二十块钱说道“大爷，您要是不着急走，就在这多等我会，我去逛一圈就回来，回镇子我再给你二十，行不”<br/>　　<br/>　　    “成，成，我等你”老头眉开眼笑的接过二十块钱说道。<br/>　　<br/>　　    王惊蛰说完，就溜溜达达的顺着唐梁山下的山路，走了起来。<br/>　　<br/>　　    这山占地面积不大，山顶暂且不说了，山下四周都是田地围拢而建，基本上也看不见什么人家，这一片是滕州的主要产量地，地广人稀。<br/>　　<br/>　　    王惊蛰围着唐梁山，用两脚开始一步一坑的走了起来，两眼始终都是山下一眼，山上望两眼，走几步停一会。<br/>　　<br/>　　    他是真不信，这唐梁山就是座普通的唐梁山，墨家巨子若是居住在此的话，恐怕整座唐梁山都被笼罩在一片奇门机关之中了。<br/>　　<br/>　　    所以呢，知己知彼百战不殆，没上山前得要好好看看才行。<br/>　　<br/>　　    一个多钟头之后，王惊蛰围着唐梁山逛了小半圈回来了，拧着眉头面色不善，因为走的这四五里地的距离，不管是山上还是山下，他什么都没品出来。<br/>　　<br/>　　    山还是那座，没有什么出奇的地方。<br/>　　<br/>　　    老头还在村路边等着，王惊蛰回来后上了三蹦子，让他回到镇子里。<br/>　　<br/>　　    “小哥，看见啥了么”<br/>　　<br/>　　    王惊蛰摇了摇头，问道“大爷，这山顶您上去过么”<br/>　　<br/>　　    “去过几次啊，小时候吧，跟几个同村的孩子去山顶玩过，那山顶慌得很，什么都没有，只有一间茅草屋，里面住着个看着不太正常有点疯癫的老人，我们几个那时淘气啊，以为碰见了老疯子呢，还拿石头砸过人家，每次打到他身上他都笑呵呵的也不吭声”<br/>　　<br/>　　    王惊蛰又掏出根烟递了过去，接着问道“那大爷，老疯子后来呢”<br/>　　<br/>　　    “后来啊，这老疯子不知道从哪里领了个漂亮的小姑娘回来，大概七八年前的时候，嚯，这姑娘长得柳叶弯眉樱桃嘴，那叫一个漂亮，我这辈子都没见过那么好看的姑娘”<br/>　　<br/>　　    王惊蛰舔了舔嘴唇，说道“这姑娘就在山上和那老疯子一起”<br/>　　<br/>　　    老人摇头说道“那就不知道了，后来几年我也没去过了啊”
