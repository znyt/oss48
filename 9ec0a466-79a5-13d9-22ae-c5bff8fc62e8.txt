　　总算回来了！
<br/>　　叶少阳用双脚在地面上踏了踏，这种充实而且真实的感觉，让他心里说不出的舒服。
<br/>　　“少阳！你们回来了！”
<br/>　　一个熟悉的声音，从前方传来，叶少阳抬头看去，是老郭，从不远处山路上跑过来，眉飞色舞的，十分兴奋。
<br/>　　身后还跟着几个人，一个是秋莹，一个竟然是周静茹。
<br/>　　老郭跑到叶少阳身边，没等站稳，就被周静茹一下挤到了边上，一头扑进叶少阳怀里。
<br/>　　额……
<br/>　　叶少阳有点猝不及防，但内心还是兴奋的，摸着她的脑袋，说道：“你怎么来了？”
<br/>　　“周小姐早就来了，你刚出事不久，她就来了，一直在这边等着你，本来谢警官也在，呆了几天，公事实在太忙，前天刚回去，哦，还有你妹妹和徒弟都来了，在山下酒店里。”
<br/>　　老郭说着，一边跟大伙都打了招呼，横竖看了半天，纳闷道：“花和尚呢？”
<br/>　　“一言难尽，回头再告诉你，反正他没事你放心吧。”
<br/>　　叶少阳跟周静茹抱了一会分开，转头看到秋莹冲自己笑，也冲她笑了笑，“这次多久你了，首功一件。”
<br/>　　“能帮上你们的忙，我就很开心了。”秋莹有点害羞。
<br/>　　“你们怎么样，杀了星月奴没有，是怎么样出来的？”老郭追问。
<br/>　　叶少阳表示回去再细说，转头对小马说道：“你这就走吧，这么多邪物在人间，浩浩荡荡的，实在不合适，你带他们去吧。”
<br/>　　“也是，我也该回去了。”小马看看大伙，眼神有些留恋，这次虚幻世界之行，是他去太阴山之后，第一次回到人间、跟从前的小伙伴们一起行动。
<br/>　　小马吐出一口气，道：“我都有点不想回风之谷了。”
<br/>　　叶少阳听这话心有点酸，道：“你要真不想回去，就留下来呗。”
<br/>　　“得了，反正我想找你们的话，随时都能来，就这样，我也走了。”小马跟大伙打招呼，最后特意跟老郭打趣了一会，然后打开虚空裂缝，那些自愿跟他走的邪物纷纷对叶少阳拜别，跟着小马一起下去了。
<br/>　　等他们这一走，人还剩下不多了，都是自愿跟着小青他们去阴阳司的，自然也不方便在人间久留，叶少阳嘱咐了他们一番，也让他们走了。
<br/>　　“对了，你们到了阴司之后，多跟几个大佬套套近乎，看能不能问出点什么……具体我也不知道什么事，但我想知道，为什么九面猫说要保护我，钟馗却又不让她说，明显是有什么秘密，你们偷偷打听下。”
<br/>　　小青等人答应下来，一一告别，带着最后一拨人去了阴司。
<br/>　　这样一来，人立刻就没几个了，只剩下捉鬼联盟的几个：瓜瓜、包子、秋莹，他们三个是不愿意去阴司的，还有碧清和雪琪。
<br/>　　还有包括腾永清和清波在内的几十个人间法师。
<br/>　　“清波你留下，其余诸位……你们先回吧，我这边还有点私事，就不送你们了。”
<br/>　　这些人间法师纷纷拜别，心中也满满地都是感慨，在古墓事件之前，他们都没见过叶少阳，只是听说过他：叛逆不恭，放浪形骸什么的，这种话没少听，然而，在这段离奇而凶险的经历之后……现在，对于叶少阳，他们内心深处都是深深折服的。
<br/>　　新一代的人间道神，他绝对配得上。
<br/>　　“叶师兄，我师父……”曲波怀里抱着张无生的骨灰盒，身后跟着几个龙虎山的弟子，巴巴地望着叶少阳。
<br/>　　一到烟从叶少阳腰带里飞出来，落在地上，化成人形，正是张无生。
<br/>　　老郭等人看到张无生这个样子，吓得不行，怔怔地站在一边。
<br/>　　“结束啦……”张无生伸了个懒腰，望着叶少阳，“感觉怎么样啊？”
<br/>　　“就这么回事吧。”
<br/>　　叶少阳摸了摸鼻子，“感觉少了点什么。”
<br/>　　“你本来是想杀了星月奴的。”
<br/>　　“她现在可能已经证道了。”
<br/>　　张无生笑了笑，“你怕吗？”
<br/>　　“那倒没有，总有一天我能杀了她。”叶少阳信心满满地说道。只是……本来是预备好了有一场恶战的，结果……居然用这种方式结束了，感觉有点无奈，甚至有点搞笑。
<br/>　　不过，叶少阳安慰自己，至少兄弟们都在，如果真在九星阁那种地方打起来，肯定危险地很——星月奴不光是一个人，那两个盘古僧是他们看见的，还有看不见的影魅等人……
<br/>　　相比杀星月奴来说，至少自己小伙伴们都没事，这已经是最好的结果了。
<br/>　　“我要走了，报仇的事，交给你了。”
<br/>　　“你现在就走？”
<br/>　　叶少阳很吃惊。
<br/>　　曲波也劝张无生跟他一起回龙虎山去。
<br/>　　张无生没理会，问老郭龙阳真人所在，回答在山下酒店里，之前在山上守了很久，实在困得不行，下山睡觉去了。
<br/>　　“那算了，我对人间实在没什么留恋的了，一分钟也不想多呆，我去阴司报道了，后面的事情，交给你们了。”叶少阳知道他去了就是投胎，下次再见到，他就已经不是龙虎山的张真人了，心中有些不舍，刚要说点什么，张无生率先说道：“少阳，关于证道，你知道多少？”
<br/>　　叶少阳摇摇头，实话说自己知道得不多。
<br/>　　“证道之后，跳出三界之外，不在五行之中，对于我们这些人类来说，差不多是无敌的……但是第一，用不同法门证道之后，实力也是不一样的，有高下之分，这一点，道佛二门都是一样，第二，证道之后，对于他们那个层面来说，也只是一个新的开始，还有很多路要走，这个我也不太了解，不说了，第三，证道之后的生灵，是杀不死的。”
<br/>　　叶少阳蓦地一惊，自己居然忘了这个！
<br/>　　“难道星月奴也杀不死了？”
<br/>　　“世上没有绝对，别的我不知道，就人类来说，想杀死证道强者，只有一种方法，就是用强大的法器——光是一把龙泉剑估计都不够，差不多……九鼎应该可以将其炼化，或者诛仙剑阵，总之一个人、一件法器，肯定是不行的。”
<br/>(本章完)
