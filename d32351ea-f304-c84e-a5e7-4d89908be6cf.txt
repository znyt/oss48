　　“我明白。”
<br/>　　慕容飞挂断电话后，返回了书房。
<br/>　　佟千雪扭头看着慕容飞，问道：“又要走了吗？”
<br/>　　“暂时还不确定，但是从现在开始进入待命状态，随时准备归队。”慕容飞弯腰在佟千雪额头上轻轻吻了一下，带着愧疚说道：“对不起，本来还想着这次能多陪你几天，可是没想到又要离开了。”
<br/>　　“不用跟我说对不起，我能明白。”佟千雪轻轻摇头，说道：“你不仅仅是属于我一个人的，你还是属于部队，属于这个国家的，你有你的使命。虽然不能对身边的亲朋好友说明你的工作，但是每当有人问起我的时候，我都会觉得非常骄傲，因为我的男人是个真正的英雄。”
<br/>　　没过多久，罗昊的电话又打了过来，他的语气比起刚才严肃了很多，说道：“无常立刻归队。”
<br/>　　“明白！”
<br/>　　慕容飞站直身体，对罗昊问道：“真的跟他们有关系吗？”
<br/>　　“具体背后是不是他们在搞鬼还不清楚，但是子午教官有发现。”罗昊对慕容飞说道：“如果千雪电脑里没有什么有价值的东西，就把电脑系统重装一下吧，除了向对方支付比特币外，这是目前唯一的办法，另外如果不是必要就断开网络连接，根据小熙的说法，这种病毒可以根据网络自主传播，也就是说，只要你连接着网络，就算什么都不做都有可能感染到这种病毒。”
<br/>　　“现在就要走吗？”佟千雪看着慕容飞，眼眶微红。
<br/>　　慕容飞抱住佟千雪，又是在她额头上轻吻了一下，道了一声“对不起”。
<br/>　　“在你离开之前，至少帮我把电脑系统给重装好，我可不会装系统。”佟千雪对慕容飞说道：“另外帮我把面条煮好，最好多煮一点，这样你不在，我自己也能弄着吃。”
<br/>　　“别傻了。”慕容飞在佟千雪鼻子上刮了一下，说道：“面条放的时间长了就糊了，那就跟面疙瘩一样不好吃了。不过，我可以帮你把面汤做好，这样我放冰箱里，你想吃的时候只要自己煮点面条，再把汤加热一下，把面条放进去就能吃了。”
<br/>　　“好啊。”
<br/>　　佟千雪眼圈微红，声音有些沙哑对慕容飞说道：“我帮你订机票。”
<br/>　　慕容飞松开佟千雪返回了厨房，帮她煮面条并且做汤料，把做好的汤料装进保鲜盒，端着一碗热气腾腾的面条放在放在餐桌上，招呼佟千雪来吃面条。
<br/>　　佟千雪从书房走出来的时候，慕容飞发现她眼睛红红的，慕容飞知道，她刚才一定是躲在书房里哭过了，但是慕容飞并没有拆穿她，走进书房帮佟千雪重装电脑。
<br/>　　餐桌上那碗面条跟那天晚上自己吃到那碗一模一样，不管是香味还是面条的配菜、外观都是一般无二，可是佟千雪却偏偏没有任何的食欲。
<br/>　　强迫着自己拿起筷子，吃了几口面条后，佟千雪放下筷子趴在桌子上低声抽泣起来。佟千雪哭泣的声音虽然很轻，但是并没有瞒过慕容飞的耳朵。
<br/>　　慕容飞紧握着拳头，强迫着自己没有走出书房。作为龙隐部队的队员，他有着属于他的骄傲，他也有着属于他的无奈。
<br/>　　他们是共和国的第一道防线，他们需要将对国家安全造成威胁的隐患扼杀在国境线之外；他们也是共和国的最后一道防线，如果有一天侵略者的铁蹄踏破国门的时候，他们需要孤军深入，怒斩敌酋首级。
<br/>　　等到书房外的抽泣声小了，慕容飞才起身走出书房，对坐在餐桌边的佟千雪说道：“电脑帮你重装好了，昊子说，如果没有特别需要的话，最近这段时间最好别上网。”
<br/>　　佟千雪点了点头，放下筷子对慕容飞说道：“面很好吃，只是就吃不到这么新鲜的面条了，我倒是无所谓，就是可怜了肚子里那个小的。”
<br/>　　“别把责任推在你肚子里那个身上。”慕容飞说道：“他（她）现在最多也就是个胚胎，他（她）能知道好吃不好吃？”
<br/>　　“我说他（她）能知道，他（她）就能知道。”佟千雪抬起下巴，说道。
<br/>　　“行行行，你是他（她）妈，一切你说了算！”慕容飞端起餐桌上的面碗，折身走向厨房去洗碗。
<br/>　　“我去帮你把东西整理一下。”
<br/>　　佟千雪说着起身离开了餐桌，走进卧室帮慕容飞整理背包。慕容飞带回来的东西就没有多少，但是他并没有阻止佟千雪，任由她去帮自己去收拾背包，或许是想让自己的东西上带上一些属于她的气息吧。
<br/>　　……
<br/>　　台湾，高雄市。
<br/>　　刘经理和chen现在很不安，非常不安，如热锅上的蚂蚁一样焦躁。
<br/>　　最开始，当他们把从SHADOW手中以五百万美金高价买来的病毒投入到使用中的时候，他们的确是从一些公司手里敲诈到了一些钱，但是这些钱跟他们所支付出去的五百万美金还相差甚远，可是就当他们把目标投向更大的目标时，他们突然发现这个病毒不受他们控制，发了疯一样在网上传播，就连他们自己公司的电脑都是感染到这种病毒！
<br/>　　尤其是欧洲几个国家的感染情况特别严重，这自然就引起了各国政府的重视，并且成立了相关的调查组对这次勒索病毒的大规模爆发事件进行调查。
<br/>　　刘经理在办公室踱着步，最后快步走到办公桌后面，拉开抽屉把里面的护照拿了出来，塞进自己的公文包内，然后拿起车钥匙准备离开。刘经理知道这次事情搞大了，他必须要离开了，再不走，他就是想走也走不了了。
<br/>　　可是还没等刘经理走出办公室，办公室的门就被人从外面推开，公司董事会成员还有好几个警察走进办公室。
<br/>　　一名警察走上前对着刘经理出示了一下自己的证件，还有搜查单和逮捕令，说道：“刘在铭先生，现在我们怀疑你跟这次全球各地大规模爆发的勒索病毒事件有关，请你跟我们回去协助调查，另外我们还需要对你的办公室进行搜查取证！”
