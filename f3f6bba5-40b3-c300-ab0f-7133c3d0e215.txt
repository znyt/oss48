　　入夜的金光险地，再次恢复了‘热闹’。
<br/>　　只是，这‘热闹’之中，带着的，并非欢呼。
<br/>　　而是激烈打斗声、痛苦哀嚎，以及凄厉怒吼。
<br/>　　热闹之下，并无半分欢愉气息，有的，仅仅是飘荡在险地之内空气中的浓浓刺鼻血腥味。
<br/>　　萧逸整整花了大半晚的时间，一路战斗，一路前行，最终来到了险地深处。
<br/>　　此时，他停下了脚步。
<br/>　　因为，面前两头庞然大物，已拦住了去路。
<br/>　　不，准确来说，是封锁了周遭所有方向。
<br/>　　那是一头巨大的蟒类妖兽，以及一头浑身黝黑的蜥蜴类妖兽。
<br/>　　“夜影流光蟒，荆棘圣蜥。”萧逸双眼一眯，脸色微凝。
<br/>　　据他所知，这两种妖兽，皆是圣皇境五重妖兽。
<br/>　　论实力，绝对远在疾风烈鸟之上。
<br/>　　夜影流光蟒，浑身漆黑，表皮光滑。
<br/>　　巨大的身躯下，却丝毫没有笨拙感，反倒速度快如流光。
<br/>　　等闲的武道皇者遇之，也不过是一道流光之下，便葬身蟒腹的下场。
<br/>　　荆棘圣蜥，浑身坚硬如石，宛若黑岩。
<br/>　　背上荆棘密布，如若铁刺，却锋锐如剑。
<br/>　　“有些意思。”萧逸嘴角微咧。
<br/>　　手中冷焱剑凭空而现，随后等待着。
<br/>　　两只妖兽，毫无疑问，张开了嘴巴，大口吞食着金光。
<br/>　　半晌，一缕缕金光被它们吸食。
<br/>　　它们身上气势，亦陡然飙升。
<br/>　　“达到圣皇境九重初阶了。”萧逸眉头一皱。
<br/>　　在他的感知中，这两只妖兽的实力，已然踏入圣皇境九重的水准。
<br/>　　他记得，昨日遇到的疾风烈鸟，乃是圣皇境四重妖兽。
<br/>　　待得吸食金光后，实力达到圣皇境八重巅峰，接近圣皇境九重，连跳四重。
<br/>　　可现今，这两头妖兽，夜影流光蟒与荆棘圣蜥，本就是实打实的圣皇境五重妖兽。
<br/>　　按理说，它们吸食金光后，理应得到圣皇境九重以上实力，而非现今的堪堪达到圣皇境九重水准。
<br/>　　萧逸眉头微皱。
<br/>　　对比之前的金光幽狼与疾风烈鸟。
<br/>　　萧逸得出的结论是，越是强大的妖兽，能吸食的金光就越多，增长的实力也越多。
<br/>　　现在看来，应该也没错。
<br/>　　夜影流光蟒与荆棘圣蜥，明显吸食了比昨夜遇到的疾风烈鸟更多的金光。
<br/>　　可看起来，提升的实力重数，却差不多。
<br/>　　如果没有猜错的话，应该是圣皇境九重中的实力差距极大。
<br/>　　故这些金光的效果看起来变小了。
<br/>　　据他所知，圣皇境九重，乃是武道皇者中一个极大的坎。
<br/>　　当然，具体如何，萧逸暂时不得知。
<br/>　　毕竟他距离那个境界还极远。
<br/>　　总之，现今这两头妖兽，夜影流光蟒与荆棘圣蜥的实力，便是堪堪达到圣皇境九重的水准。
<br/>　　当然，换一个角度来看的话。
<br/>　　圣王境中，每差一重，尚且是秒杀的差距。
<br/>　　可想而知，圣皇境中，每差一重，差距有多大。
<br/>　　这些金光力量，能让其突破这些差距，已经效果足够可怕的了。
<br/>　　嗖…
<br/>　　这时，一道流光，朝萧逸激跃而来。
<br/>　　流光，极其庞大。
<br/>　　萧逸反应极快，一剑劈出。
<br/>　　他既已身入金光险地深处，自是无时无刻处于最好的战斗状态。
<br/>　　八万星光，早已加身。
<br/>　　这一剑，乃是他全力劈出。
<br/>　　然而，当流光冲撞而来，与冷焱剑交锋的一瞬间。
<br/>　　嘭…一声爆响。
<br/>　　萧逸只觉一股惊人巨力撞到了自己身上。
<br/>　　冷焱剑剑身毫发无损。
<br/>　　但萧逸的双手，却顷刻一麻。
<br/>　　还未反应过来，已然身影一退，被撞飞数十米。
<br/>　　甚至于，手中之剑，竟在这巨大的冲击力下，脱手而出。
<br/>　　“好大的力气。”萧逸脸色一惊。
<br/>　　待得站稳身子后，看了眼双手，已然双掌通红。
<br/>　　嗖…
<br/>　　巨大的流光，再次而来。
<br/>　　“回来。”萧逸低喝一声，手掌虚摄。
<br/>　　被撞飞的冷焱剑，瞬间飞跃回手中。
<br/>　　“喝。”萧逸低喝一声，一剑劈出。
<br/>　　一条星光长河，霎时碾压而出。
<br/>　　巨大的流光袭来，重重地撞到了星光长河之上。
<br/>　　霎那间，星光长河震颤不已，星光溃散不停。
<br/>　　仅仅是数秒之后，整条星光长河，已然消散一空。
<br/>　　夜影流光蟒冰冷的眸子，直视萧逸，仿佛在看一只再渺小不过的蝼蚁。
<br/>　　“好猖狂的孽畜。”萧逸双眼一眯。
<br/>　　嗖…流光再次跃来。
<br/>　　萧逸这一次并未硬扛。
<br/>　　流光袭来，萧逸一剑劈出。
<br/>　　待得剑身与流光碰撞的瞬间，萧逸身体一侧，随后借力一躲，轻松躲开了流光的冲撞。
<br/>　　刚才的一躲，让得萧逸没有被撞飞。
<br/>　　却也近距离地看到了流光之下的情况。
<br/>　　夜影流光蟒，竟是化作了流光，以口而噬。
<br/>　　若他刚才没有躲避开来的话，恐怕夜影流光蟒那巨大的血盆大口，已然将他吞下。
<br/>　　嘶嘶…
<br/>　　夜影流光蟒，嘶嘶地吐着猩红信子。
<br/>　　嗖…下一秒，再度化作流光袭来。
<br/>　　萧逸嘴角咧过一道冷笑。
<br/>　　身为猎妖师，他还没怕过什么妖兽。
<br/>　　即便此刻这头妖兽的实力高于他，他也并不放在眼中。
<br/>　　流光袭来，萧逸如法炮制，一剑挡之，随后借力而躲。
<br/>　　但这一次，堪堪躲避开来后，剑锋陡然一转，化挡为劈。
<br/>　　冷焱剑锋利的寒芒，重重地劈在了夜影流光蟒的蟒身之上。
<br/>　　萧逸冷笑一声。
<br/>　　这一剑，以冷焱剑的锋利，足以将这头孽畜重伤。
<br/>　　然而，剑光重重落下。
<br/>　　嗤…的一声。
<br/>　　锋利的剑身，竟是在触碰到蟒身时，霎时打滑，剑力被卸去大半。
<br/>　　夜影流光蟒除了发出一声痛呼嘶吼外，毫发无损。
<br/>　　“该死。”萧逸脸色一惊。
<br/>　　夜影流光蟒的蟒身，竟是光滑如丝。
<br/>　　锋利的剑身，竟是打滑而过。
<br/>　　嗖…夜影流光蟒，再次化作流光。
<br/>　　此时的它，显然在疼痛中暴怒，速度抖增。
<br/>　　“不好。”萧逸脸色一变。
<br/>　　星光长河，顷刻凝聚；体内元力，亦瞬间爆发。
<br/>　　流光来到身前之际，萧逸一剑劈出。
<br/>　　一剑出，十数剑齐至。
<br/>　　十数道星光长河，连绵而至，瞬间挡下了流光，且将其巨大的身影轰退百米。
<br/>　　萧逸微微松了口气。
<br/>　　金光险地深处的妖兽，果真危险异常。
<br/>　　然而，他的这口气还未彻底松下。
<br/>　　一头庞然大物，以极快的速度朝他快速爬来。
<br/>　　“荆棘圣蜥？”萧逸眼眸一冷，反手一剑劈出。
<br/>　　锵…
<br/>　　锋锐的冷焱剑，劈在荆棘圣蜥之上，竟是发出一声嘶鸣。
<br/>　　萧逸全力一剑，竟连它背部荆棘分毫都损伤不到。
<br/>　　“好坚硬的荆棘。”萧逸吃了一惊。
