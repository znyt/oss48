夏夕绾跑出去将查房的医生给打发了，医生看她小脸通红的还很奇怪，以为她生病了，夏夕绾扯了一个谎说自己大概太累了，这才蒙混过关。
<br/>
<br/>        她站在窗户那里吹了好一会儿的冷风，心里将陆寒霆那个色鬼又骂上了好几遍，等脸上的热度消散了她才准备回病房。
<br/>
<br/>        这时前方走来了一道熟悉的身影，是陆茵茵。
<br/>
<br/>        陆茵茵来了。
<br/>
<br/>        前两天在圣得堡酒店闹得不欢而散，虽然她不知道陆寒霆和陆茵茵这个姑姑聊了些什么，但肯定是不愉快的。
<br/>
<br/>        夏夕绾停下脚步，“姑姑，你怎么来了，你是来看陆先生的么？”
<br/>
<br/>        陆茵茵看着夏夕绾，“我听说寒霆出车祸了，所以来看一看，夕绾，我正好找你有事。”
<br/>
<br/>        夏夕绾看了一眼前方紧闭的病房门，“姑姑，关于陆先生的过去我并没有兴趣知道，所以你以后不必再提。”
<br/>
<br/>        “不，我要跟你说的是寒霆车祸的事情。”
<br/>
<br/>        “车祸？”
<br/>
<br/>        陆茵茵走过来，将手上的一份资料递给她，“夕绾，你自己看看吧，车祸不是偶然发生的，那晚寒霆并不在公司里，而是开着车跟了你们一路。”
<br/>
<br/>        夏夕绾接过了资料，资料上都是道路监控拍摄到的画面，陆寒霆那辆已经报废的劳斯莱斯幻影豪车出现在了枢密研究院的大门口，然后跟了她一路。
<br/>
<br/>        夏夕绾羽捷轻颤，他说他在公司的。
<br/>
<br/>        他为什么不告诉她？
<br/>
<br/>        “夕绾，寒霆的病想必你也知道些，现在很明显了，看到你和子羡在一起，寒霆又发病了，他将油门踩到了底，根本控制不住自己的情绪，在转弯的时候路上了疾驰而来的大卡车，所以出了车祸。”
<br/>
<br/>        夏夕绾抬头，澄亮的翦瞳落在了陆茵茵的脸上，“姑姑，就算是这样，你想跟我说些什么呢？”
<br/>
<br/>        陆茵茵一滞，“夕绾，难道你不怕他吗？他随时会发病！”
<br/>
<br/>        夏夕绾摇头，“我不怕他，相反看了这些我只会觉得内疚，心疼，陆先生撒谎说自己在公司里却偷偷跟了我一路，因为我而出了车祸，我以后会更加警觉小心些，与陆子羡还有其他男人都保持距离，晚上准时回家，守在他的身边。”
<br/>
<br/>        “姑姑，承蒙你的喜爱，虽然我并不知道你究竟想要干些什么，但是你好像对陆先生一直保持恶意，那我只能奉劝姑姑一句，以后离我的陆先生远一点，你弃之如敝履的人恰巧是我的心上人，不要试图去伤害他！”
<br/>
<br/>        说完夏夕绾将那份资料丢进了垃圾桶里，直接抬脚离开。
<br/>
<br/>        陆茵茵僵在了原地，她完全没有料到夏夕绾会是这个反应，她一点都不怕陆寒霆，她说陆寒霆是她的心上人，还奉劝告诫她以后离陆寒霆远一点，不要试图去伤害他！
<br/>
<br/>        陆茵茵拽紧了拳，她看着夏夕绾的背影道，“夕绾，没用的，你的爱拯救不了他的！”
<br/>
<br/>        前面的夏夕绾缓缓停下了脚步。
