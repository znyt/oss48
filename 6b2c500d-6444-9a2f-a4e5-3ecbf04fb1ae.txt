　　混乱虚空，位于洪荒东北部。
<br/>　　混乱虚空，因虚空混乱而得名，那里，是一片空间的交织之地，是空间混乱之所。空间破碎，裂缝交织，空间之门纵横，风暴不断，空间层不清，一不小心，就会一辈子陷入到无尽空间之中，找不到出路。
<br/>　　一般没有人愿意会来混乱虚空，因为这里除了混乱的空间之外，就再无其他的东西。此处除了精修空间法则的修士之外，就没有其他人了。
<br/>　　这天，混乱虚空来了一个客人，他，就是元始。
<br/>　　元始看着眼前这片虚空，他能够感受到，此处空间之中的混乱，其余法则尽皆被压制，也就空间法则与混乱法则浓郁，一般人进去，实力至少要被压制三成。
<br/>　　这让元始疑惑的是，如此一个鬼地方，龙族是怎么找到的？
<br/>　　元始不知道的是，龙族有一虚空烛龙就精修空间法则，一次意外，这烛龙在这混乱虚空中找到一个小世界，其内充满黑暗死亡的气息，后来此处就被作为龙祖的坟墓。
<br/>　　不过，元始也没有兴趣知道龙族是怎么发现的，他现在要做的是进入其中，找到龙墓。
<br/>　　一步踏进混乱虚空，出现在他眼前的是一片荒漠。元始的神识一扫，便覆盖了整片荒漠，只是到了荒漠边缘却是一片无穷无尽的空洞，无法触摸到尽头。
<br/>　　元始迈开脚步，几步之后，冲出荒漠，接触到的仍然是无边无际的空间，没有颜色，没有形态。
<br/>　　他就好像是在三维或者四维空间中转换一般，只能感受到空间时有时无，却无法真正的掌控这种空间的时有时无之转换。
<br/>　　这里没有方向，没有位置，也没有任何生命气息。
<br/>　　元始不断地穿梭在这无尽的虚空之中，经历了数天，元始之前站立的虚空早已不见，在他面前的却是另一篇空洞的灰色。地面、天空、甚至连呼吸之中都感受到一种灰色的感觉。
<br/>　　元始知道，这里是无数空间转换之地，他刚刚经历的荒漠只是一个空间，现在经历的是这种灰色空洞又是另一片空间。
<br/>　　在这个地方，只要你不断走动，就可能从一片空间到另一片空间，也可能永远陷入一片空间之中，迷失方向。
<br/>　　元始回忆敖玉给的那个龙墓位置，他发现自己尴尬了。位置是有了，但此处没有任何方向，他到底要向哪个方向走？
<br/>　　元始不知道的是，龙族每次来龙墓，都会带着烛龙特有的印记，烛龙在龙墓之中留有手段，那印记会在冥冥之中给出一个指引方向。那样，龙族之人就能找到准确位置了。
<br/>　　无奈，元始只好开始在这无尽虚空之中不断转换空间，不断地穿梭。
<br/>　　时间就在这种无趣的空间转换之中流逝，元始从一开始的无奈到后来的无喜无悲。渐渐地，他沉浸在这其中，在这不断转换空间之中，元始察觉到，每一次从一个空间转换到另一个空间，他都会感受到一种不一样的律动，这种律动最是美妙，让他感悟颇深。
<br/>　　后来，渐渐地，元始就在这种漫无目的的空间转换之中不断穿梭，不断地感悟，甚至忘记了时间，忘记了方向，忘记了空间，只是这样毫无意识的不停走动。
<br/>　　同时，因为空间混乱的风暴撕裂，元始的真身在不断地被损伤，之后又不断地修复，就这样，他的真身强度在不断地提高。
<br/>　　在这些转换之中，元始经历过各种各样的空间。沙漠空间，沼泽空间，森林空间，海洋空间，山脉空间，……
<br/>　　毫无任何东西的灰色空间、空洞世界、风刃空间、雷电空间、破碎空间……
<br/>　　他经历过空间坍塌、空间错位、空间裂缝，空间风暴……
<br/>　　在这里，他更是感受到空间折叠，空间扭曲……
<br/>　　元始的脑海之中，不断地回想起空间和时间的问题，空间和时间是宇宙的本质属性，时间不可逆转。当然，只要实力达到一定程度，时间一样可以逆转，只是这种逆转实现对的，是在一定范围内的。
<br/>　　时间与空间本身就存在着无法分割的关系。
<br/>　　时间就在元始感悟之中不断消失，不知过了多久，也许是一天，也许是一年，也许是万年。
<br/>　　“轰隆！”
<br/>　　一声爆炸声响起，元始被惊醒过来，他睁开双眼，入眼所见，却是虚空之中，一颗巨大的陨石撞击在他身上，发生爆炸。
<br/>　　“嗯？”元始皱起眉头。
<br/>　　不是因为陨石撞击在他身上，以他的肉体真身，不说这些年在混乱虚空之中的炼体了，就是在这之前，这点陨石撞击他也还不放在眼里。他皱眉是因为他突然间将空间问题想复杂了。
<br/>　　有必要想得如此复杂吗？他只要抓住一点，无论任何空间，都是上下四方。
<br/>　　两点之间的一切距离，都是空间。他只要抓住这两点，强行压缩这两点之间的距离，就可以跨越空间，到达另外一点。哪怕这两点之间相距无数个星球，无数界面，只要他的神识可以触摸到，他就可以实现这种想法。在瞬间将这种空间压缩成虚无，化成虚无。
<br/>　　这种将空间压缩成虚无的手段，对于别人来说，可能无比困难，但对于元始来说，他的无极印既可以解决。
<br/>　　之后，元始再次沉浸在空间的感悟之中，不断转换空间。
<br/>　　一年，十年，百年，千年……
<br/>　　整整五千年，也许从未有一个人能够像元始一样，在这种感悟之中不断转换行走五千年。他甚至陷入了一种魔障，没有感悟到空间法则，就誓不罢休。
<br/>　　随着感悟越深，念头就越是通达。
<br/>　　“何为空间，空间者，宇也。何为宇？上下四方谓之宇。上下四方？上下四方？上下？四方？不错，就是上下四方！哈哈哈！”
<br/>　　元始像是自言自语，轻声呢喃，像是自问自答，越到后面，眼神越是明亮，声音越是洪亮，到最后，竟然大声笑了出来。
<br/>　　元始大手向着虚空一抓，顿时一片巨大的虚空被他抓在手中，慢慢地，大手恢复寻常，伴随着大手的是，那片巨大空间也渐渐变小。
<br/>　　元始知道，这空间并没有变小，虽然它外表看上去是变小了，但实际上它内部仍是一片巨大的空间，它丝毫没变。
<br/>　　“上下四方，这，就是空间！”元始低头看着手中的空间，低语道。
<br/>　　“轰隆！”
<br/>　　虚空传来一阵爆响，抬头看去，原来是那片被他抓走的空间瞬间变得无光无色，漆黑一片，成了虚无。周边四处，空间坍塌，裂缝纵横，空间风暴迭起。爆炸声响不断。
<br/>　　“唔？被玩坏了，看来此处是不能待了，需换个地方。”元始看着这片虚空，低声道。
<br/>　　“咦？那是——”
<br/>　　穿透虚空，元始看到不远处有一个黑点，在虚空裂缝之中若隐若现。这个地方，元始有些熟悉。
<br/>　　元始仔细一想，原来是目的地到了。
