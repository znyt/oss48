        这是她昨天晚上想了一晚，得出来的结论，不管是她自以为是也好，还是她的胡乱揣测也好，终归她是猜得八九不离十。
<br/>
<br/>        “筱颖，我希望我们是在心平气和地谈话，而不是你一味地在跟我说气话。”傅夜沉耐着性子，再次压低自己的嗓音。
<br/>
<br/>        “你昨天为什么就不选择跟我好好说话？是不是每一次，我都得顺着你的脾气来？你心情好的时候，可以对我百般呵护，心情不好的时候，便对我为所欲为？”一想到昨天的事情，童筱颖呼吸一窒，脸色苍白，说话的声音也不禁在微微颤抖着，“你对我，除了‘性’，只怕就没有别的兴趣了吧！或者说，我于你而言，唯一的好处就是合法‘睡’。要不以后，这样吧！你睡我一次，我就吃一次事后药，一直吃到我终生无法生育为止。反正，你从始至终，一直都在怀疑我。”
<br/>
<br/>        闻言，傅夜沉的心瞬间犹如被冰封了般阵痛。
<br/>
<br/>        她的话对他来说，就像一记巴掌，重重打在他脸上。
<br/>
<br/>        “童筱颖！”他硬生生地板着俊脸，冷冷的声音恍若从遥远的地方飘来，“你到底想我怎样？”
<br/>
<br/>        他向她道歉，向她解释，结果，她油盐不进。
<br/>
<br/>        各种负面情绪在胸口不断地积压，童筱颖嘴角却漾起突兀的微笑：“想你不要再来伤害我，离我越远越好，你能做到吗？”
<br/>
<br/>        傅夜沉能感觉到她童筱颖对他的那颗心，在逐渐自我封闭起来。
<br/>
<br/>        他再看着她脸上心灰意冷的笑，那种莫名的不安突然窜进他心头，让心慌意乱到束手无策。
<br/>
<br/>        “筱颖，不要再跟我说气话了。昨天是我不对，我跟你道歉。我没有怀疑你，我昨天只是气不过。我真的很想要个属于我们俩的孩子。”她对他已经说了够多的狠话了，但是他仍旧耐着性子，将自己所有的脾气都收敛起来，想要好好地哄她，好好地跟她谈话。
<br/>
<br/>        在他看来，与自己爱的女人结婚生子，把家变得更加圆满，难道也有错吗？更何况，他已经到了最适宜的生子年龄。
<br/>
<br/>        他的语气已经算是低声下气了，就在他将大手伸过去，想要握住她放在桌上的左手。
<br/>
<br/>        她却无情地将手挪开，避开他那亲密的举动。
<br/>
<br/>        他没料到她会躲开他，他的手僵在半空中显得有些孤伶。
<br/>
<br/>        傅夜沉竭力忍住自己的怒火，不想在她面前再次发作。
<br/>
<br/>        “你要的只不过是傅氏集团的继承人。”到此刻，她仍旧气不过他强暴她，还强迫她生孩子的那事。
<br/>
<br/>        昨天在他强暴她之前，她原本想跟他解释的。
<br/>
<br/>        但是现在，一切都晚了。
<br/>
<br/>        她什么解释也不想跟他说，甚至也不想听他那些冠冕堂皇的话。
<br/>
<br/>        在她爱上他后，他第一次对她动怒，让她双耳失聪；他第二次对她动怒，让她那儿撕裂下到不了床；他第三次对她动怒，就连把尺子都能割破她的手。
<br/>
<br/>        是不是接下来还有第四次、第五次，然后最后一次干脆直接要了她的命？
<br/>
<br/>        “我要的是你为我生的继承人，其他女人不可以，筱颖，你懂吗？”傅夜沉冷嗓低喝，浑身紧绷，被她较劲搞得心慌不已，“我爱你。”
<br/>
<br/>        “我知道。”她突然十分淡定，脸上没有任何表情，抬眸冷眼看着坐在对面的他，“你爱我，但你不却把信任给我。你爱我，但你却不去了解我。你爱我，但你只是想跟我上床。”
<br/>
<br/>        “你可不可以，不要曲解我的意思？”傅夜沉的心猛然一抽，脸色铁青。
<br/>
<br/>        童筱颖深深吸了口气，缓缓开口：“瞧！我也不了解你，所以才会曲解你。”
<br/>
<br/>        “筱颖，适可而止，好吗？”傅夜沉满脸惊痛，怒吼在她耳畔爆开来。
<br/>
<br/>        “那我可以告你婚内强奸吗？”童筱颖以为自己很冷静，但话一出口，才赫然发现自己的口气，仍旧又冲又怒。
<br/>
<br/>        “筱颖，你在说什么？”傅夜沉只觉心口一阵紧缩，呼吸沉重起来，不可置信地带着怒气的嗓音，让周遭的气氛冷凝。
<br/>
<br/>        “我要告你婚内强奸。”她十分冷静地复述。
<br/>
<br/>        “你确定？”他阴郁的黑眸凝视着她。
<br/>
<br/>        “我确定。”
<br/>
<br/>        童筱颖这句斩钉截铁的回答令傅夜沉浑身一震，俊逸的五官仿佛遭到重击般表情极其难看地拧成了一团。
<br/>
<br/>        “好。”傅夜沉脸色阴沉地扯动了一下嘴角，露出一个令人心颤而扭曲的笑容，顿了顿后，他突然从餐桌前站了起来，居高临下地俯视着她，“我会帮你请最好的律师团队来告我！”
<br/>
<br/>        语毕，他毅然离去，从她面前消失。
