翌日清晨。
<br/>
<br/>        夏夕绾睁开了惺忪睡眸，昨晚一夜好眠，她不由得往温暖的被子里埋了埋，还将自己的小脸往旁边蹭去。
<br/>
<br/>        可是，身旁已经没有人了。
<br/>
<br/>        陆寒霆起床了。
<br/>
<br/>        夏夕绾纤长的羽捷一颤，昨晚她没有醒，但是迷迷糊糊的感觉到身旁睡了一个人。
<br/>
<br/>        这个人除了陆寒霆还能有谁？
<br/>
<br/>        是她的错觉么？
<br/>
<br/>        夏夕绾将小脸埋在枕头里，迅速嗅到了他身上那股干净清冽的男人味道，就连被子里都沾染着他残留的体温。
<br/>
<br/>        昨晚，他真的跟她睡在一起的，两个人相拥而眠。
<br/>
<br/>        夏夕绾轻轻的闭上眼，明明说好了跟他划清界限，但是又是亲吻又是一起睡觉的，这究竟是搞哪般？
<br/>
<br/>        夏夕绾起床，林婶还在昏迷，她给林婶施了针，又去主治医师那里讨论了一下病情，然后回了幽兰苑。
<br/>
<br/>        ……
<br/>
<br/>        幽兰苑。
<br/>
<br/>        陆老夫人拉住了夏夕绾的小手，“绾绾，林婶好些了没，我看你一直往医院跑真的很辛苦的，马上黑眼圈都有了，不如将林婶接到我们幽兰苑，再请专业的医护人员来照看，这样一举两得。”
<br/>
<br/>        夏夕绾心里真的很温暖很感动，老夫人对她是极好的，十分慈祥和宠爱，但是她不希望再添什么麻烦，所以就婉拒了，“奶奶，医生说林婶需要在医院里观察几天，等她苏醒了才能确认情况，这个时候不能转院的，谢谢奶奶。”
<br/>
<br/>        陆老夫人摸了摸夏夕绾的小脑袋，嗔道，“绾绾，你跟奶奶还客气什么，我们是一家人。”
<br/>
<br/>        “知道了奶奶。”夏夕绾乖巧的笑道。
<br/>
<br/>        喵~
<br/>
<br/>        喵~
<br/>
<br/>        这时耳畔响起了几声小猫叫，夏夕绾感觉有什么东西在她脚边蹭，她垂眸一看，脚边多了一只小奶猫。
<br/>
<br/>        全身雪白的小奶猫一看就是高级品种，刚出生没多久，乌亮的大眼睛，奶奶的，让人一看就心里化了。
<br/>
<br/>        夏夕绾盈亮的翦瞳瞬间一亮，当即俯下身将小奶猫给抱在了怀里，惊喜道，“奶奶，哪里来的猫啊？”
<br/>
<br/>        “绾绾，你喜欢这只猫么，这是寒霆出差给你带回来的礼物。”老夫人笑道。
<br/>
<br/>        陆寒霆送给她的？
<br/>
<br/>        他出差还给她带了礼物？
<br/>
<br/>        夏夕绾摸着小奶猫身上的软毛，唇边勾出了一道弧线。
<br/>
<br/>        这时陆老夫人怀里抱了一个精致的洋娃娃，炫耀道，“绾绾，这是寒霆带给我的礼物，怎么样，好看吧？”
<br/>
<br/>        陆老夫人特别喜欢洋娃娃，她房间的床上都堆放着各式各样的洋娃娃，一个老夫人喜欢小女孩的洋娃娃，这很滑稽好笑的。
<br/>
<br/>        陆老夫人还振振有词道，“不要觉得我喜欢洋娃娃很奇怪，绾绾你是小女孩，奶奶我是老女孩，都是女人谁还不是一个公主呢？”
<br/>
<br/>        夏夕绾真的笑了，她小时候的事情很多忘了，九岁女孩的记忆本来就很少，印象里她的妈咪十分温柔，很爱她，身上还香香的，但是妈咪性格冷清，还深居简出，见过妈咪的人很少很少。
<br/>
<br/>        印象里她的父亲很爱她的妈咪，父亲对妈咪说话时都会笑，双眼还亮亮的，相敬如宾。
<br/>
<br/>        但是，她总感觉那个家缺少了些什么，一切都像是表面现象。
