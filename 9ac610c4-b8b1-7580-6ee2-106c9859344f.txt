　　“好！”
<br/>　　神王听到这话，也当即大喝一声，
<br/>　　“布阵！”
<br/>　　唰唰唰！
<br/>　　数十名神界强者，几乎同时出手，
<br/>　　轰轰轰！
<br/>　　刹那间，风云变幻，天地变色，整个华山之巅，犹如被一块巨大的乌云所笼罩，彻底没了踪影。
<br/>　　此时在华山脚下，密切关注着的众人，也全都脸色大变，纷纷露出担忧之色。
<br/>　　“嫂子，我们不该让尘哥去冒险的，你刚刚怎么也不拦着他？”
<br/>　　李虎忍不住冲着一旁的萧若曦道。
<br/>　　‘萧若曦’的那一双美眸，忽然变得深邃无比，语气也一改平常，淡淡道：
<br/>　　“既然是他决定要做的事情，谁也阻止不了，我们能做的，就是尽全力帮助他！”
<br/>　　李虎似乎也察觉到，此时的萧若曦和平日里的萧若曦似乎有些不同，不过具体哪里不同，却又说不上来，苦笑道：
<br/>　　“嫂子，你这话就有些异想天开了吧？就凭我们这些人的实力，怎么帮他啊！”
<br/>　　‘萧若曦’微微一笑，没有再多说什么。
<br/>　　……
<br/>　　华山之巅，一场旷世大战，一触即发。
<br/>　　而此时，离华山数百里外，一座隐蔽山峰之上，
<br/>　　“看来神王路西法已经出手了！魔祖，你和此子交过手，你怎么看？”
<br/>　　说话的，是一名身穿白色道袍，而且须发尽白的老者，飘飘然恍若神仙中人。
<br/>　　原来，这老者，便是仙界第一强者，天榜上排名第一的无涯老人。
<br/>　　而此时，同无涯老者并排而立的，则是一名身穿黑袍，一身邪气的阴鸷老者。
<br/>　　若是叶尘在此，自然便能一眼认出此人，赫然正是当初在魔界，与他交过手的那位魔界至强者，魔祖。
<br/>　　魔祖听到无涯老人这么问，顿时冷冷一笑，
<br/>　　“本祖的确在数月之前，曾与其交过手，那时的他，最多也不过就是半步通天境的实力，如今时隔数月，纵然他天赋再逆天，也绝不可能真正踏入那个境界！”
<br/>　　“神界的弑神大阵，是何等威力，你我都很清楚，再加上路西法亲自坐镇，便是通天境或者化神境的强者，都足以有一战之力，那小子自投罗网，肯定必死无疑了！”
<br/>　　魔祖说完自己的看法之后，转而瞥了一眼身旁的无涯老人，
<br/>　　“无涯，你怎么看？”
<br/>　　无涯老人淡淡一笑，捋了捋雪白修长的胡须，摇了摇头道：
<br/>　　“从表面上看，他的确没有任何胜算，可是我总觉得，此子没有那么容易死掉！”
<br/>　　魔祖听到无涯老人的这番话，顿时眉头微微一皱，继而冷冷道：
<br/>　　“即便路西法杀不了他，不还有我们吗？这可是数千年以来，我们魔、仙、神三界，第一次合作，难道还杀不死区区一个世俗小子？”
<br/>　　后方人群中，仙、魔两界的高手们纷纷称是。
<br/>　　而在人群最后方，妖界的一众高手，则个个满脸尴尬之色。
<br/>　　魔祖只说魔、仙、神三界，故意不提他们妖界，显然是没有把他们放在眼里。
<br/>　　不过这也难怪，毕竟四界之中，以妖界实力最弱，而且当初叶尘大闹妖界，连他们妖界最强者九首妖皇，都被叶尘诛杀，实力自然更加虚弱，虽然名为四界联盟，其实根本没有任何话语权。
<br/>　　……
<br/>　　就在魔祖和无涯老人交谈之上，此时华山之巅上，那弑神大阵之中，大战早已经掀起。
<br/>　　外面的人无法看到里面的情况，不过却能清晰的感受到，整座华山都在剧烈的晃动，仿佛随时都要倒塌一般！
<br/>　　“圣光洗礼！”
<br/>　　弑神大阵之中，随着神王路西法一声暴喝，数十名神界高手，同时施展神通手段，朝踪影的叶尘发起了进攻！
<br/>　　轰轰轰！
<br/>　　一道道仿佛可以撕碎虚空的白色圣光，瞬间将叶尘笼罩其中。
<br/>　　“琉璃金身！”
<br/>　　叶尘大喝一声，身上金芒暴现！
<br/>　　铿铿铿！
<br/>　　然而在那数十道白色圣光的攻击之下，叶尘的琉璃金身只支撑了片刻，周身上下的金芒，便被无情的破开！
<br/>　　“哈哈哈！”
<br/>　　神王路西法看到这一幕，顿时哈哈大笑，
<br/>　　“叶狂仙！原来你也不过如此！死去吧！”
<br/>　　路西法得意之下，再次施展出一道圣光洗礼，
<br/>　　噗！
<br/>　　叶尘肩头被那道圣光砍中，一道鲜血飘飞，虽然并未斩断叶尘的胳膊，却也留下了一道血痕！
<br/>　　“滋滋滋~~”
<br/>　　那道白光顺着叶尘的伤口，竟然又开始朝他的身体内侵蚀！
<br/>　　“哈哈！你已经中了我的圣光洗礼，待会实力最少也会削弱三成！”
<br/>　　“不得不说，你实在太令我失望了！哈哈哈哈！”
<br/>　　路西法显然没有想到，他费尽心思设下陷阱，没想到竟然这么容易，就重创了叶尘，他都还没有使出全力呢！
<br/>　　“你！根本不配做我神界的对手！今天，我就让你知道，得罪我神界的下场！”
<br/>　　神王路西法手握一根金色权杖，忽然高高举起，
<br/>　　轰！
<br/>　　一道璀璨的金色光芒，自那权杖上发出，继而狠狠朝下方的叶尘砸落！
<br/>　　而与此同时，周围数十名神界强者，也同时出手，封住了叶尘所有退路。
<br/>　　“死！”
<br/>　　巨大的金色权杖，落在下方叶尘的头顶，整个人瞬间淹没在那璀璨的金芒之中。
<br/>　　然而，不知道是不是幻觉，就在权杖落下的那一刹那，路西法隐约注意到，叶尘的脸色平淡无比，甚至嘴角还挂着一抹笑意。
<br/>　　“嗯？难道是我的错觉？”
<br/>　　路西法心头惊疑不定，待那金芒散去，下方却早已经没有了叶尘的身影，似乎已经被大阵的力量，绞杀的一干二净。
<br/>　　“哈哈哈哈！什么全球第一人！还横扫仙、魔、妖三界，本王还以为有多大能耐，没想到也不过如此！魔祖！无涯老儿！看来数百年不见，你们两界的实力，着实令人担忧啊！哈哈哈！”
<br/>　　原来路西法早就知道，其他三界的高手在一旁观战，于是开始肆无忌惮的嘲讽起来。
<br/>　　“这就结束了？”
<br/>　　另外一边，就连之前并不看好叶尘的魔祖，也不由得两眼一瞪，露出一抹意外之色。
<br/>　　以之前叶尘在魔界的表现来看，应该不至于这么弱吧？
<br/>　　华山脚下，易山河等人听到路西法的狂妄声音，顿时个个脸色惨白，
<br/>　　“难道叶帅他已经……”
<br/>　　不料，
<br/>　　一旁的‘萧若曦’却脸带笑容，淡淡道：
<br/>　　“想杀他，可没有那么容易呢！”
<br/>　　‘萧若曦’的话才刚刚落下，华山山巅之上，陡然响起一道，淡漠中透着戏谑的声音，
<br/>　　“神王，灭掉我区区一道分身，你至于这么兴奋吗？”
