　　金身佛尊的骸骨不在，楚风眠的眼神中不由的露出了几分失望。
<br/>　　金身佛尊，毕竟可是赫赫有名的佛门战佛，以肉身踏入天人境的无敌强者。
<br/>　　只怕是这金身佛尊，也是一尊凝练了不朽真身的强者。
<br/>　　楚风眠如今身躯之中的半步不朽真身，就是因为楚风眠得到了那青铜天魔骸骨。
<br/>　　以神树炼化了青铜天魔骸骨，楚风眠才凝炼出了这半步不朽真身。
<br/>　　一尊天人的骸骨，几乎不亚于藏金精这等至宝的存在，如果被楚风眠得到，同样是被神树炼化的话，对楚风眠的提升极大。
<br/>　　只可惜在这陵墓之中，并没有这金身佛尊的骸骨。
<br/>　　楚风眠的目光再度向着陵墓石室之中的其他角落看去。
<br/>　　除了藏金精之外，倒是还有着几道宝光，这些宝光，多都是一些有着炼体之效的诸天奇物。
<br/>　　虽然在效果上，无法是跟藏金精媲美，倒是也算是极为难得的宝物了。
<br/>　　楚风眠也不客气，凝聚灵力在掌心之中，猛然一抓。
<br/>　　这些诸天奇物，尽数收取到楚风眠的身躯之中去。
<br/>　　轰然之间，吞天秘术运转起来。
<br/>　　将这些诸天奇物的灵力，尽数吞噬，一件件诸天奇物的灵力，进入到楚风眠的身躯之中。
<br/>　　便是分开开来，涌入到每一枚血肉晶体之中，在融入了这些诸天奇物后。
<br/>　　楚风眠的血肉晶体，都变的坚韧了几分，其中的联系也变的更加的紧密起来。
<br/>　　“那六翼蜈蚣，不在这石室之中。”
<br/>　　楚风眠将这石室之中，都给仔细的搜寻了一边，其中留下的炼体之道的至宝，无数诸天奇物，楚风眠都没有放过，尽数收取。
<br/>　　只是没有见到六翼蜈蚣的踪影。
<br/>　　之前在赤红大陆，在这佛寺开启之前，显现而出的无数宝光之中。
<br/>　　楚风眠最为看重的，便是两件至宝，一为藏金精，如今已经是落入到了楚风眠的手中。
<br/>　　二则是六翼蜈蚣。
<br/>　　这两件都是炼体之道的至宝，在古籍的记载之中，如果是想要冲击肉身极致，不朽真身。
<br/>　　这两件炼体之道的辅佐，必不可少。
<br/>　　而且这两件至宝，更是有着相辅相成之用。
<br/>　　藏金精之中，蕴含纯粹的厚土之力，所以足以提炼肉身之用，但是如此纯粹庞大的力量，一旦是稍有不慎，便是可能会被其中的厚土之力所反噬。
<br/>　　倒时候肉身石化，轻则失去肉身，重则可能当场陨落。
<br/>　　想要炼化藏金精，必须要有着六翼蜈蚣作为中和之用。
<br/>　　两者一同炼化，才可以发挥出这两件至宝的全部作用。
<br/>　　如今这藏金精，是在金身佛尊的陵墓之中，可是那六翼蜈蚣，却是不在。
<br/>　　这让楚风眠有些可惜。
<br/>　　不过在这藏地之中，埋葬的佛门强者，何止千万。
<br/>　　六翼蜈蚣，葬在其他佛门强者的陵墓之中，也极为正常。
<br/>　　楚风眠看了一眼手心之中的藏金精，还是将其收起到了空戒之中。
<br/>　　等楚风眠寻找到六翼蜈蚣时，在将这两件至宝，一同炼化再说。
<br/>　　“走！”
<br/>　　石室之中，已经是被收取一空，楚风眠也没有在留在这里的必要了。
<br/>　　这藏地之中，处处都是宝藏，楚风眠收取的宝藏越多，也越是足以提升他的实力。
<br/>　　楚风眠一股灵力注入到手腕之上的万念佛珠之中。
<br/>　　一股精粹的佛门之力，降临在楚风眠的背后，化为了一道佛光。
<br/>　　随即楚风眠便是身形一动，向着佛寺之外飞去。
<br/>　　没有转轮和尚故意催动这佛寺之中的佛门禁制，单单凭借万念佛珠的力量，还是足以瞒天过海。
<br/>　　一路上的佛门禁制，在接触到楚风眠背后的佛光一刻，力量便是削弱了十倍。
<br/>　　楚风眠横冲直撞，连连穿越了无数佛门禁制，一下子离开了佛寺，化为了一道遁光，直接向着远方飞去。
<br/>　　就在楚风眠遁光离开佛寺之后没多久，两道身影从这佛寺边缘之中，显现了出来。
<br/>　　“转轮和尚，你刚刚为何不催动这佛寺之中的佛门禁制，对付这小子？”
<br/>　　这两道身影，正是转轮和尚跟小耀王寒夜二人。
<br/>　　如今这小耀王寒夜身上的气息，已经是虚弱无比，显然是失去了大量的精血，他的实力，都是受损严重。
<br/>　　看着楚风眠从这佛寺之中轻易离开，小耀王寒夜的目光冰冷无比。
<br/>　　尤其是他也看得出来，如今楚风眠身上的气息，比起刚刚的时候，还要强横几分。
<br/>　　显然是这金身佛尊陵墓之中的宝藏，已经是落入到了楚风眠的手心之中，如今全部化为了楚风眠的力量。
<br/>　　这小耀王寒夜，前来邀请楚风眠一起去探索那金身佛尊的宝藏，本来也是想要让楚风眠作为炮灰，来让他坐收渔翁之利的。
<br/>　　可现在，却是引狼入室，凰公子陨落，他也已经是差一点就死在了楚风眠的手中。
<br/>　　至于金身佛尊陵墓之中的宝藏，更是都被楚风眠得到，这一切让小耀王寒夜，已经是接近疯狂。
<br/>　　“不，也此人的实力，哪怕是我催动这佛寺之中的禁制，也最多给他带来一点麻烦罢了，伤不到他，还容易打草惊蛇。”
<br/>　　听到小耀王寒夜的质问，转轮和尚摇了摇头道。
<br/>　　这佛寺之中的佛门禁制，虽然厉害，可是他一旦是催动起来，最多也就是拖延一下楚风眠的时间罢了。
<br/>　　想要真正伤到，甚至是轰杀楚风眠，单单是凭借这些禁制，还远远做不到。
<br/>　　“那怎么办？一旦是让此人的实力再进一步，你我谁都会死在这小子的手里。”
<br/>　　小耀王寒夜冷哼一声的说道。
<br/>　　“这小子的性格狠辣无比，对待仇敌，向来是斩草除根，我们难道就这样坐以待毙，看着小子的实力一点点增长起来？”
<br/>　　“不用着急，这里可是净土佛寺，是我们净土宗的地盘，有的是机会。”
<br/>　　转轮和尚脸色平静，但是他的目光之中，也是看向楚风眠，蕴含杀意。
<br/>　　“走，我知道这藏地之中，有着几处我们佛门强者的藏地，其中的一些宝物，可以助你恢复实力。”
